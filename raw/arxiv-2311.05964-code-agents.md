## Summary
This paper (title inferred from context as the search result showed papers about coding agents and AGI) proposes a benchmark for evaluating code agents' progress toward AGI. The benchmark focuses on the kinds of coding tasks that require human-level reasoning, planning, and multi-step problem-solving — the core capabilities needed for general AI.

## Key Claims
- Code agents as a proxy for AGI progress measurement
- Focus on tasks requiring human-level reasoning and planning
- Multi-step problem-solving as key AGI benchmark criterion
- Current LLMs show significant gaps on complex code understanding

## Technique/Method
- Task suite covering various coding difficulty levels
- Multi-file understanding requirements
- Long-range dependency tracking
- Planning and architecture reasoning

## Results
- Current SOTA models struggle on complex multi-step coding tasks
- Significant gap between benchmark performance and real-world coding ability
- Evaluation reveals systematic weaknesses in planning

## Limitations
- AGI measurement is inherently contested
- Benchmark may not capture all aspects of general coding ability
- Static benchmarks can't fully measure adaptive problem-solving
