## Summary

Gate 6 (evidence) was completely passive — computed but never included in the merge-blocking `ALL_PASS` check. This means PRs with zero video evidence were merging freely.

## Changes

1. **EVIDENCE_STATUS now emits `FAIL` when evidence is missing** (was `N/A (optional)`)
2. **$EVIDENCE_STATUS added to `ALL_PASS` loop** — missing evidence now blocks merge
3. **Evidence request comment** posted to PR when missing — tags @author with instructions for terminal (asciinema) and UI (loom/PNG) video proof
4. **PR author captured** from `.user.login` for @mention in evidence request

## Option B implementation

Without evidence-review-bot installed, the fail-closed gate blocks merges until workers post video evidence links. When evidence is missing:
- skeptic-cron posts a comment: `@author — Merge blocked: Gate 6 (evidence) requires video proof`
- AO workers must post tmux terminal video + UI video to PR thread
- Once evidence is linked, skeptic-cron's next run passes Gate 6
