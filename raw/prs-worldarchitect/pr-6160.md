## Raw Body
### Background
We recently locked down the evidence gate to strictly require video proofs via `.mp4`, `.cast`, `loom.com`, etc. However, users are occasionally bundling their video proofs in self-contained GitHub Gists to bypass size limitations on PR comments.

### Goals
- Expand the `skeptic-cron.yml` evidence regex rule to permit `gist.github.com` URLs as valid evidence links.

### High-level description of changes
- Modified the ERE regex in Gate 6 to allow ` gist.github.com/ ` links and replaced brittle bracket matching.

### Testing
- `echo "https://gist.github.com/..." | grep -qEi 'new-regex'` returns MATCHED.
