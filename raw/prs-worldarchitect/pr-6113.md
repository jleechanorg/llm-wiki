## Summary
- Align Cursor `PreToolUse` / `PostToolUse` Bash hooks with the existing portable pattern in `.claude/settings.json` (invoke `.claude/hooks/run-metadata-updater.sh` via `git rev-parse --show-toplevel`).
- Removes machine-specific absolute paths to worktrees and AO session directories so clones and CI behave consistently.

## Production code changes
- **Before:** Local-only `.cursor/settings.json` pointed at fixed `/Users/...` paths for hooks.
- **Now:** Same wrapper commands as Claude Code nested hooks: resolve repo root, run `run-metadata-updater.sh` if present and executable.
- **Why:** Agent Orchestrator metadata updates should work for any checkout; developers still install a local (gitignored) `.claude/metadata-updater.sh` as documented.

## Test changes
- None.

## Known limitations
- Hook is a no-op until a local `.claude/metadata-updater.sh` exists (gitignored; same as Claude Code).
