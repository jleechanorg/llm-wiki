## Raw Body
Implement email notifications for deployment success/failure across all environments.

Key features:
- Centralized composite action for reusable email notifications
- Automatic success ✅ / failure ❌ detection and formatting
- Rich email content with deployment URLs, health checks, console links
- Non-blocking email delivery (continue-on-error: true)
- SHA-pinned actions for security (dawidd6/action-send-mail@v6)
- Test workflow for validating email configuration

Integration:
- deploy-dev.yml: Auto email on dev deployments
- deploy-production.yml: Auto email on production deployments
- test-email-notification.yml: Manual testing workflow

Setup required:
- GitHub Secrets: EMAIL_USER (Gmail address)
- GitHub Secrets: EMAIL_APP_PASSWORD (Gmail app password)

Files:
- .github/actions/send-deploy-notification/action.yml
- .github/actions/send-deploy-notification/README.md
- .github/workflows/test-email-notification.yml
