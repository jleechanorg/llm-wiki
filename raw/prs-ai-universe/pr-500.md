## Summary

This PR adds GitHub Actions auto-deployment workflow for the backend repository, enabling automatic deployments to the development environment when changes are pushed to the main branch.

## Changes

- **New workflow:** `.github/workflows/auto-deploy.yml`
  - Triggers on push to `main` branch
  - Deploys to `ai-universe-backend-dev` service in GCP
  - Uses existing `GCP_SA_KEY` secret for authentication
  - Runs `./deploy.sh dev` for deployment
  - Includes health check validation with 10 retries
  - Provides deployment summary in GitHub Actions UI

- **Documentation:** `docs/GITHUB_ACTIONS_AUTO_DEPLOY.md`
  - Explains the auto-deployment workflow
  - Documents branch mapping (main → dev)
  - Provides usage instructions and monitoring links
  - Includes troubleshooting guidance
