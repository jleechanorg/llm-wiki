## Summary

Adds `/secondo` (alias: `/secondopinion`) slash command for getting comprehensive multi-model AI feedback on code, design decisions, or bug analysis.

## Features

✅ Multi-model consensus analysis from 5+ AI models:
- Cerebras (Qwen 3 Thinking) - Primary
- Gemini 2.5 Flash
- Perplexity Sonar
- OpenAI GPT-4o
- Grok 3

✅ Three feedback types:
- `design` - Architectural decisions and best practices
- `code-review` - Security, performance, quality
- `bugs` - Potential issues and edge cases
- `all` - Comprehensive review (default)

✅ Production-ready implementation:
- Authentication integration with auth-cli.mjs
- Beautiful colored output with synthesis parsing
- Comprehensive error handling (auth, rate limits, timeouts)
- Token usage and cost tracking
- Industry-sourced recommendations with citations

## Usage

```bash
# Get second opinion on current context
/secondo

# Specify feedback type
/secondo design
/secondo code-review
/secondo bugs

# With specific question
/secondo "Should I use Redis or in-memory caching for rate limiting?"
```

## Authentication Required

Requires authentication via:
```bash
node scripts/auth-cli.mjs login
```

## Rate Limits

- **Authenticated**: 60 requests/hour
- **Multi-model synthesis**: 1 per hour
- Token expires after 1 hour
