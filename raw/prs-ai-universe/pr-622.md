## Raw Body
- Create mcp-smoke-test-mocks.mjs module with comprehensive mock responses
- Modify mcp-ci-tests.mjs to support MCP_TEST_MODE environment variable
  - Default mode: 'mock' - uses example responses, no real API calls
  - Real mode: 'real' - hits actual APIs (set MCP_TEST_MODE=real)
- Mock responses include all required diagnostics for validation:
  - OpenAI web_search tools
  - OpenRouter with Exa search engine
  - Perplexity with web search mode
  - Gemini with googleSearch and dynamicRetrievalConfig
  - Grok with search_parameters
- Mock conversation persistence with stateful conversation tracking
- Add /smoke slash command for real API testing
- Tests pass in mock mode without requiring MCP_SERVER_URL

Benefits:
- Fast smoke test execution during development (no API calls)
- No API credentials needed for basic testing
- Predictable test results for CI/CD
- Real API testing still available via /smoke command or MCP_TEST_MODE=real
