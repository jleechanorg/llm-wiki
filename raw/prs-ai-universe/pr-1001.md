## Changes

- **Dev auto-deploy permissions**
  - Add `pull-requests: read` permission at workflow and deploy job levels so dev deployments can reliably resolve pull request metadata for email/notification workflows.
  - Files: `.github/workflows/auto-deploy.yml`, `.github/workflows/deploy-dev.yml`.

- **OpenAI GPT-5 Mini model support**
  - Add official GPT-5 Mini pricing (`$0.25/$2.00` input/output per 1M tokens) to `CostCalculator`.
  - Add normalization aliases for `gpt5mini`/`gpt-5-mini`.
  - Add user-facing display name `OpenAI GPT-5 Mini`.
  - Files: `shared-libs/packages/mcp-server-utils/src/CostCalculator.ts`, `shared-libs/packages/mcp-server-utils/src/ModelDisplayNames.ts`.

- **Redis rate limit client hardening**
  - Configure Redis client to fail fast in degraded environments: disable offline queueing, set 5s connect timeout, disable auto-reconnect.
  - Keep rate limit behavior resilient to Redis failures by allowing fallback paths to handle temporary availability issues.
  - File: `shared-libs/packages/mcp-server-utils/src/RateLimitTool.ts`.
