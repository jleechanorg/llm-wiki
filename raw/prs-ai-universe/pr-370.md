Added interactive browser-based MCP Inspector tool for testing AI Universe backend endpoints.

- Server configuration dropdown (localhost, Render, GCP dev/prod)
- Editable connection fields (URL, credentials)
- Connection testing with JSON-RPC initialize

- Real-time response display with error handling

Location: static/mcp-inspector.html
Usage: Open in browser to test MCP endpoints interactively

GitHub Pages compatible - can be hosted as static site.
