**Key Features:**

- Rebase state detection (.git/rebase-merge)
- Context-aware error messages
- Step-by-step fix instructions
- References to AGENTS.md documentation

Addresses the root cause of rebase workflow confusion where hooks block silently, making it unclear why `git rebase --continue` appears to hang indefinitely.
