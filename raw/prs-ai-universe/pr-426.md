## Raw Body
PR reviewer feedback identified that synthesis prompts required AI models to duplicate the same references in both "Top 5 References" and "All References" sections, creating unnecessary redundancy and potential confusion.

## Changes

- **Synthesis prompt clarification** (`SecondOpinionAgent.ts` lines 1717, 2406): Updated "All References" section to explicitly list only additional references beyond the Top 5, eliminating duplication requirement

**Before:**
```typescript
- **All References**: [Complete list of every reference you used, ordered by importance (most important first) with full, non-truncated titles and URLs. Include the same entries from the Top 5 References section before listing any additional references, or state "No credible sources available" if applicable.]
```

**After:**
```typescript
- **All References**: [List any additional references beyond those in the Top 5 References section, ordered by importance (most important first) with full, non-truncated titles and URLs. If there are no additional references, state "No additional credible sources available."]
```

This applies to both two-step and three-step synthesis flows.
