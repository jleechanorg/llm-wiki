## Summary

Adds a script to analyze Firebase Firestore user activity and generate usage reports.

## Features

- **User Activity Analysis**: Queries all conversations and messages to count activity per user
- **Top Users Report**: Shows top 20 users ranked by message count with statistics
- **Message History**: Displays last 20 messages per user with timestamps and conversation IDs
- **Email Matching**: Attempts to match user IDs with email addresses from Firebase Auth
- **Report Export**: Saves comprehensive report to `~/Downloads/user-activity-report-YYYY-MM-DD.txt`

## Usage

```bash
node scripts/query-user-activity.mjs
```

## Requirements

Requires Firebase Admin SDK credentials (one of):
- `FIREBASE_SERVICE_ACCOUNT_BASE64` environment variable
- `GOOGLE_APPLICATION_CREDENTIALS` environment variable pointing to service account JSON
- `gcloud auth application-default login`

## Output

- Console output with user activity statistics
- Report file saved to `~/Downloads/user-activity-report-YYYY-MM-DD.txt` with:
  - Summary statistics (total users, conversations, messages)
  - Top 20 users ranked by message count
  - Last 20 messages per user with full content (up to 500 chars per message)
