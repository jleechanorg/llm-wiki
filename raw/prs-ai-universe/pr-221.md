> [!NOTE]
> Introduces tree-hash caching across hooks, adds a new post-commit hook, and skips pre-push checks when the tree is unchanged and the working tree is clean.
> 
> - **Git hooks setup (`scripts/setup-git-hooks.sh`)**
>   - Detects existing `post-commit` and includes it in overwrite prompt.
>   - Installs new `post-commit` hook and sets executable permissions.
> - **Pre-commit hook**
>   - Caches staged tree hash to `hooks/../.pending_presubmit_tree` after successful `presubmit.sh`.
>   - Resolves file paths via `git rev-parse --git-path`; cleans pending file on failures.
> - **Post-commit hook (new)**
>   - On successful commit, persists matching tree to `hooks/../.last_presubmit_tree` and clears pending cache.
> - **Pre-push hook**
>   - Skips checks when `HEAD^{tree}` equals `hooks/../.last_presubmit_tree` and working tree is clean; otherwise runs tests.
>   - Adds informative skip/dirty-tree messages.
