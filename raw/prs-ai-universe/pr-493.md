…-utils

Extracted generic utilities that are not second-opinion-specific:

**New Utilities in mcp-server-utils:**
- HttpClient: HTTP client with connection pooling and security
- CostCalculator: LLM cost calculation with model pricing database
- FirebaseAuthTool: Firebase authentication with admin management
- RateLimitTool: Rate limiting with memory-based atomic operations
- ModelTimeout: Timeout utilities for LLM calls with abort control
- ConcurrencyManager: Mutex-based concurrent request slot management
- ModelDisplayNames: Model ID to display name mapping
- types.ts: Shared type definitions (User, RateLimit, LLMResponse, etc.)

**Benefits:**
- Reusable across multiple MCP servers and agents
- Centralized security features (header sanitization, rate limiting)
- Consistent cost tracking and timeout handling
- Pluggable logger interfaces for all utilities

**Updates:**
- Backend now imports from @ai-universe/mcp-server-utils
- Logger configuration via setter functions
- Added dependencies: async-mutex, axios, firebase-admin

Related utilities (HumanLogger, SecondOpinionAgent core logic) remain in backend as they are second-opinion-specific.
