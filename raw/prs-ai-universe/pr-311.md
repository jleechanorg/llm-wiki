## Key Lessons Documented

1. **Console COOP errors don't prevent Firebase OAuth from working**
   - Errors appear but authentication succeeds with manual interaction
   - COOP warnings are cosmetic, not functional blockers

2. **Manual interaction in Playwright succeeds where automation fails**
   - Automated popup form filling may fail
   - Manual typing and clicking works correctly
   - This proves the production site is functional

3. **Production site is fully functional despite console warnings**
   - Multi-model AI consultation works end-to-end
   - Authentication completes successfully
   - All features operational

4. **Always test with manual interaction before declaring auth broken**
   - Don't jump to conclusions based on console warnings
   - Don't create alarmist documentation without verification
   - Trust production evidence and user reports
