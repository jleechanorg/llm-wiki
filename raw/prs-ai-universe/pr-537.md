## Usage

```bash
# Install auto-start service (macOS only)
./scripts/setup-mcp-inspector-autostart.sh

# Check service status
launchctl list | grep ai-universe
lsof -i :9000

# View logs
tail -f ~/.ai-universe/logs/mcp-inspector.log

# Uninstall
./scripts/setup-mcp-inspector-autostart.sh --uninstall
```
