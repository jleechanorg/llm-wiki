## Raw Body
## Problem

The deployment script had a conditional check that only ran npm ci if the TypeScript compiler binary (tsc) was missing. This caused deployment failures when new dependencies were added but tsc already existed from a previous install.

## Solution

Always run npm ci to ensure all dependencies (including new ones) are properly installed with symlinks.

This ensures:
- All dependencies are installed fresh
- Symlinks for file: dependencies are always created
- New dependencies are never missed
- Deployment is deterministic and reliable
