## Problem
OpenAI Chat Completions API validation requires tool names to match `^[a-zA-Z0-9_-]+$` (only letters, digits, underscores, and hyphens). The AI Universe MCP server had **12 tool names containing periods (`.`)** which caused Codex CLI validation errors.

## Solution
Replaced all periods with underscores in MCP tool names across ConversationAgent and SecondOpinionAgent.
