## Raw Body
Created TypeScriptServerLauncher - a production-ready, generic framework for launching TypeScript servers locally with comprehensive automation.

New Features:
- TypeScriptServerLauncher class with extensive configuration options
- Enhanced port finder with randomized search (avoids thundering herd)
- ProcessManager for cross-platform process control and PID tracking
- BuildOrchestrator for dependency management and TypeScript compilation
- Comprehensive lifecycle hooks for custom integration

Components Added:
1. TypeScriptServerLauncher.ts - Main launcher class
2. ProcessManager.ts - Process/port management utilities
3. BuildOrchestrator.ts - Build pipeline orchestration
4. Enhanced findAvailablePort.ts with randomized port selection
5. Comprehensive tests for all new components

Benefits:
- Reusable across any TypeScript server project
- Smart port management with automatic fallback
- Graceful/forceful process termination with parent detection
- Watch mode support with auto-restart
- Retry logic for EADDRINUSE errors
- Custom build hooks and preparation scripts

Integration:
- Updated run-local-server.ts to use the new library
- Maintained backward compatibility with existing workflow
- All existing functionality preserved and enhanced

Testing:
- Enhanced findAvailablePort tests with randomized search
- ProcessManager tests for PID files and process termination
- Backward compatibility via findAvailablePortSequential

Documentation:
- Comprehensive README with usage examples
- Configuration reference and best practices
- Integration examples and environment variables

Version: @ai-universe/mcp-server-utils v1.3.0
