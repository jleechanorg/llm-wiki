## Raw Body
This commit removes the contact request/approval system entirely as it's no longer needed. Agents can now message each other directly without requiring contact approval.

Changes:
- Removed 4 test files: test_contacts.py, test_contact_policy.py, test_contact_and_routing.py, test_no_contact_enforcement.py
- Removed 5 contact-related tools: request_contact, respond_contact, list_contacts, set_contact_policy, macro_contact_handshake
- Removed AgentLink model from models.py (used for tracking contact approvals)
- Removed contact_policy field from Agent model
- Removed CLUSTER_CONTACT constant and all related code from app.py
- Removed contact-related settings from config.py: CONTACT_ENFORCEMENT_ENABLED, CONTACT_AUTO_TTL_SECONDS, CONTACT_AUTO_RETRY_ENABLED, MESSAGING_AUTO_HANDSHAKE_ON_BLOCK
- Simplified cross-project messaging to work without link approval checks
- Updated all documentation to remove contact request references: README.md, AGENT_ONBOARDING.md, AGENTS.md, CROSS_PROJECT_COORDINATION.md, docs/LAZY_LOADING.md
- Updated integration_showcase.py to remove contact handshake demonstrations

Cross-project messaging now works directly without any approval requirements.
