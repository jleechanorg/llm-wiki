## Raw Body
Add support for deleting agents and all associated data:
- New MCP tool `delete_agent` to permanently remove agents
- Helper function `_delete_agent` that handles cascading deletes
- Deletes agent from database along with:
  * MessageRecipient records
  * Messages sent by the agent
  * File reservations held by the agent
  * Agent links (contacts) involving the agent
- New `write_agent_deletion_marker` in storage.py to write audit trail
- Deletion marker written to Git archive at agents/<Name>/deleted.json

The operation is destructive and irreversible, with clear warnings in the tool documentation. All deletion statistics are returned and preserved in the Git archive for audit purposes.
