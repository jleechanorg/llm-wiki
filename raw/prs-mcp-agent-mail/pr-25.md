## Summary

This PR updates the project to support **Python 3.11+** instead of requiring Python 3.14, and publishes the package to PyPI as `ai-universe-mail`.

### Changes

#### Policy Update (Added in commit 62749c1)
- **AGENTS.md**: Updated Python version policy from "ONLY target Python 3.14" to "target Python 3.11+"
- **README.md**: Updated all references from "Python 3.14" to "Python 3.11+"

#### Package Configuration
- **Fixed Python version requirements**: Changed from `>=3.14` to `>=3.11` in `pyproject.toml`
- **Updated repository URLs**: Replaced placeholder URLs with actual GitHub repository
- **Renamed package**: Changed from `mcp-agent-mail` to `ai-universe-mail`
- **Published to PyPI**: Package is now available as `ai-universe-mail` v0.1.0 at https://pypi.org/project/ai-universe-mail/

#### Documentation & Scripts
- **Added PyPI installation instructions**: Updated README with pip/uv installation commands
- **Created production deployment script**: New `scripts/run_server_pypi.sh` for running the server from PyPI package
- **Updated CLAUDE.md**: Enhanced with instructions for both PyPI (production) and local (development) deployment
- **Created cleanup documentation**: Added `PYPI_CLEANUP.md` with instructions for old package removal

### Policy Justification

The Python 3.14-only policy was updated to 3.11+ based on:

1. **Technical Analysis**: Comprehensive code scan found ZERO Python 3.14-specific features
   - No t-strings (template literals)
   - No sys.version_info checks for 3.14+
   - No 3.14-only stdlib imports
   - No PEP 695 extended syntax

2. **Multi-Version Testing**: Successfully tested on:
   - Python 3.11.10: ✅ Syntax + Import = SUCCESS
   - Python 3.12.11: ✅ Syntax + Import = SUCCESS
   - Python 3.13.7: ✅ Syntax + Import = SUCCESS

3. **Production Deployment**: Server verified running from PyPI package with Python 3.13

4. **Accessibility**: Python 3.14 was released only 1 month ago (Oct 7, 2025); supporting 3.11+ significantly broadens
