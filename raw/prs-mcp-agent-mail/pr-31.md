## Primary Changes

### 🆕 New Feature: Local Build Testing Script
- **New Script**: `scripts/run_server_local_build.sh`
  - Runs MCP server from locally built wheel (from `dist/`)
  - Creates isolated temporary virtual environment
  - Auto-selects Python 3.11-3.13 (avoids 3.14+ due to Pydantic compatibility)
  - Bridges gap between local development and PyPI deployment

**Use Case:**
1. Make changes to source code
2. Build package: `uv build`
3. Test build locally: `./scripts/run_server_local_build.sh`
4. Validate functionality before publishing to PyPI
