when all gates are green.

## 7-Green Criteria

| # | Gate | Check |
|---|------|-------|
| 1 | CI green | GH Actions checks all return `success` |
| 2 | No conflicts | `mergeable=true`, `mergeState=CLEAN` |
| 3 | CR APPROVED | `coderabbitai[bot]` latest review is `APPROVED` |
| 4 | Bugbot clean | `cursor[bot]` has zero error-severity comments |
| 5 | Comments resolved | Zero unresolved non-nit inline review comments |
| 6 | Evidence pass | `evidence-review-bot` has `APPROVED` review |
| 7 | Skeptic PASS | `VERDICT: PASS` comment from `github-actions[bot]` |

## Changes

- Added `check_ci()`, `check_no_conflicts()`, `check_cr_approved()`,
  `check_bugbot_clean()`, `check_comments_resolved()`,
  `check_evidence_pass()`, `check_skeptic_pass()` helper functions
- Skip draft PRs (not eligible for AO dispatch)
- AO dispatch / nudge fires **only when all 7 greens are met**
- Skeptic gate can be skipped via `AO_MONITOR_SKIP_SKEPTIC=1` for repos
  that don't yet have `skeptic-cron.yml`
- Updated `LOG_FILE` output to show `N/7 green` per PR
