## Summary

Full repo review of all 23 Python source files. Fixes 3 code issues + 1 stale test file.

### Bugs Fixed
- **`hmac.new()` → `hmac.HMAC()`** in `webhook_bridge.py`: `hmac.new()` does not exist in Python's `hmac` module. The bug was masked by a broad `except Exception`, causing HMAC webhook signature verification to **silently always fail** — all payloads were accepted when a webhook secret was configured.

### Dead / Duplicate Code Removed
- **Duplicate `_CROSS_REPO_CONTEXT`** in `dispatch_task.py`: Identical constant defined twice (line 227 and 336). The second definition silently overwrote the first. Removed the duplicate.
- **`import uuid` inside function body**: Hoisted from inside `dispatch()` to top-level module imports (PEP8).

- **Stale `test_dispatch_cross_repo.py`**: Test parametrization expected old keyword-based `_is_cross_repo_task` behavior, but the implementation was refactored to delegate to `_extract_repo_name_hint`.
