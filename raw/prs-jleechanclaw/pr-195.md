## Changes

**Bug 1 (High) — `ao-backfill.sh`: `pokedAt` never written on first poke**
`sed -i ''` on macOS exits 0 even on no-match → `|| echo` fallback silently skipped. Rate-limiting was completely non-functional on first poke cycle. Fixed with grep-check + conditional append/update.

**Bug 2 (Medium) — `agento-report.sh`: `cr_approved == "1"` fails for count > 1**
After re-reviews, CodeRabbit can have 2+ APPROVED reviews. `== "1"` comparison mislabeled those PRs as needing CR. Fixed by normalizing to 0/1 and using `-ge 1`.

**Bug 3 (Medium) — `agento-report.sh`: `unstable` mislabeled as merge conflict**
`mergeable_state=unstable` means CI failing, not merge conflict. Fixed: `dirty`→conflict, `unstable`→`⚠️ ci_failing` with separate counter.
