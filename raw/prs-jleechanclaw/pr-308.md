## High-level description of changes
Rewrites the /claw command's gateway dispatch to eliminate several security and reliability issues: token is written to a chmod 600 temp file and deleted immediately after read (no argv/env leak), response body goes to a file instead of a shell variable (no truncation), temp paths use mktemp with random suffixes (no predictable paths), and the SSE parser is replaced with a proper Python heredoc accumulator that handles multi-line events correctly.
