## Background

## Goals
- jleechanclaw has zero Python logic that duplicates AO TypeScript
- Any lingering callers fail loudly with a helpful ImportError pointing to the AO equivalent
- Test suite stays green (31 passing)

## Tenets
- Fail loudly, not silently — ImportError stubs, not silent no-ops
- Minimal diff per module — template stub, consistent format
- Every stub names the exact AO file/plugin/command to use instead
