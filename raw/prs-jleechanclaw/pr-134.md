## Changes

| Module | Purpose |
|---|---|
| `ao_events.py` / `ao_cli.py` | AO event types and CLI integration |
| `action_executor.py` | Executes AO reactions (send-to-agent, notify, escalate) |
| `auto_review_trigger.py` | Triggers PR review on CI-green + approval signals |
| `coderabbit_gate.py` | Waits for CodeRabbit review before proceeding |
| `decomposition_dispatcher.py` | Breaks tasks into subtasks, dispatches agents |
| `escalation_handler.py` / `escalation_router.py` | Escalation triage and routing |
| `failure_budget.py` | Tracks per-PR retry budget, enforces `escalateAfter` |
| `pr_reviewer.py` / `pr_review_decision.py` | LLM-backed PR review with context loading |
| `task_tracker.py` | Task/subtask lifecycle state machine with JSON persistence |

Also adds `.gitleaksignore` for a test fixture that simulates a credential-touching diff (fake key used to verify escalation logic).
