## Changes

### scripts/bug-hunt-daily.sh
- **Bug-hunt loop (5 agents):** Replaced `ao spawn --task` with a `case` dispatch to each agent's verified headless CLI:
  - `claude`: `claude --dangerously-skip-permissions -p "$TASK_PROMPT"`
  - `codex`: `codex exec --dangerously-bypass-approvals-and-sandbox "$TASK_PROMPT"`
  - `gemini`: `gemini --yolo -p "$TASK_PROMPT"`
  - `cursor`: `cursor-agent --print "$TASK_PROMPT"`
  - `minimax`: `claude --dangerously-skip-permissions -p` with `ANTHROPIC_BASE_URL=https://api.minimax.io/anthropic` (MiniMax backend)
- **Fix-agent loop:** Same minimax CLI invocation, preserving the background (`&`) + watchdog timeout pattern

## Testing
- Verified CLI flags via `--help` for each binary (claude, codex, gemini, cursor-agent)
- Confirmed `ao spawn --help` shows no `--task` flag (only `--agent`, `--runtime`, `-p/--project`)
