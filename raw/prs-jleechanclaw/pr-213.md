## Raw Body
Bug: inc_cap() only reset window_start when count==0, but after check_cap() returns 0 due to expiry (count>0 but window old), inc_cap() reused stale window_start.

Fix: In inc_cap(), check if window has expired (same logic as check_cap) and reset window_start to now and count to 0 when it has.
