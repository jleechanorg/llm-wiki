## Changes

1. **New Merge Gate**: "Evidence review passed" - requires PASS verdict from CodeRabbit  or Codex evidence reviewer

2. **New  command** ():
   - Runs evidence reviewer on PR's evidence bundle
   - Checks structure, integrity, and measurement quality
   - Returns PASS/WARN/FAIL verdict

3. **Updated coderabbit command** ():
   - Now also requests CodeRabbit to run  on the evidence file
   - Provides direct GitHub link to evidence bundle

## Merge-Readiness Contract (updated)

| Gate | Source | Check |
|------|--------|-------|
| CI green | GitHub Actions | All required checks pass |
| No conflicts | GitHub API | mergeable == MERGEABLE |
| No serious comments | CodeRabbit, Copilot, Cursor | No REQUEST_CHANGES |
| **Evidence review passed** | CodeRabbit /er or Codex | PASS verdict on evidence bundle |
| OpenClaw approved | pr_reviewer.py LLM | ReviewDecision.approve |
