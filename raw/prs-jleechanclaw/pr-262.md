**`scripts/bug-hunt-daily.sh`**
| Bug | Fix |
|-----|-----|
| `timeout "$TIMEOUT_SECONDS" wait "$PID"` — `wait` is a shell builtin, `timeout` can't wrap it; broken on macOS (no `timeout` by default) | Replace with `kill -0` polling loop with absolute deadline |
| Claude output piped only to log file — `$OUTPUT_FILE` always empty, aggregation always counted 0 bugs | `tee -a "$OUTPUT_FILE"` so output goes to both log and findings file |
| `grep -c ... \|\| echo "0"` — `grep -c` exits non-zero on no-match, `echo "0"` fires and produces two-line string `"0\n0"`, crashing arithmetic | Use `\|\| true` + `${FILE_BUGS:-0}` |
