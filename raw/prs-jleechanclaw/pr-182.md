## Key behaviours introduced

**ao-backfill Pass 3** — every run now detects live agento sessions that:
1. Have been idle > N minutes (updatedAt/createdAt based)
2. Own an open PR with ≥1 unresolved review thread (GraphQL, not REST — excludes resolved threads)
3. Have a tmux pane sitting at a shell prompt

…and sends a reminder poke. Rate-limited to 1 poke per 60 min via `pokedAt` field written back to the session file.

**SOUL policy additions** — three new rules:
- Must read+quote exact source lines before declaring a CR comment stale
- Must run pre-exit checklist (CI, open inline threads, PR state) before exiting
- Must not exit while PR is open with unresolved items
