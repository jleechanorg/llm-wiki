## Summary

Adds a daily 9am launchd job that runs the harness analyzer to check for harness engineering compliance.

## Changes

- **launchd/ai.openclaw.schedule.harness-analyzer-9am.plist**: New launchd plist that runs at 9am Monday-Friday
- Uses existing `scripts/harness-analyzer.sh` which checks:
  - Core documentation files (CLAUDE.md, AGENTS.md, SOUL.md, etc.)
  - OpenClaw config validation
  - Agent skills presence and health
  - Agent Orchestrator configs
  - GitHub CI workflows
  - LaunchD job configurations
  - Test suite presence
  - Documentation freshness (90-day window)
  - Memory files
  - Skills health

## Schedule

- Runs at 9:00 AM Pacific time
- Monday through Friday (weekdays only)
- Logs output to `/Users/jleechan/.openclaw/logs/harness-analyzer.log`

## Testing

To test locally:
