## Background

Automated job that periodically checks `~/.openclaw` for uncommitted changes to git-tracked files, stages+commits them on a feature branch, and opens/updates a PR. Untracked files trigger a Slack warning but are never auto-added.

## Goals
- Detect uncommitted changes every 30 min via launchd
- Stage + commit only tracked files (never untracked)
- Open/update a PR on `auto/commit-pending` branch
- Warn via Slack when untracked files exist
- AO LLM fallback on failure

## Tenets
- **Safety first**: untracked files are NEVER auto-added
- **Idempotency**: overlap lock + timestamp guard prevent redundant commits
- **Durability**: git-tracked, installed via `install-openclaw-scheduled-jobs.sh`

## High-level description
- `scripts/commit-pending-changes.sh` (new): main job script
- `launchd/ai.openclaw.schedule.commit-pending.plist.template` (new): launchd plist
- `scripts/install-openclaw-scheduled-jobs.sh` (modified): registers new job
