## Summary
A Baldur's Gate 3 campaign focused on bug reproduction testing for the SYSTEM RESTORE mechanic. Features Nocturne Sosuke, a Level 5 Half-Elf Bard (College of the Abyssal Siren) with CHA 20, in the Shattered Sanctum. The campaign explores the "Nerf" narrative where a Level 6 Abyssal Sovereign is trapped in a Level 5 cage.

## Key Claims
- Campaign tests SYSTEM RESTORE functionality for restoring game state to a specific moment
- Character verification process confirms all stats and mechanics
- Level-up notification system causes issues (shows pending level-up when already at Level 5)
- Player uses "God Mode" to skip level-up notification and proceed with adventure
