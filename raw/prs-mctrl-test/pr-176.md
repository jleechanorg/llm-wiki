## Problem
Given an n x n binary matrix, find the shortest clear path from top-left to bottom-right where:
- All path cells contain 0 (not blocked)
- Path is 8-directionally adjacent (including diagonals)

## Solution Approach
- **BFS**: Standard breadth-first search since all edges have equal weight
- **8-directional movement**: Cardinal + diagonal directions
- **Visited set**: Track visited cells to avoid cycles
