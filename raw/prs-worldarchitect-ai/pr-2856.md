## Raw Body
The +3 gear reward from Alexiel's Cache now adapts to the player's class and build instead of being locked to Paladin/Fighter loadout.

Changes:
- Universal items (Cloak, Ring, Boots, Amulet) work for all classes
- Class-specific loadouts for all 12 D&D 5e classes
- Weapon forms adapt (longsword, greatsword, rapier, bow, staff, etc.)
- Armor adapts to class proficiency (plate, leather, robes, unarmored)
- Secondary items support class mechanics (shield, focus, quiver, etc.)
- Multi-class/custom build guidance for hybrid loadouts

Covers: Paladin, Fighter, Barbarian, Rogue, Ranger, Wizard, Artificer, Sorcerer, Warlock, Cleric, Druid, Bard, Monk, and multi-class builds.
