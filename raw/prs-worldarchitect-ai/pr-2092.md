## Summary

Adds two complementary browser automation skills for WorldArchitect.AI:

- **Playwright Skill**: Comprehensive E2E testing with visual regression, DOM optimization (50-80% token reduction), and RPG-specific helpers

## Key Features

### Playwright Skill
- 🎭 **RPG Helpers**: `createCampaign()`, `createCharacter()`, `performDiceRoll()`, `sendGameAction()`
- 🔬 **DOM Optimization**: FocusAgent/D2Snap-inspired 50-80% token reduction
- 🛡️ **Security-Aware**: Stagehand-inspired observe-before-execute pattern
- 🤖 **Modular Agents**: Planner → Generator → Healer architecture with auto-healing
- 📸 **Visual Regression**: Multi-viewport screenshot capture and baseline comparison

### Superpowers Chrome Skill
- ⚡ **Zero Dependencies**: Native Chrome DevTools Protocol
- 🚀 **Fast**: 2-4x faster than Playwright for simple tasks
- 💻 **CLI-Friendly**: 17 commands for interactive automation
- 🔄 **Persistent Sessions**: Reuses existing Chrome instances

### Unified `/browser` Command
- Smart tool selection based on task requirements
- Auto-detects: visual regression → Playwright, quick smoke → Chrome
- Subcommands: `smoke`, `campaign`, `visual`, `debug`, `e2e`
