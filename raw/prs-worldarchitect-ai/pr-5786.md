## Changes

### P1 - Serious
- **FD leak** (orchestration/task_dispatcher.py): Wrap `open()` in try/except with `Popen` so `log_fd` is closed on any exception
- **Command injection** (workflows/claude-processor.yml): Pass prompt/endpoint via env vars; use multiline GITHUB_OUTPUT for response
- **OAuth help_args** (jleechanorg_pr_monitor.py): Honor `skip_help` from `build_preflight_execution_args`; supply `--help` fallback only when Phase 1 runs with empty args

### Additional
- **env_set guard**: Validate `env_set` is dict before iterating (avoids crash on malformed profile)
- **--search filtering**: Handle `--search=value` and orphan value; remove from Codex exec args
- **Project paths**: stream_logs.sh, ralph_custom.md generic paths
- **coerce_cli_args**: Filter `None` from list input (was becoming literal `"None"`)
- **Placeholder tests**: Mark with `@pytest.mark.skip`
- **Dashboard**: Bind to 127.0.0.1
- **build-ci-runner-image**: Use `${{ github.repository_owner }}` for image tag
