# Remove Gemini 3 Pro Support

## Description
This PR removes all support and references to the `gemini-3-pro-preview` model, which has been deprecated. The system now defaults to `gemini-3-flash-preview` for all operations.

## Changes

### 🔧 Configuration & Constants
- Removed `gemini-3-pro-preview` from `ALLOWED_GEMINI_MODELS` and `GEMINI_MODEL_MAPPING`.
- Removed `GEMINI_PREMIUM_MODEL` constant and `GEMINI_3_ALLOWED_USERS` allowlist.
- Updated default model to `gemini-3-flash-preview` across the codebase.

### 💻 Frontend
- **Settings (V1 & V2)**: Removed logic that dynamically displayed the Pro option for allowlisted users.
- **UI**: Removed "Gemini 3 Pro Preview" radio button option.

### 🧪 Tests
- Updated `fake_llm.py` to use `gemini-3-flash-preview` as the default fake model.
- Updated unit and end-to-end tests to verify system behavior with Flash instead of Pro.
- Removed tests specifically targeting Pro-only inference logic.

### 📚 Documentation
- Updated `README.md`, `CLAUDE.md`, and project overview to reflect the removal.
- Added legacy deprecation warnings to roadmap documents.

## Verification
- verified `test_llm_model_selection.py` passes.
- verified `test_debug_mode_end2end.py` passes.
- verified `test_settings_api.py` passes.
