## Production Code Changes

### .github/actions/send-deploy-notification/action.yml
**What changed** - Added `pr_title` and `pr_description` inputs, updated email body template to display PR details
- **Before:** Only supported `pr_url` input, email didn't show PR title or description
- **Now:** Accepts `pr_title` and `pr_description` inputs, displays them in email body if available
- **Why:** Enables email recipients to see PR context without clicking through to GitHub

### .github/workflows/deploy-dev.yml
**What changed** - Added "Fetch PR details from commit" step, updated email notification steps to pass PR details
- **Before:** Only passed `pr_url` (which was empty for workflow_dispatch triggers)
- **Now:** Fetches PR info using `gh pr list --search` with commit SHA, passes URL/title/description to email action
- **Why:** Provides complete PR context in deployment emails even for non-PR-triggered deployments
