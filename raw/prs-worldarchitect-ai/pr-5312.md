## Raw Body
### Motivation

- Continue the agent-prompt-optimization work by extending deterministic prompt-size measurement beyond Phase 1 (God Mode) so subsequent roadmap phases can reuse the same utilities for validation.  
- Provide reusable helpers to resolve required prompt types/files and produce per-file character-counts for multiple agents to enable consistent TDD and measurement across phases.

### Description

- Generalize `testing_mcp/lib/prompt_file_utils.py` to support `GodModeAgent`, `DialogAgent`, and `CombatAgent`, add helpers `get_agent_required_prompt_types`, `get_agent_required_prompt_files`, `get_agent_prompt_size_by_file`, and centralize prompt-file resolution and measurement logic.  
- Refactor `get_agent_prompt_size()` to reuse the new helpers and remove duplicated resolution loops.  
- Export the new helpers from `testing_mcp/lib/__init__.py` so optimization tests can import them from the test utilities package.  
- Update `testing_mcp/test_god_mode_character_budget.py` to use prompt/path constants from `mvp_site.constants`, adopt the generalized helper APIs, and add measurable guardrails for Dialog/Combat prompt sets (measurability + combat > dialog baseline assertion).
