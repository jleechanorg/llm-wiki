## Summary

Adds a new `/reviewstatus` command that provides a comprehensive overview of outstanding pull requests, their CI status, and important review comments.

## Features

- **Recent PR Overview**: Shows PRs from the last 2 days with CI status
- **Review Comment Summary**: Highlights critical feedback from CodeRabbit, Copilot, and reviewers
- **Issue Prioritization**: Identifies which PRs need immediate attention
- **Clear Status Icons**: ✅ for passing CI, ❌ for failures

## Use Case

This command addresses the need to quickly check PR health and review status, as demonstrated when we manually checked multiple PRs for CI issues. Now this functionality is available as a simple slash command.

## Example Usage

```bash
/reviewstatus
```

Returns a formatted summary showing:
- Recent PRs with CI status
- Key issues to address
- Older open PRs that may need attention
