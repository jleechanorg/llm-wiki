## Summary
This PR increases sanctuary protection durations across all arc scales and introduces a comprehensive quest generation system to the living world event engine. The changes make sanctuary more meaningful for longer campaigns while adding structured quest offerings as a core part of scene events.

## Changes
- **Sanctuary Duration Adjustments:**
  - Medium mission: 5 → 8 turns
  - Major arc: 10 → 15 turns
  - Epic campaign arc: 20 → 30 turns
  - Updated all references in both instruction files and examples

- **Quest Generation System:**
  - Added 25% probability trigger for quest offerings on every living world turn
  - Applies to ALL scene events (companion, NPC, faction, messenger, discovery sources)
  - Defined quest types: rescue, investigation, elimination, delivery, diplomacy, exploration, gathering, protection, personal_mission
  - Created structured quest offering JSON format with quest_source, quest_title, quest_objective, estimated_turns, refusal_consequence, and reward_hint fields
  - Added companion quest subtypes: personal mission, resource gathering, location visit, NPC interaction, skill development, moral choice

- **Event System Enhancements:**
  - Added new scene event types: NPC Quest Offer, Faction Quest, Environmental Discovery
  - Updated scene event output format to include quest-specific fields when type == "quest_offered"
  - Clarified that 25% quest trigger is independent of event source (companions get 80% of events, but any event can trigger a quest)
  - Added comprehensive quest examples for each source type (companion, NPC, faction, discovery, messenger)

- **Documentation Improvements:**
  - Clarified quest probability applies to all events, not just companion events
  - Added "What Constitutes a Quest" definition (explicit request, multi-turn, clear objective, consequences, optional but meaningful)
  - Provided detailed quest source examples and narrative integration guidance
  - Updated non-companion event variants to note
