## Raw Body
Add comprehensive documentation explaining the distinction between:
- story_entry_count/turn_number: internal counter of ALL entries
- sequence_id: absolute position in story array
- user_scene_number: user-facing "Scene #X" (AI responses only)

Key insight: scene ≈ turn/2 because scenes only count AI responses, not user submissions. This prevents the "increment-by-2" display bug.
