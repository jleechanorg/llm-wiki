This PR improves the composition hook to properly parse and execute single slash commands embedded in natural language, addressing the core request: **"I want you to do /research and figure out X"**

**Enhanced Single Command Detection:**
- ✅ Detects commands like `/research` in natural language: *"I want you to do /research and figure out X"*

- ✅ Maintains backward compatibility - simple commands like `/help` still pass through unchanged
