## Raw Body
Added support for including living world background events in campaign downloads (scripts/download_campaign and UI export button).

Changes:
- Added _format_living_world_updates() to extract and format background events from debug_info
- Modified format_story_entry() to include living world updates when present in story entries
- Handles both debug_info.background_events and debug_info.world_events.background_events locations
- Formats events with status icons (⏳ pending, ✅ resolved, 🔍 discovered)
- Added unit tests for living world export formatting

Now when users download campaigns, they'll see sections like: 🌍 Living World Updates:
📜 Background Events:
⏳ The remaining Archdevils: Initiated a defensive mobilization [immediate, pending] ⏳ Orcus: Began observing the descent [immediate, pending]

Fixes user's request to include living world updates in campaign downloads.
