## Summary

Adds the **4-layer minimal repro testing protocol** for efficiently reproducing PR blockers.

**Extracted from**: PR #5506 (BYOK surgical extraction)  
**Rationale**: This testing methodology is independent of BYOK and should be available in main branch.

## Testing Ladder

1. **Unit** → `mvp_site/tests/` (seconds)
2. **End2end** → `mvp_site/tests/test_end2end/` (seconds to minutes)
3. **MCP API** → `testing_mcp/` (minutes)
4. **Browser** → `testing_ui/` (final escalation)

## Key Principle

**Stop climbing when blocker is reproduced** - don't waste time on slower layers.

## Usage

```bash
/4layer [blocker_description]
```

Or manually execute the ladder:
```bash
# Layer 1: Unit
./vpython -m pytest mvp_site/tests/test_[relevant].py -q

# Layer 2: End2end
./vpython -m pytest mvp_site/tests/test_end2end/test_[feature]_end2end.py -q

# Layer 3: MCP API
./vpython testing_mcp/[domain]/test_[feature]_real.py

# Layer 4: Browser (final escalation)
./vpython testing_ui/[domain]/test_[feature]_browser.py
```
