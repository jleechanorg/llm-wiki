## Problem Being Solved
Currently, when chaining slash commands like `/think /arch`, the composition system loses specific command implementations. For example, `/arch` should trigger 12+ thoughts and Gemini consultation, but instead just gets interpreted as "use architectural approach".

## Recommended Solution
Execution Graph Builder that:
1. Detects conflicts early and asks user for resolution
2. Attempts to combine compatible commands
3. Presents clear execution plan with parallel phases for approval
