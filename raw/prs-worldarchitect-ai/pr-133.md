## Changes

### 🎮 Gameplay Features
- **Time Tracking**: All actions now have realistic time costs (combat rounds, rests, travel)
- **Living World**: NPCs autonomously pursue their agendas in the background
- **Resource Management**: World resources deplete over time (food supplies, healing items)
- **Warning System**: Escalating alerts for time-sensitive events (subtle → clear → urgent)
- **Consequences**: Missed deadlines trigger permanent world changes
