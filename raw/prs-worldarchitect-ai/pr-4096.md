## Critical Bug Fix (P1) - Milliseconds Conversion

The Gemini SDK's `HttpOptions.timeout` expects **milliseconds**, not seconds:

- **Before (BROKEN)**: Passing `600` meant a 0.6 second timeout, causing all LLM inference requests to fail
- **After (FIXED)**: Now correctly converts to milliseconds (`600 * 1000 = 600000ms = 10 minutes`)

Caught by Codex during code review.

## Security Fix - XSS Prevention

Server-provided `error.message` was passed directly to `appendToStory()` which uses innerHTML. Now sanitizing with `sanitizeHtml()` before DOM injection.

Flagged by CodeRabbit as a direct XSS vector.
