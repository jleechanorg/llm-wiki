## Raw Body
## Summary
- Add `download_campaign.py` script that reuses existing export logic to download campaigns locally
- Supports email lookup, UID direct access, and batch downloads
- Multiple export formats (txt, docx, pdf)

## Features
- Download by email (auto-lookup Firebase UID) or UID directly
- Download specific campaign by ID or all campaigns for a user
- List campaigns without downloading (`--list`)
- Configurable output directory and format

## Usage
```bash
# Download by email
python scripts/download_campaign.py --email user@example.com --campaign-id abc123

# Download all campaigns
python scripts/download_campaign.py --email user@example.com --all

# List campaigns
python scripts/download_campaign.py --uid USER_UID --list

# Download as DOCX to specific directory
python scripts/download_campaign.py --email user@example.com --all --format docx --output-dir ~/Desktop
```
