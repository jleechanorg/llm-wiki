## Changes

1. **New file: `mvp_site/worldai_mcp_stdio.py`**
   - Reads newline-delimited JSON-RPC from stdin
   - Uses in-process `WorldAIToolsProxy` with `AuthContext` from `WORLDTOOLS_ACTOR_EMAIL`
   - Writes JSON-RPC responses to stdout with flush
   - Logs errors to stderr only

2. **Modified: `.claude-plugin/plugin.json`**
   - Added `worldai` stdio MCP server config with env vars

3. **Modified: `mvp_site/CLAUDE.md`**
   - Added documentation for worldai MCP tools

## Testing

```bash
echo '{"jsonrpc":"2.0","id":1,"method":"tools/list","params":{}}' | python -m mvp_site.worldai_mcp_stdio
```

Verified output includes local tools (diag_*, admin_*, ops_*) and attempts to fetch upstream tools.

## Known Limitations

- Upstream MCP server (`WORLDTOOLS_UPSTREAM_MCP_URL`) must be running for passthrough tools to work
- No authentication verification in this adapter (relies on upstream server)
