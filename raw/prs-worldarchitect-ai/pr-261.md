## New Commands
1. **`milestones N`** - Break current work into N specific milestones
   - Automatically updates scratchpad file after each milestone
   - Commits and pushes to GitHub to save state
   - Provides status updates between milestones

2. **`milestones suggest`** - Analyze task and suggest optimal milestone breakdown
   - Analyzes task complexity and dependencies
   - Suggests 3-7 milestones with rationale
   - Lists objectives, complexity, and success criteria
