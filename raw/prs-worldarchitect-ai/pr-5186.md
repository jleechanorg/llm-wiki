## Detailed changes vs `origin/main`
### Orchestration runtime and CLI
- Added `--no-worktree` wiring through:
  - `orchestration/live_mode.py`
  - `orchestration/orchestrate_unified.py`
  - `orchestration/task_dispatcher.py`
- Added no-worktree behavior in dispatcher agent creation:
  - run in current directory
  - preserve/checkout intended `existing_branch` when applicable
  - adjust completion instructions and reporting for no-worktree mode
- Improved worktree robustness:
  - detect invalid base refs and support interactive retry with alternate base branch
  - cache successful interactive base ref for subsequent agents
  - retry path-related worktree failures in `/tmp` mktemp-style fallback directories
  - cleanup temp fallback dirs on failed retry
- Added/updated helper methods for:
  - path-related error detection
  - safe cleanup
  - branch existence checks
  - current branch lookup
  - tmp worktree path generation
