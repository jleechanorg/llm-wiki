## Details
This PR implements several important changes to the character creation system:

### 1. Calibration Checkbox Removal
- Removed from main campaign form (index.html)
- Removed from app.js form initialization
- Removed from campaign wizard (card, checkbox references, and arrays)
- Cleaned up all references throughout the codebase

### 2. Character Creation Relocation
- **Moved from**: `narrative_system_instruction.md` 
- **Moved to**: `mechanics_system_instruction.md`
- Character creation now only triggers when the mechanics checkbox is selected
- If mechanics is not selected, campaigns start directly without character creation

### 3. More Creative Freedom
The third character creation option has been updated:
- **Old**: "Use a custom character system - If you've specified custom rules or classes"
- **New**: "Create a custom character - Design your own unique character concept with custom class/abilities"
- Custom classes/races are now welcomed if they fit the campaign world
- Focus on balance and appropriateness rather than restriction
