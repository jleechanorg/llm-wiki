## Features
- **Usage checking**: `/usage` command shows current API quota
- **Threshold alerts**: Notifications at 75%, 50%, and 25% remaining
- **Notification types**: Balloon notifications (info/warning) and popup dialogs (critical)
- **Monitoring modes**: `--with-api` (display only) and `--monitor` (with alerts)
- **Test script**: Direct API testing with formatted rate limit headers

## Alert System
- ≤75% remaining: Balloon notification (info)
- ≤50% remaining: Balloon notification (warning) 
- ≤25% remaining: Popup dialog (critical)
