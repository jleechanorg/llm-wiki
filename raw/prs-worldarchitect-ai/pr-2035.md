## Summary

This PR adds a script to test API keys against the `jleechanorg/ai_universe` repository.

## Changes

- **New script**: `scripts/test_ai_universe_api_keys.sh`
  - Loads API keys from `~/.bashrc`
  - Clones the ai_universe repository to `/tmp` for analysis
  - Tests API keys against actual API endpoints:
    - GitHub (repo access)
    - Gemini (model listing)
    - OpenAI (chat completion)
    - Anthropic (messages API)
    - Perplexity (chat completion)
    - OpenRouter (chat completion)
  - Provides colored output and summary
  - Auto-cleans up temporary files

- **Documentation**: `.claude/skills/test-api-keys-ai-universe.md`
  - Usage instructions
  - Troubleshooting guide
  - Integration examples

## Usage

```bash
bash scripts/test_ai_universe_api_keys.sh
```

## Testing

The script has been tested and all 6 API keys pass validation:
- ✅ GitHub token
- ✅ Gemini API key
- ✅ OpenAI API key
- ✅ Anthropic API key
- ✅ Perplexity API key
- ✅ OpenRouter API key

## Benefits

- Validates API keys are working before use
- Helps debug authentication issues
- Can be integrated into CI/CD pipelines
- Provides clear feedback on key validity
