## Summary

- Adds **Spell Slot Validation Protocol** to `game_state_instruction.md` to prevent spellcasting without available slots
- Adds `test_spell_slot_validation_real_api.py` for regression testing spell slot validation

## Root Cause

LLM was allowing spells to be cast without available spell slots. Evidence from campaign transcript showed player cast Teleport (7th-level) without any 7th-level slots:
> "wtf pause the story. I had no spellslots you shouldn't have allowed it"

Multi-model analysis (Cerebras, Grok-4-Fast, GPT-5, Perplexity Sonar Pro) confirmed the issue: spell slots were tracked but never validated.

## Changes

### Spell Slot Validation Protocol (lines 719-801)
- MANDATORY check before any spell cast
- No auto-upcasting without player consent (must ASK first)
- Explicit rejection when no slots available
- Proper upcast choice offering when higher slots exist
- Exceptions for cantrips, rituals, innate spellcasting
