## Raw Body
Adds three strict anti-hang rules to GEMINI.md and the shell-safety workflow:

1. **NEVER chain commands** — No `&&`, `||`, or `;` in `run_command` calls. Git writes push/fetch progress to stderr; chaining causes phantom hangs.
2. **Git push 3-min timeout** — Always run `git push` with `WaitMsBeforeAsync=5000` then `command_status(WaitDurationSeconds=180)`. Terminate-and-diagnose if still running.
3. **Paranoid investigation** — Any stuck command must be assumed to be the agent's fault. Diagnose (reachability, lock files, credential prompts) before retrying.
