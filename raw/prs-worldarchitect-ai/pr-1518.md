## Problem
The `agent_monitor.py` script was calling `claude {prompt}` without specifying a model, causing it to default to whatever the system default model was set to (Opus 4 in this case). Since the agent monitor runs every 2 minutes and auto-restarts stuck agents, this led to thousands of expensive Opus API calls.

## Solution
Modified line 324 in `orchestration/agent_monitor.py`:
```bash
# Before:
claude {enhanced_prompt}

# After:  
claude --model sonnet {enhanced_prompt}
```

## Cost Impact
- **Prevents**: Accidental expensive model usage during automated operations
- **Observed**: Aug 31st usage pattern showed 8,164+ Opus API calls costing $823+
- **Future**: Agent restarts will now use cost-effective Sonnet model
