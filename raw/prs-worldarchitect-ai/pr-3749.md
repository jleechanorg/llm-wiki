# Add Gemini Support to MCP Server Installer

This PR extends the `install_mcp_servers.sh` script to support the Gemini CLI, in addition to Claude and Codex.

## Changes

- **scripts/install_mcp_servers.sh**:
  - Added `gemini` target to support installing MCP servers for Gemini.
  - Added `all` target to install servers for all supported products (Claude, Codex, Gemini) in one go.
  - Updated usage instructions and argument parsing.

- **scripts/mcp_common.sh**:
  - Updated test configuration logic to handle `gemini` by creating a `gemini.json` config file during tests.

## Usage

```bash
./scripts/install_mcp_servers.sh gemini    # Install for Gemini only
./scripts/install_mcp_servers.sh all       # Install for Claude, Codex, and Gemini
```
