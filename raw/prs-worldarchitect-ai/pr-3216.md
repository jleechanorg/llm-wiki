## Raw Body
The CSS theme system uses CSS variables scoped to [data-theme='...'] selectors, but the HTML element had no data-theme attribute until JavaScript initialized. On mobile devices with slower JS execution, this caused CSS variables to be undefined during initial render, resulting in invisible content (text color matching background).

Fix: Add data-theme="light" as the default on <html> element. JavaScript (theme-manager.js) will still override with user preference.
