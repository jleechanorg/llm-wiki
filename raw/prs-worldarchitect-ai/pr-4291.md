## Rationale

### Why Remove Scrubbing?

1. **Firebase API keys ARE safe to expose publicly**
   - Not secret credentials - designed for client-side use
   - Protected by domain restrictions in Firebase Console, not by secrecy
   - Standard practice for web/mobile apps

2. **AI Universe is a shared service**
   - Users should use it as-is with the actual credentials
   - No need to make credentials generic - they're meant to be shared

3. **Simpler user experience**
   - Exported commands work immediately
   - No confusion about what credentials to use
   - Follows Firebase's official guidance

### From Firebase Documentation

> "API keys for Firebase are different from typical API secrets... It is safe to embed them in code or checked-in config files."
