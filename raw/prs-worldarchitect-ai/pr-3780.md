## Raw Body
Comprehensive design for a Small Language Model (SLM) resource validation gateway that enforces D&D 5e action economy before Gemini processes actions.

Key components:
- SLM validates resources (spell slots, action economy, class features)
- Python deducts costs deterministically (no LLM drift)
- Gemini handles narrative and complex action resolution
- Two-tier alternative suggestions (SLM fast, Gemini fallback)
- Faction RAG for historical queries across 100+ turns

Architecture principle: "SLM Validates, Python Deducts, Gemini Narrates"

This addresses Gemini's weakness in enforcing resource constraints while maintaining its narrative strengths.
