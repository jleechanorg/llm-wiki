## Root Cause
The campaign list HTML structure uses Bootstrap flexbox classes (`d-flex justify-content-between`) but lacks sufficient CSS specificity to handle width constraints consistently across different browser/server environments.

## Fix Details
Added CSS rules in `style.css`:
- Force `flex-wrap: nowrap` to prevent wrapping
- Set `white-space: nowrap` for timestamp text
- Use `flex-shrink: 0` for Edit button and timestamp
- Add proper overflow handling for campaign titles
