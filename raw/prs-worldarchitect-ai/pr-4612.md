## Raw Body
### Motivation
- Decommission the GitHub-side dispatcher for the Claude bot to disable automatic `/claude` comment capture and reduce surface area for the feature. 
- Align documentation and local startup logic with the removed dispatcher so users are not misled by stale instructions. 

### Description
- Remove the dispatcher workflow file `/.github/workflows/slash-dispatch.yml` to stop capturing `/claude` issue comments. 
- Update `claude-bot-commands/docs/OVERVIEW.md` to mark the system as disabled and remove references to the dispatch workflow. 
- Update `claude-bot-commands/docs/SETUP_GUIDE.md` to note the dispatch workflow removal and adjust the workflow list and usage steps accordingly. 
- Remove the startup script dependency check for the deleted workflow in `start-claude-bot.sh` so the local server can start without the dispatcher file. 
- Clean up a related entry in `roadmap/scratchpad_dev1753495823_claude_slash_commands_setup.md` to remove the dispatcher from the generated file list.
