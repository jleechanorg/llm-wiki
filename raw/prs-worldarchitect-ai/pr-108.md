## Problem
The AI sometimes wraps JSON state updates in various markdown formatting patterns that break JSON parsing:
- Bold formatting: `**{...}**`
- Mixed formatting: `**```json{...}```**`
- Commentary: `*Here's the update:* {JSON} *End of update*`
- HTML tags: `<strong>{...}</strong>`

## Solution
Added `_clean_markdown_from_json()` function that:
1. **Removes HTML tags** (`<strong>`, `</strong>`)
2. **Filters commentary lines** (lines with `*update*`, pure markdown decorations)
3. **Handles mixed patterns** (bold around code blocks)
4. **Cleans code block markers** (````json`, `````)
5. **Strips formatting** (bold `**`, italic `*`)
