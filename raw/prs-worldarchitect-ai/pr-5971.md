## Raw Body
Reverts #5962.

PR #5962 changed critical CSS from render-blocking to `rel="preload"` with onload swap, which broke the fantasy theme—styles were loading asynchronously and override order was lost, causing Edit buttons, filter dropdowns, stepper, and form controls to show default/light styling on dark backgrounds.

This full revert restores:
- Critical CSS (loading-messages, planning-blocks, inline-editor, avatar) as blocking
- Non-critical CSS via media="print" + data-async-css (theme-bootstrap.js)
