## Problem
The `mapfile` command is a bash 4.0+ feature not available in all environments (e.g., older macOS bash 3.x). This caused:
- `EXPORTABLE_COMPONENTS` array to be empty
- 0 components exported to `~/.claude/`
- Missing slash commands like `processmsg.md`

## Solution
Changed from:
```bash
mapfile -t EXPORTABLE_COMPONENTS < <(python3 -c "...")
```

To:
```bash
EXPORTABLE_COMPONENTS=($(python3 -c "..."))
```

This portable syntax works across bash 3.x and 4.x versions.
