## 📋 Summary

This PR introduces a specialized **DialogAgent** for conversation-focused NPC interactions, reducing token overhead by ~44% for dialog scenes while improving character voice and personality. It includes comprehensive testing, documentation, and multiple bug fixes from the /copilot workflow.

## ✨ Key Features

### DialogAgent Implementation
- **New Agent**: `DialogAgent` class with conversation-focused prompt and routing
- **Priority 7**: Positioned between RewardsAgent (6) and StoryModeAgent (8)
- **Keyword Triggers**: Detects dialog intent via keywords ("talk to", "speak with", "ask about", etc.)
- **Exclusion Patterns**: Avoids false positives for combat/action keywords
- **Token Efficiency**: ~44% reduction in token usage vs StoryModeAgent for dialog scenes

### Intent Classifier Enhancements
- Expanded dialog trigger patterns (D&D social skills, speaking verbs, social interactions)
- Added meeting/council and check-in patterns
- Exclusion logic to prevent combat/action misclassification as dialog

### Prompt Engineering
- New `dialog_system_instruction.md` prompt (411 lines)
- Focused on character voice, personality, conversation flow
- Optimized for dialog-specific context without full story overhead
