## 🎮 MCP Tools Included

The package exposes **8 specialized MCP tools**:

1. **`create_campaign`** - Create new D&D campaigns
2. **`get_campaign_state`** - Retrieve current campaign state
3. **`process_action`** - Process player actions with AI response
4. **`update_campaign`** - Update campaign settings
5. **`export_campaign`** - Export campaigns to multiple formats
6. **`get_user_settings`** - User preference management
7. **`get_campaigns_list`** - List user campaigns
8. **`update_user_settings`** - Modify user settings
