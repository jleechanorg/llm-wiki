## Issue Summary

The LLM was not automatically awarding XP/items/abilities for significant player actions. Players had to manually request rewards via God Mode.

**Evidence from Dragon Knight game log:**
```
God Mode:
Why didnt you give me exp or powers for consuming titan casket?

Game Master:
The user is correct; the ingestion of the "Assiah-Spine-Alpha" engine is a
milestone of cosmic proportions that was not formally quantified...
```

## Root Cause

The prompts had specific reward protocols for combat and social victories, but no **generic catch-all** telling the LLM to think like a tabletop DM and award rewards for ANY significant action.
