## Raw Body
Adds 8 optional CLI arguments to orchestrate_unified.py:
- --context: Inject markdown file into agent prompt as pre-computed context
- --branch: Force checkout of specific branch (prevents new branch creation)
- --pr: Existing PR number to update (prevents new PR creation)
- --mcp-agent: Pre-fill agent name for MCP Mail registration
- --bead: Pre-fill bead ID for tracking
- --validate: Semantic validation command to run after completion
- --no-new-pr: Hard block on PR creation
- --no-new-branch: Hard block on branch creation

All arguments are optional to maintain backward compatibility. The wrapper script (orchestrate.py) already passes through all arguments correctly.
