Addresses critical combat balance issues where the AI was giving enemies unrealistically low HP and hallucinating damage dice. Changes:

- Add mandatory stat block display for bosses/named enemies at combat start
- Add CR-to-HP reference table (CR 12 = 221+ HP, not 21 HP)
- Add Boss vs Minion classification system
- Add damage calculation validation (max Sneak Attack = 10d6/20d6 crit)
- Add combat_state schema for tracking enemy HP accurately
- Require bosses to use defensive abilities (Legendary Resistance, Uncanny Dodge)
- Add "No Paper Enemies" rule - CR must match HP

This prevents scenarios like:
- CR 12 Paladins dying to 21 damage
- 40d6 Sneak Attack (impossible - max is 20d6)
- Level 22 Generals dying without using any reactions
