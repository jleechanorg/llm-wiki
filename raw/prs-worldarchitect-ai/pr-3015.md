## 🎮 Divine Tier Gameplay: The Divine Deception Protocol

When a player ascends to divine tier, they become a **hidden god** operating under a multi-layered deception protocol. The core tension: using divine power risks detection by the existing Pantheon.

### The Tri-Layer Simulation

Players operate through three identity layers:

| Layer | Name | Description | Risk Level |
|-------|------|-------------|------------|
| **0** | The Mask | Mortal identity with normal HP/AC/slots. Physics applies. | HIGH - Any divine power generates massive Dissonance |
| **1** | The Persona | Fabricated deity identity (Tier 2+) with own portfolio/church | MODERATE - Power must match persona's brand |
| **2** | The Source | True cosmic self. Physics optional. Hidden unless "Dropping Mask" | N/A - Never revealed |

### Divine Dissonance System (0-100%)

The primary detection mechanic. Actions that violate mortal expectations generate Dissonance:

| Trigger | Dissonance Gain |
|---------|-----------------|
| Metric Mismatch (100+ dmg as mortal) | +15-25% |
| Probability Violation (impossible success) | +10-20% |
| Direct Observation by deity | +20-30% |
| Forced Manifestation (survive lethal) | +10-15% |

**Thresholds:**
- **0-20% Safe**: Strange luck dismissed
- **21-50% Suspicion**: Detection gods begin scrying, +2 DC to Social Stealth
- **51-80% Investigation**: Divine agents dispatched, combat tests
- **81-100% Exposure**: Mask fractures, Pantheon aware, cult forms
