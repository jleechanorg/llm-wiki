## Summary
Fixed a bug where World Assiah content was being included even when the 'Include World of Dragon Knight' checkbox was unchecked in the campaign wizard.

## Bug Details
- **Issue**: World content was always included regardless of checkbox state
- **Root Cause**: The `collectFormData()` function was using the `isDragonKnight` checkbox value instead of checking the actual `wizard-default-world` checkbox
- **Fix**: Now properly checks `document.getElementById('wizard-default-world')?.checked`

## Changes
- Modified `mvp_site/static/js/campaign-wizard.js` 
- Added proper checkbox variable `useDefaultWorld` to check the correct element
- Updated the customOptions array to use the correct checkbox value
