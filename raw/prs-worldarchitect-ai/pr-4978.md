## Summary
Introduces a comprehensive ability score progression system for divine entities (player characters, NPCs, and gods) at epic levels (26+). This system provides automatic scaling and ASI feat mechanics that apply consistently across all divine ranks from Quasi-Deity through Greater Deity.

## Changes
- Added "Ability Score Progression (Post-Level 25)" section to divine leverage system documentation
- Implemented automatic +2 ability score increases per level for all divine entities beyond level 25
- Defined ASI Feat mechanics granting +6 to a chosen ability score (in addition to automatic increases)
- Created ability score calculation formula: `Base + ((Current Level - 25) × 2) + ASI bonuses`
- Established divine ability score caps by rank (ranging from 40 for Quasi-Deity to unlimited for Greater Deity)
- Added enforcement guidelines for NPC divine ability score generation with detailed example format
- Included worked examples for both player characters and divine NPCs (Mystra, Helm)

## Benefits
- **Consistent scaling:** All divine entities follow the same progression rules, eliminating ambiguity
- **Clear progression path:** Players understand how their abilities grow through epic levels
- **NPC consistency:** Game masters have a standardized system for generating divine NPCs
- **Flexibility:** ASI feats allow divine entities to specialize while maintaining baseline growth
- **Scalability:** System supports gods of any level without hard caps (except mortals at 30)
- **Transparency:** Detailed examples and formulas make calculations straightforward
