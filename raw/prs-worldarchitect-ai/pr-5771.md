## Raw Body
Simplifies the Git Operations & Remote Tracking section to 3 clear, non-negotiable rules:

1. **Always Push** — after finishing any unit of work, push to the PR remote branch, no exceptions
2. **Always Infer Remote Branch via `gh`** — use `gh pr view --json headRefName` to get the exact remote branch; never guess
3. **Always Report Remote Commit URL** — end every task with the full `https://github.com/…/commit/<sha>` URL; never just say "pushed"
