**Problem:** Initial implementation failed with "Invalid target: Claude" error

**Root Cause:**
- Wrapped `mcp_common.sh` sourcing in function
- `mcp_common.sh` calls `safe_exit` which only returned from function
- Script continued to case statement's default branch

**Solution:**
- Refactored to source `mcp_common.sh` at top level (matches old pattern)
- Uses `exit $?` to propagate exit code properly
- Changed shebang to `#!/usr/bin/env bash` for bash 5.x
