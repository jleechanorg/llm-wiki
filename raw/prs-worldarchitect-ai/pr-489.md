## Enhanced Character Co-Design System with Planning Block Fixes

This PR implements a comprehensive character co-design system with three collaboration options and fixes critical planning block display issues.

### 🎯 Key Features

#### 1. Three Character Creation Paths
- **AIGenerated** - AI creates complete D&D character based on campaign/prompt
- **StandardDND** - Traditional step-by-step D&D character creation
- **CustomClass** - Collaborative design of unique character concepts

#### 2. Planning Block Architecture Fixes
- 🐛 **Fixed**: Planning blocks not displaying in UI during character creation
- Implemented JSON-first approach (structured_response.planning_block)
- Removed skip logic that prevented planning blocks during character creation
- Enhanced validation to ensure planning blocks are always populated

#### 3. Character Creation Improvements
- Strong anti-narrative warnings (NO SCENES, NO NARRATIVE during creation)
- Mandatory character approval step before story begins
- Campaign summary display at creation start
- Proper camelCase choice IDs throughout
