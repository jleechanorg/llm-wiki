## Problem
The current main branch uses `PostCommit` which is not a valid hook type in Claude Code CLI. This causes the auto-sync functionality to fail silently.

## Solution
Changed to `PostToolUse` with a matcher for git commit operations. This maintains the same functionality while using a valid hook type.

## Valid Hook Types
According to Claude Code documentation, the valid hook types are:
- PreToolUse
- PostToolUse
- Notification
- UserPromptSubmit
- SessionStart
- Stop
- SubagentStop
- PreCompact

## Testing
The hook has been tested and successfully triggers after git commit operations, automatically pushing unpushed commits to the remote repository.
