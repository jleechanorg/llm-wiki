## Raw Body
Planning block validation and sanitization were silently dropping the pros, cons, and confidence fields from choice options. This caused the LLM's success percentages and analysis to be lost when users asked "give me success % chances and pros/cons" in Think Mode.

The fix adds preservation for these fields in both:
- _validate_planning_block_json(): Now validates and keeps pros/cons/confidence
- _sanitize_planning_block_content(): Now sanitizes and preserves these fields

Includes regression tests to prevent future recurrence.
