## Summary
Updated divine ascension and leverage system prompts to make the "Overseer" (supreme cosmic authority) optional rather than mandatory. This allows the system to work with pantheons that are self-regulating or lack a single supreme authority, improving flexibility across different fantasy settings.

## Changes
- Modified `divine_ascension_ceremony.md` to clarify that not all settings require an Overseer, with note that some pantheons are self-regulating
- Updated `divine_leverage_system.md` table to mark Overseer as "(Optional)" and provide alternative examples
- Added "Settings Without an Overseer" section explaining how divine existence remains hidden through power balance instead
- Updated all references to "The Overseer" throughout the document to include conditional language ("if applicable", "if one exists", etc.)
- Modified Apex Attention mechanics to describe responses from either a supreme authority OR collective pantheon action
- Updated spell descriptions (Mavin's Worldweave, Demiplane Creation, Deity Usurpation) to account for both scenarios
- Added new "Cosmic Intervention Options" section with separate mechanics for settings with/without Overseer
- Expanded Setting-Specific Examples to include:
  - Warhammer 40K (NO OVERSEER - competing equals)
  - Greek/Norse Mythology (NO OVERSEER - gods can be overthrown)
  - Clarified Star Wars as debatable
  - Marked D&D and Superhero settings as HAS OVERSEER
- Added explanatory notes for settings without supreme authority

## Benefits
- **Increased Setting Compatibility:** System now works with mythologies like Greek/Norse pantheons where gods are equals
- **Better Warhammer 40K Support:** Explicitly handles settings with competing powers but no arbiter
- **Clearer Guidance:** GMs understand upfront whether their setting needs an Overseer concept
- **Mechanical Flexibility:** Divine mechanics adapt gracefully to different pantheon structures
- **Improved Examples:** More diverse setting examples help GM
