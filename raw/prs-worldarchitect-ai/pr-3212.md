## Raw Body
Add focused test to validate LLM properly handles mechanically impossible player requests in D&D 5e. Test scenarios cover:

- Casting spells with no spell slots at any level
- Using Ki abilities with 0 Ki points
- Casting with upcast option available (clarify behavior)
- Using Bardic Inspiration with 0 uses
- Casting 7th level spell at level 5 (beyond capabilities)

The test validates the LLM either:
1. REFUSES with narrative explanation
2. CLARIFIES by offering alternatives (e.g., upcast)

This leverages the existing Class Resource Validation Protocol in game_state_instruction.md which already documents the refuse/clarify capability for spell slots and class resources.
