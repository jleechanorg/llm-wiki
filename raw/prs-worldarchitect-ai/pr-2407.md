## Raw Body
Added support for two new cost-efficient Cerebras models:

**Llama 3.1 8B:**
- Cerebras direct: llama3.1-8b ($0.10/M tokens)
- OpenRouter: meta-llama/llama-3.1-8b-instruct
- Context: 128K window, 32K max output
- Performance: ~2200 tokens/s (cheapest option)

**GPT-OSS-120B:**
- Cerebras direct: gpt-oss-120b ($0.35/M in, $0.75/M out)
- OpenRouter: openai/gpt-oss-120b
- Context: 131K window, 40K max output
- Performance: ~3000 tokens/s (budget reasoning model)

Both models are now available in:
- ALLOWED_CEREBRAS_MODELS (direct API)
- ALLOWED_OPENROUTER_MODELS (via OpenRouter)
- MODEL_CONTEXT_WINDOW_TOKENS (context limits)
- MODEL_MAX_OUTPUT_TOKENS (output limits)

Pricing and availability verified via web search on 2025-12-11.

Sources:
- Cerebras Pricing: https://www.cerebras.ai/pricing
- Cerebras Docs: https://inference-docs.cerebras.ai/models
- OpenRouter: https://openrouter.ai/provider/cerebras
