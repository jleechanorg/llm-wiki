## Summary

Created comprehensive browser automation suite to automate "Update branch" clicks on Github Mention tasks in Codex. Successfully tested and processed 13/16 tasks automatically.

## What's New

### 1. Codex GitHub Mentions Automation (codex_github_mentions.py)
- Connects to existing Chrome via CDP (Chrome DevTools Protocol)
- Automatically finds all "Github Mention:" tasks in Codex
- Opens each task and clicks "Update branch" button
- Returns to task list and continues with next task
- **Tested successfully:** 13/16 tasks automated (3 didn't have Update branch button)

### 2. Oracle CLI Tool (oracle_cli.py)
- Ask GPT-5 Pro questions via browser automation
- Single question mode
- Interactive mode for multiple questions
- No API key required - uses browser session

### 3. Chrome Debug Launcher (start_chrome_debug.sh)
- Starts Chrome with remote debugging enabled on port 9222
- Proper CDP configuration for Playwright connection
- Cross-platform support (macOS and Linux)
- Opens ChatGPT by default for easy login

## Technical Details

**Browser Automation Approach:**
- Uses Chrome DevTools Protocol (CDP) instead of WebDriver
- Connects to existing Chrome instance to preserve login state
- Non-headless mode to avoid detection
- Reuses user's actual Chrome profile

**Dependencies:**
- Python 3.11+
- Playwright
- Chrome/Chromium browser
