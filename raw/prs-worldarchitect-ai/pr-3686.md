### 1. coverage.sh Fix
Tests use `from mvp_site import ...` but the script runs from `mvp_site/` without setting PYTHONPATH. Added:

### 2. GitHub Actions Coverage Workflow
New `.github/workflows/coverage.yml` using [py-cov-action](https://github.com/py-cov-action/python-coverage-comment-action):
- Posts coverage delta comments on PRs
- Annotates uncovered lines directly in PR diffs
- Shows per-file coverage changes
- Generates markdown summary in Actions tab
- Uploads coverage reports as artifacts
