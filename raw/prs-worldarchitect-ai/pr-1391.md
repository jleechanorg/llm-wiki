## Summary

Added iOS Simulator MCP server configuration to the Claude MCP installation script to enable iOS app development, testing, and automation capabilities.

## Changes Made

- **New iOS Simulator MCP Server**: Added `ios-simulator-mcp` from joshuayoes/ios-simulator-mcp repository
- **Platform Detection**: Added macOS platform detection and Xcode dependency checks
- **Custom Installation**: Implemented git-based installation since package isn't on npm registry  
- **Batch Integration**: Added to Batch 3 for development tools grouping
- **Error Handling**: Comprehensive error handling for platform compatibility and dependencies

## Technical Details

### Server Configuration
- **Package**: `ios-simulator-mcp` (GitHub: joshuayoes/ios-simulator-mcp)
- **Installation Method**: Git clone + npm install to `~/.mcp/servers/ios-simulator-mcp`
- **Platform**: macOS only (iOS Simulator requirement)
- **Dependencies**: Xcode Command Line Tools, xcrun, iOS Simulator

### Features Added
- iOS app control and automation
- iOS Simulator management (boot, shutdown, device switching)
- Screenshot capture for testing validation
- App installation and launching
- Comprehensive logging and error reporting

### Installation Flow
1. Platform validation (macOS required)
2. Dependency checks (xcrun, iOS Simulator availability)
3. Git clone from GitHub repository
4. NPM dependency installation
5. Move to permanent location (`~/.mcp/servers/`)
6. Register with Claude MCP configuration
