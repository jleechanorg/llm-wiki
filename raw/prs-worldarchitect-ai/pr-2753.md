## Raw Body
Reproduces bug where LLM over-narrativizes count queries:
- User asks "how many soldiers/spies do I have?"
- LLM responds with prose that buries or omits actual numbers
- User frustrated: "how come you never answer how many?"

Test validates that responses to count queries contain:
- Explicit numbers in first 2 sentences
- Minimum number of numeric values
- Expected counts for troops/spies/companions
