## Summary
Fixed three critical bugs in claude_mcp.sh that caused MCP server installation failures for grok-mcp, ddg-search, and render servers.

## Bugs Fixed

### 1. grok-mcp & ddg-search: CLI Argument Order Bug
Location: claude_mcp.sh:609, 611

Root Cause: Environment flags were placed BEFORE the server name in claude mcp add command, violating the CLI specification.

Error Message: error: missing required argument 'name'

Fix: Moved environment flags to END of command per CLI spec

### 2. render: Invalid JSON Input Method
Location: claude_mcp.sh:816-817

Root Cause: Using - (stdin) with claude mcp add-json, but the command expects JSON as a parameter, not from stdin.

Error Message: Invalid configuration: : Invalid input

Fix: Build JSON config string directly and pass as parameter instead of heredoc stdin
