## Summary
Fixes issue where fixcomment did nothing because race condition check prevented re-run when agent failed silently.

## Problem
- Queued comment was posted (agent dispatched)
- Agent failed silently (no completion marker)
- Race condition check sees queued comment and skips forever
- Fixcomment becomes permanently stuck

## Solution
Add timeout check: if queued comment is older than 1 hour with no completion marker, allow re-run (agent likely failed).

## Changes
- Added `_get_fix_comment_queued_info()` method to calculate queued comment age
- Updated race condition check to allow stale re-runs (>1 hour old)
- Prevents permanent lock while maintaining race condition protection

## Testing
- Tested locally on PR #3617 (has stale queued comment 6.5 hours old)
- Fix allows re-run after timeout threshold
