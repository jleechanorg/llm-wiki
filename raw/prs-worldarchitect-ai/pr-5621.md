- `.github/workflows/test.yml` - Added concurrency group
- `.github/workflows/presubmit.yml` - Added concurrency group

- `.github/workflows/coverage.yml` - Added concurrency group
- `.github/workflows/test-deployment.yml` - Added concurrency group

**What changed**: Added `concurrency` blocks to top 5 cost workflows to cancel in-progress runs when new commits are pushed to the same branch.

**Why**: Prevents wasted compute on outdated commits - when you push multiple commits quickly, only the latest runs to completion.
