## How It Works

### Default Log Directory
- Logs automatically go to ~/Library/Logs/worldarchitect-automation/
- Directory created automatically if it doesn't exist
- No need to specify --log-dir parameter

### Parallel Processing (Default)
1. Collect all eligible PRs up to --max-prs limit
2. Pre-check eligibility (conflicts, failing checks, etc.)
3. Dispatch all agents concurrently instead of one-by-one
4. Agents run asynchronously in tmux sessions

**Why parallel by default:**
- Original: Check PR 1 → dispatch → check PR 2 → dispatch → ... (sequential)
- New: Check all PRs → dispatch all at once (parallel, much faster)
- Automation should be efficient by default
