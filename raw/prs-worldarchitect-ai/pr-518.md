## Summary
Adds a new command line flag to claude_start.sh that allows users to start fresh conversations without continuing from previous context.

## Features
- `--clean` or `-c` flag forces a fresh conversation
- Shows clear message: "🧹 Starting fresh conversation (--clean flag detected)"
- Prevents using `--continue` flag when clean mode is requested
- Preserves all other functionality of the script

## Usage
```bash
# Normal mode (continues existing conversations if found)
./claude_start.sh
claudep

# Clean mode (always starts fresh)
./claude_start.sh --clean
./claude_start.sh -c
claudep -c
claudep --clean
```

## Implementation
- Added argument parsing at the beginning of the script
- Sets `FORCE_CLEAN` flag when --clean or -c is detected
- Skips conversation continuation logic when clean flag is set
- Passes remaining arguments to claude command unchanged

Tested locally - correctly starts fresh conversations when flag is used.
