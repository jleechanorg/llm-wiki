  - chrome-superpower, sequential-thinking, context7, gemini-cli-mcp
  - ddg-search, grok-mcp, perplexity-ask, worldarchitect, beads

## Rationale
`.claude/settings.json` is for:
- Behavioral configuration
- Permissions and security settings
- Hooks and automation
- Environment variables

`.claude/settings.json` is **NOT** for:
- MCP server configurations (use `~/.claude.json` instead)

## Impact
- No functional changes - MCP servers should be configured per-user
- Removes confusion about proper configuration file location
- Aligns with official Claude CLI documentation
