## Summary
This PR introduces deterministic shell scripts to replace Claude Code's unreliable slash command interpretation. Instead of Claude interpreting commands differently each time, it now executes tested, consistent scripts.

## Changes
- Created 4 core workflow scripts (integrate, push, branch-cleanup, new-branch)
- Added 5 test command scripts (test-ui, test-ui-full, test-http, test-http-full, test-integration)
- Implemented --help flags for all scripts for discoverability
- Added comprehensive validation test suite
- Updated SLASH_COMMANDS_README.md with complete documentation

## Benefits
- **Deterministic behavior** - Same result every time
- **Reduced token usage** - No thinking required, just execution
- **Independent testability** - Scripts can be tested outside Claude
- **Better error handling** - Consistent validation across workflows
- **Improved UX** - Clear, formatted output with actionable next steps
