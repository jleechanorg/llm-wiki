## Summary

This PR adds a debug mode feature for transparency and debugging, along with several bug fixes and improvements discovered during implementation.

## Changes

### Core Feature: Debug Mode
- Added debug mode toggle commands (`\!debug on/off`) that persist per campaign
- When enabled, shows dice rolls, state changes, and AI reasoning in special debug blocks
- Debug content is automatically stripped when debug mode is disabled
- Added comprehensive test coverage for debug mode functionality

### Bug Fixes
1. **Fixed STATE_UPDATES_PROPOSED blocks showing to users** (58be466)
   - Added STATE_UPDATES_PROPOSED to debug content stripping logic
   - These internal state management blocks are now properly hidden from users
   - Added tests to verify proper stripping

2. **Fixed think block protocol tests** (a3f3938)
   - Updated test assertions to match new "Planning Block" protocol language
   - All 39 tests now pass successfully

### Code Quality Improvements
