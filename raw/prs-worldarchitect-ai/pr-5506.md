## Summary

**Surgical BYOK extraction from main branch** - Adds complete Bring Your Own Key (BYOK) support for Gemini, OpenRouter, and Cerebras providers with end-to-end validation, secure storage, and comprehensive testing.

- ✅ **User-supplied API keys**: Settings UI + backend validation + secure Firestore storage with redaction
- ✅ **LLM routing context**: Provider/key/model threaded through `llm_service.py` → `*_provider.py` → API clients
- ✅ **Navigation race fix**: Settings persistence during SPA nav using `keepalive: true` + `visibilitychange` event
- ✅ **Test infrastructure**: Browser-based BYOK flow tests with parallel execution + evidence collection
- ✅ **Secure key reveal**: Authenticated endpoint to show stored keys to owner with rate limiting + no-cache headers
- ✅ **Clear-confirm modal no longer uses Bootstrap fade**: removes opacity wash artifact and enforces transparent non-dimming modal root (UI + browser regression assertions)
