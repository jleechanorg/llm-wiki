## Summary
Simple UX improvement - makes the "WorldArchitect.AI" text in the header clickable to return to the homepage.

## Problem
Users couldn't click the site name to go home - a common web navigation pattern.

## Solution
- Changed `<span>` to `<a>` tag with `href="/"`
- Added hover effect for visual feedback
- Preserved all existing styling

## Changes
- **index.html**: Changed navbar-brand from span to anchor
- **style.css**: Added hover transition effect

## Testing
1. Navigate to any campaign page
2. Click "WorldArchitect.AI" in header
3. Returns to campaign list (homepage)
4. Hover shows visual feedback
5. Works with all themes

## Before/After
```html
<\!-- Before -->
<span class="navbar-brand mb-0 h1">WorldArchitect.AI</span>

<\!-- After -->
```
