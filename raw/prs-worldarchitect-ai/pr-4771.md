Summary:
Include dormant NPCs in entity tracking by default and only drop them when the token budget is exceeded. Adds tests that enforce tier drop ordering.

Changes:

- Drop tiers in order: dormant, present, active; log drops in ENTITY_TIERS.

Benefits:
Improves entity tracking coverage and validation confidence when budget is underused, while still protecting token limits.
