## Summary
Refactored the Divine Leverage system and Divine Ascension Ceremony prompts to be setting-agnostic while maintaining D&D Forgotten Realms as the primary example. Added comprehensive adaptation guides, terminology mapping tables, and setting-specific examples for other RPG systems (Star Wars, Warhammer 40K, superhero settings).

## Changes
- Added "⚠️ SETTING-AGNOSTIC SYSTEM" warnings to both ceremony and leverage system documents
- Created Setting Adaptation Guide with concept-to-example mapping table (Overseer, Pantheon, Reality Fabric, Divine Rank, Detection Deities, Magic Ceiling)
- Replaced hardcoded D&D deity names with generic terms throughout:
  - "Ao" → "The Overseer" with D&D example notation
  - "Mystra" → "Detection deities" / "magic/reality deities"
  - "Helm" → "Guardian entities" / "Vigilance deities"
  - "Inevitables/Solars" → "Divine agents or cosmic enforcers"
- Added inline D&D examples in parentheses for all setting-specific references
- Expanded Detection Domain Bonus section with generic guidelines for any pantheon
- Converted 12th-level magic section from "Karsus's Avatar" to "The Deity Usurpation Spell" with generic concept explanation
- Added Setting-Specific Examples section with detailed adaptations for:
  - Star Wars (The Force, Force entities, Bendu)
  - Warhammer 40K (The Emperor/Chaos Gods, The Warp, Tzeentch)
  - Superhero/Comic settings (The One Above All, Cosmic entities)
- Updated HUD examples with generic labels and D&D notation
- Added "Adaptation Principle" clarifying that mechanical structure stays constant while narrative flavor changes

## Benefits
- **Broader Applicability:** System can now be used with any RPG setting that has deity/cosmic power structures, not just D&D
- **Clearer Examples:** D&D Forgotten Realms remains the default example but is now clearly marked as such
- **Easier Adaptation:** GMs have explicit guidance on how to map generic concepts to their specific setting
- **Maintained Clarity:** D&D us
