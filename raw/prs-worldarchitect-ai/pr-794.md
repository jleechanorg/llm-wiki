- **Slash Commands** (`/testui`, `/testuif`): Updated to use Playwright MCP by default

- **Accessibility-tree based automation**: Microsoft's 2025 architecture using structured DOM data
- **AI-optimized interactions**: Designed specifically for LLM-driven browser automation

- **Session sharing capabilities**: Multiple clients can share browser sessions
- **Deterministic testing**: Structured data approach eliminates screenshot ambiguity
