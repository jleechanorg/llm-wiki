## Summary
This PR fixes a critical bug in `token_utils.py` that's causing test failures in 13 open PRs.

## The Bug
The file imports `logging_util` but then references `logging` (which is not imported), causing a `NameError`.

## The Fix
Changed line 37 from:
```python
logger = logging
```
to:
```python
logger = logging_util
```
