## Root Cause
The `/new-campaign` endpoint was failing with 404 errors because Cerebras API returned:
```
Model llama-4-maverick-17b-128e-instruct does not exist or you do not have access to it.
```

Llama 4 Maverick is a real Meta model but is only available on Google Vertex AI, Oracle Cloud, Together AI - **not Cerebras**.
