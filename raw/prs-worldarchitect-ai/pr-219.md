## New Protocol
- **ABSOLUTE PROHIBITION**: Never push directly to main/master branch
- Always use explicit branch syntax: `git push origin HEAD:feature-branch`
- All changes to main must go through pull requests - no exceptions
- If accidental push occurs, immediately alert user

## Prevention Measures
1. Always verify current branch before pushing
2. Always use explicit branch syntax in push commands
3. Create untracked branches to force explicit remote setup
4. Verify branch tracking with `git branch -vv`
