## Summary

- Adds `.claude/skills/cmux-steer.md` — full methodology for steering another agents terminal pane via the cmux Unix socket, cross-workspace safe
- Updates `CLAUDE.md` with quick-reference cmux Socket section and registers the skill in the Skill Files Reference

## Key patterns documented

- Find workspace by NAME not index: user can switch workspaces at any time; always grep `list_workspaces` output for the display name
- Send by UUID cross-workspace: `send_surface <uuid>` works regardless of user focus; `read_screen` does not
- Human-simulation pattern: `send_surface` types, `send_key_surface enter` submits; clear with ctrl-a ctrl-k first
- Probe-before-send: send test, OK=idle, ERROR=busy/wait
