## Raw Body
Add interactive AI CLI wrapper with tmux session management:

- **Live Mode**: Interactive claude/codex CLI in persistent tmux sessions
- **CLI Binary**: `ai_orch` command for terminal use
- **PyPI Package**: Installable as `ai_orch` package
- **Session Management**: Create, list, attach, kill AI sessions
- **Multiple CLIs**: Support for claude and codex profiles

Features:
- `ai_orch live` - Start interactive session
- `ai_orch list` - List active sessions
- `ai_orch attach` - Reattach to session
- `ai_orch kill` - Terminate session
- Custom session names and working directories
- Detached mode support
- Auto-detect CLI availability

Files:
- orchestration/live_mode.py - Core live mode implementation
- orchestration/setup.py - PyPI package setup
- orchestration/pyproject.toml - Modern package metadata
- orchestration/MANIFEST.in - Package distribution files
- orchestration/README.md - Updated with live mode docs

Installation:
```bash
cd orchestration
pip install -e .  # Development install
pip install ai_orch  # PyPI install (when published)
```

Usage:
```bash
ai_orch live                    # Start interactive Claude
ai_orch live --cli codex        # Start interactive Codex
ai_orch live --name my-session  # Custom session name
ai_orch list                    # List sessions
ai_orch attach my-session       # Reattach
```

This complements the existing `/orch` autonomous agent system with direct interactive access to AI CLIs in managed tmux sessions.
