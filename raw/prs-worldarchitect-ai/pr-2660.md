## Summary
- Adds a `conftest.py` file to `mvp_site/tests/` for global pytest configuration

## Changes
- Sets environment variables for all tests:
  - `TESTING=true` - Enables test mode
  - `USE_MOCKS=true` - Uses mock services
  - `MOCK_SERVICES_MODE=true` - Enables service mocking
  - `WORLDAI_DEV_MODE=true` - Prevents clock-skew validation errors
- Provides dummy API keys (`GEMINI_API_KEY`, `GOOGLE_API_KEY`) to satisfy presence checks
- Ensures Flask module is properly loaded (prevents monkeypatch issues)

## Benefits
- Prevents real network calls to Gemini/Firebase during tests
- Ensures consistent test behavior across local and CI environments
- Centralizes test configuration instead of repeating in each test file
