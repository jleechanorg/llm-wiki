## Raw Body
The previous code overwrote MCP_PORT with find_available_port output before checking the exit status, causing MCP_PORT to become empty if the port detection failed. This led to startup crashes.

Now:
- Store default port in separate variable before detection
- Capture detection output in temporary DETECTED_PORT variable
- Only update MCP_PORT if command succeeded AND returned non-empty
- Fall back to DEFAULT_MCP_PORT on failure
