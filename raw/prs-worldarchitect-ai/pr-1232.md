### 🔍 Research Findings

#### Context Limits Identified
- **Claude Sonnet 4**: 500K tokens (Enterprise) / 200K tokens (Paid)
- **Token Estimation**: ~4 chars per token, ~75 words per 100 tokens
- **Historical Analysis**: 314-340KB conversation sizes identified as problematic

#### Context Killers Documented
1. Large file operations without targeted access
2. Sequential processing of 50+ files without optimization
3. PR analysis without Serena MCP semantic navigation
4. API response bloat from full payloads
5. Redundant operations and re-reading content

### 🎯 Optimization Strategies

#### Smart Tool Selection
- **Serena MCP Priority**: Semantic navigation over large file reads
- **Targeted Access**: `limit`/`offset` parameters for large files
- **Filtered Responses**: `--json` flags for API efficiency
- **Batch Operations**: Strategic grouping vs sequential processing
