## Problem
The script was using `grep -E "^\s*-"` to detect MCP servers, but `claude mcp list` output doesn't have dash-prefixed lines. This caused the script to always think no servers were installed and reinstall them every time.

## Solution
- Changed grep pattern to `^[a-zA-Z].*:` which matches the actual output format
- Added explicit display of detected MCP servers (shows first 5 + count)
- Makes startup more informative and prevents unnecessary reinstallation

Tested locally - now correctly detects 7 installed MCP servers and skips installation.
