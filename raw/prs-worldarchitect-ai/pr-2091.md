## Raw Body
Implements the Pair Protocol for AI-accelerated pair programming with human-in-the-loop oversight at critical decision points.

Features:
- MCP mail-based agent coordination (planner + builder roles)
- Contract-first development (tests before implementation)
- Multi-layer scheming detection and prevention
- Three mandatory human review checkpoints
- Change order protocol for handling blockers
- Comprehensive audit system for implementation quality

Files added:
- .claude/commands/pair.md: Main command implementation with workflow
- .claude/commands/pair-protocol.md: MCP mail communication reference
- .claude/commands/pair-examples.md: Usage examples and troubleshooting

Usage:
  /pair <plannerAgent> <builderAgent> "Task description"

Example:
  /pair alice bob "Add user authentication with JWT tokens"

The Pair Protocol eliminates async PR latency by creating a synchronous verification loop that keeps pace with AI generation speed. The code is cheap. The contract is everything.
