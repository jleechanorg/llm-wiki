### 📁 Core Orchestration Files
- **start_system.sh** - Main orchestration system startup script with Redis, tmux, and agent management
- **start_claude_agent.sh** - Generic agent starter with task context and headless mode support
- **start_dead_code_agent.sh** - Specialized dead code cleanup agent
- **start_testing_agent.sh** - Testing agent for ensuring all tests pass

- **task_dispatcher.py** - Intelligent task distribution system
  - Smart task analysis and type inference
  - Agent capability matching and load balancing
  - Priority-based assignment (CRITICAL/HIGH/MEDIUM/LOW)
  - Task dependency tracking and comprehensive reporting

- **dashboard.py** - Real-time orchestration dashboard
  - Live system visualization with 5-second auto-refresh
  - Agent status, workload, and uptime monitoring
  - Recent PR activity tracking and system resource monitoring
  - Interactive system control and health reporting

- **README_ENHANCED.md** - Comprehensive documentation
