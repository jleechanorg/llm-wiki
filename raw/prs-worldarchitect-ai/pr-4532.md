**Purpose:** Architectural Decision Record for unified schema approach

- **New ADR document** - Did not exist before → Created comprehensive design document → Need single source of truth for fragmented schemas

### 2. `mvp_site/schemas/game_state.schema.json` (+790 lines)
**Purpose:** Canonical JSON Schema (Draft 2020-12) for complete game state

- **CustomCampaignState typed** - Was `dict[str, Any]` → Full nested schema (FactionMinigame, ArcMilestone, GodModeDirective, etc.)
- **CombatState unified** - Inconsistent between GameState and Pydantic → Single schema with CombatantState
- **Embedded descriptions** - Every field has description for LLM prompt generation
