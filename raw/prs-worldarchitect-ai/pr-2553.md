## Raw Body
Implements a dedicated combat agent that automatically activates when game_state.combat_state.in_combat is True. This provides focused tactical prompts during active combat encounters.

Changes:
- Add CombatAgent class following existing agent architecture pattern
- Add combat_system_instruction.md with combat-focused prompts including:
  - Combat session tracking with unique session IDs
  - Combat phase management (initiating, active, concluding, ended)
  - Mandatory combat rewards display (XP, loot, resources)
  - Boss/Special NPC gear slot requirements (all 12 slots)
  - Combat dice protocol enforcement
- Update get_agent_for_input() to select CombatAgent when in combat
- Add combat session schema to game_state_instruction.md
- Add PROMPT_TYPE_COMBAT and MODE_COMBAT constants
- Update llm_service.py to import CombatAgent

Agent priority order:
1. GodModeAgent - GOD MODE: prefix (administrative override)
2. CombatAgent - combat_state.in_combat=True (tactical combat)
3. StoryModeAgent - Default (narrative storytelling)
