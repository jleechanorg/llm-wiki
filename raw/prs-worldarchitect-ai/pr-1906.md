## Summary

Creates a Model Context Protocol (MCP) server plugin for the `/secondo` command, enabling system-wide installation via Claude Code's `/plugin install` workflow.

**Key Changes:**
- ✅ MCP server with Express (implements GET /tools, POST /execute endpoints)
- ✅ Plugin manifest for Claude Code plugin system
- ✅ Wraps existing secondo-cli.sh (no code duplication)
- ✅ OAuth authentication via auth-cli.mjs
- ✅ Comprehensive documentation with architecture diagrams
- ✅ Fixed /localexportcommands to export auth-cli.mjs dependency

## Architecture

```
Claude Code → MCP Server (localhost:3003) → secondo-cli.sh → AI Universe Backend
```

The MCP server is a lightweight wrapper that:
- Exposes MCP-compliant HTTP endpoints
- Delegates to existing `secondo-cli.sh` script
- Handles authentication via `auth-cli.mjs`
- Maintains connection to AI Universe backend (5 AI models)

- `mcp_servers/secondo/server.js` - Express-based MCP server (235 lines)
