# Memory MCP Integration for /learn Command

## Overview
This PR adds Memory MCP integration to the `/learn` command, enabling persistent knowledge storage that survives across conversations.

## Key Features
- ✅ **Persistent Knowledge Graph**: Learnings stored in Memory MCP survive beyond current session
- ✅ **Entity Creation**: Convert learnings to persistent entities with proper schema
- ✅ **Relation Building**: Connect related learnings and patterns in knowledge graph
- ✅ **Duplicate Detection**: Search existing knowledge before creating new entities
- ✅ **Error Handling**: Graceful degradation with local-only fallback when Memory MCP unavailable
- ✅ **Backward Compatible**: All existing /learn functionality preserved

- **Types**: `learning`, `pattern`, `lesson`, `context`
