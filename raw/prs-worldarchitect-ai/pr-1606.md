## Root Cause Analysis

- This caused the expanded script collection (including create_worktree.sh) to be invisible in the claude-commands repository

**File**: `.claude/commands/exportcommands.py`
**Method**: `_copy_to_repository()`
**Lines**: 911-923

### Before (Broken):
```python
script_dst = os.path.join(self.repo_dir, script_file)  # ← Wrong: copies to root
```

