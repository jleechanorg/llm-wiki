## Overview

Comprehensive plan for refactoring WorldArchitect.AI's monolithic main.py into a modern layered architecture using MCP (Model Context Protocol).

## Architecture Transformation

### Current State
- **Monolithic Flask app** in main.py handling routes, business logic, and frontend serving
- Static assets in `mvp_site/static/`
- Mixed concerns: HTTP handling + D&D game mechanics + AI integration

### Target State
- **world_logic.py** - MCP server exposing D&D game mechanics as tools/resources
- **main.py** - Slim translation layer converting HTTP ↔ MCP protocol
- **frontend_v1/** - Reorganized static assets for multi-version frontend support

## ✅ Enhanced Technical Design

### Key Architecture Solutions
- **Async/Sync Bridge**: Custom decorator pattern for Flask + MCP integration
- **Redis + Firestore**: Dual-layer state management with TTL and source-of-truth
- **WebSocket Integration**: Real-time D&D gaming with SocketIO + MCP
- **Circuit Breaker Pattern**: Service resilience with automatic failover
- **Transaction Handling**: ACID properties for complex game actions
- **Docker Compose**: Complete deployment configuration
