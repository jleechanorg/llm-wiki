## 🔧 Robustness Improvements

### Pre-flight Dependency Validation
```bash
# NEW: Prevent confusing failures from missing tools
if ! command -v jq >/dev/null 2>&1; then
  echo "Error: jq is required but not installed." >&2
  exit 5
fi
if ! command -v curl >/dev/null 2>&1; then
  echo "Error: curl is required but not installed." >&2
  exit 5
fi
```

### Enhanced Error Handling
- **Strict Mode**: `set -Eeuo pipefail` for immediate failure detection
- **Clear Error Codes**: Exit code 5 for dependency issues 
- **Prevents Silent Failures**: Script fails fast when dependencies missing

### Network Robustness
```bash
# OLD: Basic curl with potential hanging
curl -s -w "HTTPSTATUS:%{http_code}" -X POST "https://api.cerebras.ai/v1/chat/completions"

# NEW: Robust curl with timeouts and error handling
curl -sS --fail-with-body --connect-timeout 10 --max-time 60 \
  -w "HTTPSTATUS:%{http_code}" -X POST "${CEREBRAS_API_BASE:-https://api.cerebras.ai}/v1/chat/completions"
```

### Configuration Flexibility
- **Configurable API Base**: Support for `CEREBRAS_API_BASE` environment variable
- **Better Error Reporting**: `--fail-with-body` provides response details on failures
- **Reasonable Timeouts**: 10s connect, 60s total prevents hanging
