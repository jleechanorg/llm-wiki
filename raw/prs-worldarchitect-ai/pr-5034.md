## Raw Body
### Motivation
- Prevent `deploy-preview` job from failing on Dependabot/forked PRs due to missing `GCP_SA_KEY` secrets which produced spurious deployment errors.

### Description
- Gate the `deploy-preview` job in `.github/workflows/pr-preview.yml` so it only runs when the PR is not draft, not from a fork, the actor is not `dependabot[bot]`, and `GCP_SA_KEY` is present.
