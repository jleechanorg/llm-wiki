## Revolutionary Architecture: Hook + LLM vs Parsing
Instead of building parsing rules, we hook into Claude's natural language understanding:

**❌ OLD (Rule-Based Parsing):**
```python
# 150+ lines of parsing logic, command registries, edge case handling
parse_composition_command() → protocol='/test', flags={'debug': True}
```

**✅ NEW (Hook + LLM Interpretation):**
```python
# ~30 lines leveraging Claude's LLM
claude_interpret("Execute: '/debug /paranoid /test src/' with available commands")
→ "Run comprehensive tests on src/ with debug output and paranoid validation"
```
