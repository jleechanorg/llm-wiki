## Changes Made

### Removed Conflicting Mechanics
- 5-aptitude system (Physique, Coordination, Health, Intelligence, Wisdom)
- Big Five personality traits replacing Charisma
- Energy Points (EP) unified resource system
- Yearly progression model with age-based attribute changes
- Aptitude potential coefficients
- Core ambition & milestone parallel progression track
- Internal conflict resolution system
- Detailed rapport/chemistry relationship mechanics
- Modified damage calculations using dual physical aptitudes
- Martial techniques with per-encounter usage

### Kept D&D-Compatible Elements
- Core d20 + modifiers vs CN resolution mechanic
- Combat state management structure
- Death saving throws (3 successes/failures)
- Environmental interaction rules (travel, traps, vision)
- Rest and recovery system (short/long rests)
- Damage types and resistances
- Critical hits and fumbles

### Added D&D 5e Reference Content
- Complete ability score list (STR, DEX, CON, INT, WIS, CHA)
- Proficiency bonus progression table
- Advantage/disadvantage mechanics
- All 15 standard conditions with full mechanical effects
- Skills organized by ability score
- Combat action options (Attack, Dash, Disengage, Dodge, etc.)
- Movement rules and cover mechanics
- Saving throw and spell save DC formulas

## Rationale
The original Destiny ruleset had many innovative mechanics but created confusion when mixed with D&D 5e. This refactor provides a clean foundation that:
1. Maintains compatibility with standard D&D 5e rules
2. Provides useful reference content for the AI
3. Preserves helpful organizational structures (like combat state management)
4. Archives the original creative content for potential future use
