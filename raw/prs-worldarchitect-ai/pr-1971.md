## Raw Body
Problem: integrate.sh was creating blank PRs when local main had commits ahead of origin/main that contained no actual file changes (merge commits only). This resulted in PRs with 0 additions, 0 deletions, 0 changed files.

Solution: Added git diff check before PR creation to detect actual file changes. If no file changes detected, auto-reset local main to origin/main (in force mode) or show instructions.

Benefits:
- Prevents useless blank PRs from cluttering GitHub
- Automatically cleans up divergent main branch histories
- Provides clear feedback about what's happening

Test Plan:
- Tested with actual blank commits scenario (4 merge commits, 0 file changes)
- Verified detection and reset behavior in force mode
- Closed existing blank PRs (#1970, #1967, #1966)
