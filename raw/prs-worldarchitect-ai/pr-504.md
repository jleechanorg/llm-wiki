## Summary

This PR adds comprehensive HTML/CSS mockups for the Dynamic World State UI feature. These interactive prototypes demonstrate how players will visualize and interact with the event-tracking system.

## What's Included

### 📊 Timeline View
- Interactive campaign timeline showing events across sessions
- Visual world state periods (stable, chaos, recovering, crisis)
- Event tooltips with cascading consequence indicators
- Zoomable interface with session markers

### 🕸️ NPC Relationship Graph
- Network visualization of character relationships
- Trust meters showing relationship strength
- Faction standing displays
- Recent relationship change timeline
- Inter-NPC connection visualization

### 🎮 In-Game Panel
- Collapsible side panel (non-intrusive during gameplay)
- Recent events mini-timeline
- Active world effects display
- Quick stats overview
- Mobile-responsive bottom bar variant

### 🏆 Session Recap Screen
- Post-session impact summary
- Key moments highlighting
- World impact chart (using Chart.js)
- Cascading consequences flow visualization
- Social sharing preview
- Next session preview teasers

## Technical Details

- **Responsive Design**: Works on mobile, tablet, and desktop
- **Modern CSS**: Custom properties, grid/flexbox layouts
- **Interactive Elements**: Hover states, collapsible panels, tooltips
- **Design System**: Consistent colors, typography, spacing
- **Accessibility**: ARIA-ready structure
