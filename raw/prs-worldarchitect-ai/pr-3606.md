## Summary
- Engineering planning document for integrating user avatars into the campaign play UI
- Covers avatar upload in gameplay view (top right button) and campaign creation (optional Step 1 field)
- Avatar display in bottom-left corner during campaign gameplay

## Key Features Planned
1. **Avatar Upload Button (Top Right)** - Camera icon to left of download button during gameplay
2. **Avatar Display (Bottom Left)** - Shows user avatar beside input textarea
3. **Campaign Creation Avatar (Optional)** - Upload character portrait during campaign creation wizard

## Technical Approach
- Firebase Storage for image hosting
- New `POST/DELETE /api/avatar` endpoints for user avatar management
- Extended `POST /api/campaigns` to accept multipart/form-data with optional avatar
- Frontend changes to `GamePlayView.tsx` and `CampaignCreationV2.tsx`
