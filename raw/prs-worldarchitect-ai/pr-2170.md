## Raw Body
Add comprehensive unit and integration tests to prevent regression of the asyncio.to_thread() fix from PR #2157 that resolved campaign concurrency blocking.

New test classes:
- TestAsyncNonBlocking: Verifies concurrent coroutines don't serialize
- TestThreadPoolExecution: Verifies blocking calls run in worker threads
- TestBlockingCallStaticAnalysis: AST-based detection of unwrapped blocking calls
- TestParallelismIntegration: CI-compatible integration tests
- TestAsyncToThreadDocumentation: Ensures documentation requirements

Key features:
- Static analysis catches missing asyncio.to_thread() at parse time
- Timing-based tests detect serialization at runtime
- Canary test alerts if asyncio.to_thread() count drops significantly
- All tests work in CI without external dependencies
