- Truncate `initial_prompt` to 100 chars for list view
- Support legacy `prompt` field fallback for older campaigns

### 2. `mvp_site/main.py` (+209/-71 lines)  
**Purpose:** Parallel I/O for game page load
- `run_blocking_io()`: Run Firestore queries in thread pool
- `_load_campaign_page_data()`: Fetch 4 queries concurrently via asyncio.gather()
