# Redesign Chat Spinner

## Overview
This PR completely redesigns the loading spinner used in the main chat/campaign interface.

### Changes
* Removed the large white rectangular card layout and flex-column setup.
* Implemented a sleek, inline glassmorphism pill (`rgba(15, 10, 25, 0.75)` with `backdrop-filter: blur(8px)`).
* Replaced the standard spinner text color with a muted lavender (`#e2d9f3`) for excellent dark mode contrast.
* Updated layout to be `flex-direction: row` for a compact, horizontal loading indicator that blends perfectly into the game UI above the chat box.

### Visual Comparison
**Before**: A glaring white box that broke the dark fantasy immersion.
**After**: A fully translucent, dark-themed indicator.
