## Summary

Adds new  command to streamline regular task delegation with structured handoffs.

## Features

### Command Usage
```
/handoff [task_name] [description]
```

### Automated Workflow
1. **Creates handoff branch**: `handoff-[task_name]`
2. **Generates scratchpad**: Complete implementation plan in `roadmap/scratchpad_handoff_[task_name].md`
3. **Updates roadmap.md**: Adds task entry with HANDOFF READY status
4. **Creates GitHub PR**: With detailed description and ready-to-implement status
5. **Generates worker prompt**: Copy-paste ready instructions for delegation
6. **Creates clean branch**: `roadmap[timestamp]` for continued work

## Benefits

- **Consistent handoffs**: Same structure and quality every time
- **Complete documentation**: Analysis, implementation plan, testing requirements
- **Roadmap integration**: Tasks automatically tracked in roadmap.md
- **Worker-ready**: Formatted prompts for easy task delegation
- **Clean separation**: Handoff work isolated from ongoing development
