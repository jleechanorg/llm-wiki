## 🔧 Technical Details

### DuckDuckGo Search Server
- **Package**: `@oevortex/ddg_search`
- **Cost**: Completely free, no API key required
- **Features**: 
  - Privacy-focused web search using DuckDuckGo engine
  - Content extraction and metadata retrieval  
  - Felo AI search capabilities
  - Web scraping with markdown conversion

### Perplexity AI Search Server
- **Package**: `server-perplexity-ask` 
- **Cost**: Requires Perplexity API key (premium)
- **Features**:
  - AI-powered search with real-time web research
  - Advanced query processing and reasoning
  - Sonar API integration for live web data
  - Enhanced search result quality and context

### Token Management System
Enhanced `claude_mcp.sh` to support:
```bash
# ~/.token file format
export GITHUB_TOKEN="ghp_***"
export PERPLEXITY_API_KEY="pplx-***"
```

- **Secure sourcing** from `~/.token` file
- **Environment variable validation** 
- **Legacy compatibility** maintained
- **Enhanced error messaging** with setup instructions
