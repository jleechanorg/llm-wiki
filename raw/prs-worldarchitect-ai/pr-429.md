## Summary
This PR enhances the narrative system to make NPCs more proactive and the world feel more alive, addressing user feedback that the game world felt static with the player as the only active participant.

## Changes
- Added NPC Initiative Guidelines for regular proactive approaches (every 5-15 scenes)
- Added Living World Guidelines with dynamic encounters and active world events
- Enhanced NPC behavior rules to show NPCs taking initiative with invitations, requests, warnings, etc.
- Added specific examples of active NPC behaviors (direct invitations, urgent pleas, business propositions, etc.)
- Balanced frequency to avoid overwhelming players while keeping world dynamic
- **NEW**: Added comprehensive integration test suite to verify all enhancements work correctly

## Key Enhancements
1. **NPC Initiative**: NPCs now actively approach players with contextual requests, invitations, and opportunities
2. **Dynamic Encounters**: More frequent in populated areas (1-2 per day) with 30-40% involving proactive NPC engagement
3. **Living World**: Shows other adventurers, ongoing conflicts, and faction activities happening independently
4. **Natural Consequences**: Realistic outcomes when players ignore or refuse encounters
