## Summary

Adds `testing_llm/run_worlds_20_ticks.py` — a Python harness that bootstraps 3 universe prompts through 20 LLM-powered turns with real OpenClaw gateway calls.

### 3 Worlds

| World | Genre | Player Character |
|-------|-------|-----------------|
| **Shattersea Reef Crown** | Maritime fantasy | Mara Vey, navy cartographer with brass tide-compass |
| **Helion Ring Afterlight** | Star Wars-style sci-fi | Captain Neris Vale, disgraced recon pilot |
| **Glassline Monsoon City** | Cyberpunk | Rin Mercado, municipal auditor with flood-risk ledger |

### Modes
- `python3 testing_llm/run_worlds_20_ticks.py` — Sequential (starts server, runs 3 worlds)
- `python3 testing_llm/run_worlds_20_ticks.py --parallel 5` — 5 parallel processes
- `WORLD_INDEX=1 BACKEND_URL=... python3 testing_llm/run_worlds_20_ticks.py` — Single world

### Verified
- Ran 5 parallel processes against real OpenClaw gateway (minimax/m2.5)
- 74 successful HTTP 200 responses out of 76 turn requests
- Fixed pipe deadlock by using file-based logging instead of stdout pipes
- Fixed 401 auth issue by forcing OPENCLAW_GATEWAY_TOKEN via extra_env
