## High-level description

Added \`applyGameRules()\` post-processor (runs after \`deepMerge()\`):

1. **XP/Level** — clamp to [0, 355000]; auto-derive level from XP table when XP changes; explicit level clamped to [1, 20]
2. **HP/MP** — player and companion clamped to max; floor at 0
3. **Companion id-merge** — \`maxHp\` from previousState preserved via per-companion id lookup
4. **Item-usage** — \`using\`/\`equipped\`/\`consuming\`/\`itemUsed\` top-level delta fields validated against inventory/equipment (case-insensitive)
5. **Player key allowlist** — \`filterToAllowedPlayerKeys()\` rejects arbitrary nested objects smuggled into player state
