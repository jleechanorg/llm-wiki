- Reduce repeated session-list scans and make ownership checks O(1) and consistent across companion endpoints to prevent IDOR edge cases and improve performance.

- Prevent stale repository state across DB resets in tests by ensuring the companion-repository singleton is cleared on DB close.  
- Make the companion catchup integration test resilient to transient upstream/LLM 502s and avoid flaky failures during CI runs.
