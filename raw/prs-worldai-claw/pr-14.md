## Summary

This branch combines the React Native AI integration planning work with foundational backend routes, frontend simplification, test infrastructure cleanup, and UI styleguide iterations.

### Design & Styleguide
- Design review (Section 8): identified 8 gaps, most critical being **deferring Expo 51→54 upgrade**
- TDD implementation roadmap (Section 9): 11 milestones (M0–M8), each ≤500 LOC with RED/GREEN commits
- Full HTML styleguide (`docs/mocks/styleguide.html`) with all page mockups (setup, dashboard, game, settings, campaigns, mobile)
- Purple fantasy theme with Snapchat-inspired gradient + translucent layers, frosted-glass cards, WCAG-compliant contrast

### Styleguide Improvements (latest commits)
- **Accessibility**: `prefers-reduced-motion` (kills all animations) and `prefers-reduced-transparency` (solid fallbacks for glass) media queries
- **WCAG 1.4.11**: border contrast bumped from 0.2→0.35 opacity (meets 3:1 for UI components)
- **Text contrast**: `--color-text-muted` lifted to `#a8a2c8` (~5.5:1 solid AA) from 4.6:1
- **Reduced purple**: choice cards desaturated at rest (neutral bg + muted border); purple gradient + glow only on hover; card title text follows hover state
- **Panel hierarchy**: `--color-bg-panel: #09080f` neutral-dark surface separates meta UI (stats, equipment) from narrative zone
- **Mobile bottom anchoring**: bottom tabs + input bar darkened to `#07060e` so active tab/send button pop clearly
- **Location breadcrumb**: replaced boxed info-block with inline `Location › … › …` text, freeing vertical space
- **Responsive**: 768px (single column, wizard labels hidden) and 480px (full-width choices, 44px touch targets) breakpoints
- **Grain texture**: 2.5% SVG fractalNoise overlay on cards via `::before` for tactile feel
