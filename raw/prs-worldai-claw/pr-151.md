> **Overview**

> 
> **Chain anchoring/verification now persists locally.** `ChainAnchor` writes anchor ownership + root-hash metadata to a new SQLite `anchor_owners` table (returning `success` with a simulated `txHash`) and `verifyOnChain` checks stored anchors for root-hash matches. MCP DM tools (`dm_anchor_entity`, `dm_verify_anchor`) now resolve entity type with a companion fallback and seed a `genesis` block when anchoring an empty chain; DM companion edit tools also rollback companion updates if entity-chain append fails.
> 
> **Tests added.** New MCP integration tests cover positive anchoring/verification, stale-anchor detection after chain updates, and authorization (owner vs contract owner override).
