## High-level description of changes

Added Gate 6 to the skeptic-gate.yml workflow. The gate:
1. Fetches the PR body via GitHub REST API
2. Extracts the ## Evidence section using awk (same boundary logic as wholesome.yml)
3. Strips HTML comment lines before scanning
4. Rejects fabricated content: placeholder text, example.com URLs, template markers
5. Accepts real media: markdown images with HTTPS URLs, code blocks, structured output references
6. Always outputs WARN (never FAIL) - logged but non-blocking
