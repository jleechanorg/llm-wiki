## High-level description

5 targeted fixes to `.claude/metadata-updater.sh`:

1. **Env-var prefix regex** (`[^= ]*` → `[^[:space:]]*`): Now allows `=` in values (e.g. `FOO=a=b`) which is valid shell syntax. Note: `FOO=bar BAZ=qux gh pr create` strips `FOO=bar` in first iteration, `BAZ=qux` in second — both env vars stripped correctly.
2. **Hook event guard** (`!= "PostToolUse"` → `!= "PostToolUse" && -n "$hook_event"`): PreToolUse with empty `hook_event` should fall through to guards rather than silent-exit. Comment clarified to match actual behavior.
3. **sed escape simplification** (`[&|\\]` → `[&\\]`): BRE `sed` replacement only treats `&` and `\` as special — `|` is literal and doesn't need escaping.
4. **Git branch existence guard**: Uses `git show-ref --verify --quiet refs/heads/$branch` — more targeted to local branches than `git rev-parse`, faster, and prevents file paths like `src/main.ts` from matching the checkout/switch regex.
5. **Branch detection comment updated**: Clarified that the guard is to prevent file paths from matching (e.g. `git checkout src/main.ts`).
