## Changes

### `packages/core/src/lifecycle-manager.ts`

Extended `statusToEventType()`:
- `spawning → session.spawned`
- `terminated → session.exited`

Extended `eventToReactionKey()`:
- `session.spawned → session-spawned`
- `session.exited → session-exited`

These map to `session-spawned` and `session-exited` reactions in `agent-orchestrator.yaml` (updated in jleechanclaw PR #392), routing to the `mcp-mail` notifier for thread reconciliation.

### `packages/plugins/runtime-tmux/src/index.ts`

Fix: queued tmux messages now properly submitted instead of marked delivered.
