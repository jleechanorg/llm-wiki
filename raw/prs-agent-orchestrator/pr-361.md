## Raw Body
## Background
1. **CI gate gap**: `test_builtin_plugin_deps.test.ts` existed (PR #334) but was not enforced in CI — a new builtin plugin added without updating `packages/cli/package.json` would silently bypass the test.
2. **wholesome.yml broken**: The `grep -qF` and `awk` patterns for evidence label detection were missing the closing `**` before the colon. Evidence Has Media Attachment was failing for all PRs.

## Goals
1. Enforce the builtin plugin deps test in CI so it cannot be bypassed.
2. Fix wholesome.yml label patterns so Evidence Gate passes correctly.

## What Changed
1. **ci.yml**: Added `builtin-plugin-deps` job that runs `pnpm --filter @jleechanorg/ao-core test -- src/__tests__/test_builtin_plugin_deps.test.ts`.
2. **wholesome.yml**: Fixed all `grep -qF` patterns to include `**` before `:` (e.g., `'**Repro gist**:'` not `'**Repro gist:'`). Fixed all `awk` block-extraction patterns to match the same.
3. **ci.yml**: Added `SELF_HOSTED_DISABLED` var fallback so CI falls back to `ubuntu-latest` when self-hosted runners are offline.
4. **evidence-gate.yml**: Added same `SELF_HOSTED_DISABLED` fallback.
