> **Overview**
> Adds a new **fail-closed** PreToolUse gate that prevents agents from closing PRs via `gh pr close` or `gh api ... state=closed` unless the target PR already contains a "Superseded by #<N>" comment (checked via paginated `gh api` calls).
> 
> Updates `protect-pr-close.sh` to detect close commands more reliably (keeps quoted args, strips heredoc bodies), extract PR number and repo (supports `--repo` and git-remote fallback), and block closes when parsing/repo detection/API validation fails. Enables the hook via `.claude/settings.json` so it runs before Bash tool invocations.
