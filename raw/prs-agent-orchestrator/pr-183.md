## Summary

Add @jleechanorg/ao-plugin-prose-polish — a runtime plugin that exposes MCP tools for detecting and fixing common fiction prose patterns in AO novels.

## Prose patterns detected

- **Not X / Not-X Repetition** — flags when 3+ 'Not X.' lines appear (overuse dilutes impact)
- **Forced Rule of 3s** — detects structurally identical triplets near each other
- **Weak Sentence Opening** — flags 'There is', 'It is', 'This is' starts
- **Filler Words** — detects literally, basically, simply, actually, obviously, clearly, etc.
- **Redundant Phrases** — flags very unique, very special, free gift, etc.
- **Weasel Words** — flags vague quantifiers: many, some, various, a lot of
- **Repeated Sentence Starters** — same first word in 3+ consecutive lines
- **Proximity Repetition** — same content word 3+ times within 10-line window

## Tools exposed

- `prose_polish_scan` — scan a markdown/prose file, return detailed report
- `prose_polish_fix` — apply auto-fixable corrections with .bak backup
- `prose_polish_rules` — list all rules with descriptions
