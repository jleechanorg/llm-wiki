## High-level description of changes

**agent-codex test** had two bugs:
1. `codex exec` in a fresh tmp dir exits immediately with *"Not inside a trusted directory and --skip-git-repo-check was not specified"*. The test never saw the process running because it was dead in <50ms. Fix: add `exec --skip-git-repo-check`.
2. Prerequisite check used `OPENAI_API_KEY` env var, but this machine (and likely others) uses OAuth tokens in `~/.codex/auth.json`. Fix: `hasCodexCredentials()` checks both.

**agent-claude-code test**: Claude Code emits a `last-prompt` JSONL entry type that was missing from the `knownTypes` allowlist. Fix: add `'last-prompt'` to the list.
