## Automated Daily Social Pipeline

Automated daily pipeline: AO workers novel → animated short → YouTube Shorts + TikTok.

### What this adds

| File | Description |
|------|-------------|
| `novel/upstream/video/src/DailyChapter.tsx` | New Remotion composition — 9:16 portrait (1080x1920) |
| `novel/upstream/video/src/Root.tsx` | Registers DailyChapter in Remotion |
| `novel/upstream/video/src/ChapterData.ts` | Chapter data interface |
| `scripts/social/pipeline.js` | Node.js pipeline — extract chapter → render → upload |
| `scripts/social/package.json` | Dependencies (googleapis, puppeteer) |
| `docs/SOCIAL_PIPELINE.md` | Full setup guide |

### How it works

```
PR merged / worker reaped
         ↓
  ai.agento.novel-daily (launchd, 6am PT)
         ↓
  novel:daily → appends chapter to the-daily-lives-of-workers.md
         ↓
  node pipeline.js
         ├→ Remotion renders DailyChapter (9:16 short, ~10s)
         ├→ YouTube Shorts upload (OAuth2)
         └→ TikTok upload (Puppeteer browser automation)
```

- `scripts/social/credentials/tiktok-session.json` — TikTok session cookies
