## Summary

Adds a built-in `minimax` agent plugin so `ao spawn <issue> --agent minimax` runs Claude Code against MiniMax's Anthropic-compatible API (`ANTHROPIC_BASE_URL`, keys from `MINIMAX_API_KEY`).

## Changes

- New package `packages/plugins/agent-minimax/`: env wiring, strips `--model` on launch, disables restore (avoids incompatible model IDs), warns when `MINIMAX_API_KEY` is unset
- Registered in `plugin-registry.ts` and CLI `plugins.ts`
- Unit tests for env and launch behavior

## Usage

```bash
export MINIMAX_API_KEY="sk-..."
# Optional: MINIMAX_ANTHROPIC_BASE_URL, MINIMAX_MODEL
ao spawn orch-xxx --agent minimax
```
