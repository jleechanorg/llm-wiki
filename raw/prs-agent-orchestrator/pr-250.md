## Summary

Improves the `metadata-updater.sh` PostToolUse/PreToolUse hook with:

- **Env assignment stripping**: Commands prefixed with `FOO=bar gh pr create` are now correctly detected (previously only `cd`-prefixed commands were stripped)
- **`[agento]` prefix guardrail**: PreToolUse now enforces PR titles start with `[agento]` using python3 shlex parsing (correctly handles `--body="--title=foo"` edge case)
- **Restructured merge guardrail**: Moved before the PostToolUse-only gate so PreToolUse denials fire correctly
- **Fixed errexit bugs**: `grep -q || true` guards prevent premature exit when grep finds nothing
- **Fixed sed escaping**: Removed `|` and `/` from BRE character class (were being treated as literals in BRE, not as delimiters)
- **Split compound branch-detection ifs**: Each git command pattern (checkout -b, switch -c, etc.) now has its own early-exit `if` block
