## Goals
- Scan open GitHub PRs via `gh pr list --json`
- Identify non-green PRs (CI failing, changes requested, review pending, merge conflicts)
- Return `PollerWorkItem[]` for the poller-manager to spawn sessions
- Integrate with plugin registry as a builtin poller
