## Background

## Goals
Fix all 8 CR Major/Minor items, restore Evidence Bundle v2 policy, and make evidence gate enforcement consistent across CI workflows.

## Tenets
- **Don't weaken existing policy**: Evidence Bundle v2 (repro gist + terminal media + test output + UI media) is the authoritative standard — shortcuts must reference it, not replace it.
- **Consistency across gates**: evidence-gate.yml, wholesome.yml, and the template doc should accept the same media/output formats.
- **Fail-closed on empty self-validation**: bare `**Self-validation**:` label ≠ proof.
