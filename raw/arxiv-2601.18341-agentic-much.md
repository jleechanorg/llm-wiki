## Summary

First large-scale empirical study of coding agent adoption on GitHub, analyzing 128,018 projects. Found 22.20%–28.66% adoption rate — "very high for a technology only a few months old." Adoption spans entire project maturity spectrum and diverse languages/topics. Agent-assisted commits tend to be larger with more features and bug fixes.

## Key Claims

- **22.20%–28.66%** estimated adoption rate among GitHub projects
- Study based on **128,018 projects** with explicit traces from coding agents
- Broad adoption across project maturity, organizations, and programming languages
- Agent-assisted commits are larger and focus on features/bug fixes

## Method

Leveraged explicit traces left by coding agents (co-authoring commits/PRs) to identify usage patterns. Authors tracked when agents were listed as co-authors on commits or PRs.

## Results

- 22-29% adoption rate
- Spans entire project maturity spectrum
- Diverse languages and topics
- Agent commits focus on features/bug fixes
