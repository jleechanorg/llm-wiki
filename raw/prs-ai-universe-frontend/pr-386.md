## Raw Body
### Motivation
- Make `scripts/setup-dev-tools.sh` safe to re-run and ensure Python OCR dependencies are installed in a usable location when the script runs as root, while reducing redundant `apt-get update` calls and avoiding GPG overwrite failures.

### Description
- Remove the `--user` flag from the pip invocation in `install_python_ocr_deps` so `python3 -m pip install --upgrade pillow pytesseract` installs system-wide when executed as root.
- Centralize `apt-get update` to run once at the start of `main` and remove the redundant `apt-get update` from `install_apt_packages` to avoid repeated updates.
- Make GCloud keyring creation idempotent by skipping the `gpg --dearmor` write when `/usr/share/keyrings/cloud.google.gpg` already exists.
