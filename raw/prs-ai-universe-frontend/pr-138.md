Added comprehensive guidelines for Claude to operate more autonomously:
- Take initiative and complete full workflows without asking permission at each step
- Make reasonable decisions based on context and best practices
- Fix discovered issues immediately (bugs, type errors, test failures)
- Clear decision framework: when to ask vs when to act
- Standard workflow expectations (change → test → commit → push → PR)
- Communication style guidance (report results, not intentions)

This reduces back-and-forth and enables faster iteration by empowering Claude to complete standard workflows autonomously while still asking about architectural decisions and ambiguous requirements.
