## Raw Body
Use @ai-universe/typescript-server-launcher package to provide a TypeScript alternative to run_local_server.sh with:
- Port management with automatic port finding
- Multiple server modes (dev, proxy, preview, test)
- Backend URL configuration
- --disable-auth and --force-default-port flags
- Colored console output

New npm scripts:
- npm run local (default dev)
- npm run local:dev
- npm run local:status
- npm run local:cleanup
