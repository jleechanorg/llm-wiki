## Changes

### New Section: Prerequisites & Tool Installation

1. **GitHub CLI (gh) Installation**
   - Step-by-step npm installation instructions
   - Verification and authentication steps

2. **General Tool Installation Pattern**
   - Template for installing any missing tool via npm
   - Detection and installation workflow

3. **Common npm-installable Tools**
   - Listed frequently used tools: gh, typescript, eslint, prettier, playwright, vite, vitest

4. **Installation Priority Guidelines**
   - Prefer npm for Node.js ecosystem tools
   - Use global installs for CLI tools
   - Check project dependencies first
   - Fallback to system package managers only when needed

5. **Autonomous Installation Protocol**
   - Auto-install without permission (aligns with autonomous operation principles)
   - Verify installation after install
   - Continue workflow automatically
   - Example auto-detection script
