## Summary

## Technical Details

- Uses `Set<string>` to track expanded cards
- Each card has unique ID: `${model}-${index}`
- Toggle function adds/removes from Set
- `isCardExpanded(id)` checks if ID exists in Set
- Empty Set = all collapsed (default)
