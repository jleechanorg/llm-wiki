## Summary

Implements the full Remote Mode + Auto-Scan architecture per the [spec](docs/superpowers/specs/2026-03-26-remote-mode-auto-scan-design.md):

- **RepoRegistry** (`src/blog/repo-registry.ts`) — per-repo config persisted to `data/repos.json` with modes for autoScan, novelBranch, novelDaily
- **SHA-256 API key auth** (`src/blog/auth.ts`) — `crypto.createHash`, `timingSafeEqual`, `requireApiKey` middleware, `loadApiKeys`/`saveApiKeys` persistence to `data/api-keys.json`
- **GitHubClient** (`src/blog/github-client.ts`) — `@octokit/rest` REST-only (no GraphQL)
- **AutoScanner** (`src/blog/scanner.ts`) — `setInterval` polling with cursor dedup (`data/scan-cursor.json`) and UTC date-crossing daily summary trigger
- **WebhookReceiver** (`src/blog/webhook.ts`) — HMAC-SHA256 validation, routes events identically to AutoScanner
- **WorkerChat** (`src/novel/chat.ts`) — regex voice extraction (0 LLM calls), character-consistent Anthropic API responses
- **Server wiring** (`src/blog/server.ts`) — auth middleware, rate limiters (100 req/min/IP), `/webhook` and `/chat` routes, AutoScanner lifecycle
- **6 new MCP tools** — `register_repo`, `unregister_repo`, `list_repos`, `update_repo`, `generate_api_key`, `chat_worker`
- **Blog CLI** (`src/blog/cli.ts`) — `repo add/list/enable/disable/remove`, `apikey generate/list`
- **AITA cliffhanger endings** — `generateClosingPOV` and `generateReaperPOV` now end mid-beat with `*TO BE CONTINUED*`
- Full docs updates to `README.md` and `docs/CONFIGURATION.md`
