## Summary

Container exits with EACCES on mkdir 'data/' on Cloud Run because non-root nodeuser cannot write to /app.

## Fix

Pre-create data/ and logs/ directories at build time and chown to nodeuser.

```dockerfile
# Pre-create data/ and logs/ so nodeuser can write at runtime
RUN mkdir -p /app/data /app/logs && chown -R nodeuser:appgroup /app/data /app/logs
```
