## High-level description

4 bugs exposed via TDD and fixed before test passed:

| Bug | Root cause | Fix |
|-----|-----------|-----|
| `server.js` not at expected path | `cp -r dir dest/` flattens contents | `mkdir -p dest/dist/blog && cp dir/. dest/dist/blog/` |
| `ERR_MODULE_NOT_FOUND express` | `node_modules/` not copied | `cp -r "${SRC_ROOT}/node_modules/"* "${TARGET}/node_modules/"` |
| `npm run build` broken with `--source` | Symlinks in `.bin/` resolve to non-existent paths | Skip `npm install` when `SOURCE_DIR` is set |
| `/health` returns `{status: "healthy"}` | Server implementation | Changed to `{status: "ok"}` |

Also includes `FirestoreBlogStorage` (storage factory + Firestore implementation) carried from `feat/firestore-storage`.
