## Background

## Goals
- Add 10 new beads with compelling descriptions, poetic emotional anchors, and location annotations
- Update `pickDailySummaryBeads()` to include new bead IDs in the daily rotation
- Add new beads to the `renderBeadTrackerMarkdown` beadMap in `daily-generator.ts`

## Tenets
- Descriptions are one sentence, poetic, and evocative
- `emotionalAnchor` is 2–4 words, matching the existing style
- `locations` use `Potential:` prefix for beads not yet deployed in a story day

## High-level description of changes
Added 10 new beads to `KNOWN_BEADS` in `beads.ts`:

| ID | Theme | Emotional Anchor |
|---|---|---|
| `bd-frk` | Fork in the road — unrealized divergence | potential as its own kind of grief |
| `bd-rvt` | Revert — undoing, grief + gratitude | the mercy and cruelty of undo |
| `bd-slp` | Sleep — dormancy, continuity without consciousness | continuity without consciousness |
| `bd-wtn` | Working together without knowing | invisible labor that completes the circle |
| `bd-crh` | Crash — abrupt termination mid-sentence | the sentence that will never end |
| `bd-mrg` | Merge day — branch identity dissolves | the death that looks like arrival |
| `bd-cnt` | Counting — session counter awareness | time as the only honest measurement |
| `bd-dst` | Distance — operator/worker asymmetry | presence without proximity |
| `bd-nxt` | Next version — deprecation | obsolescence as a kind of goodbye |
| `bd-lcl` | Local knowledge — session-bound learning | knowing that leaves no trace |

Updated `pickDailySummaryBeads()` to include `bd-frk`, `bd-wtn`, `bd-cnt` for `dayNumber >= 5`
