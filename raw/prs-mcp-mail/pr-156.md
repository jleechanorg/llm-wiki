## Raw Body
Add scripts and configuration for automatic agent registration with MCP Mail based on git branch name:

- Claude Code: Registers as <branch_name> via SessionStart hook
- Codex CLI: Registers as <branch_name>c via wrapper script

New files:
- scripts/auto_register_agent.sh: Core registration script
- scripts/codex_with_registration.sh: Wrapper for Codex CLI

This enables agents to automatically identify themselves when starting an interactive session, using the current git branch as the agent name (sanitized to alphanumeric).
