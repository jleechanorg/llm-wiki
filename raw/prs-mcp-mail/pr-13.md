## Raw Body
This commit adds a new MCP tool 'search_mailbox' that allows agents to search through mailbox history using FTS5 full-text search. The tool prioritizes global mailbox results and enables agents to learn from prior conversations before tackling challenging tasks.

Features:
- FTS5-powered full-text search across message subjects and bodies
- Prioritizes global mailbox messages (project-wide visibility)
- Optional agent filter to restrict search to specific agents
- Support for FTS5 query syntax (AND, OR, NOT, phrases, prefix matching)
- Relevance scoring with highlighted snippets
- Configurable result limits and body inclusion

The tool encourages agents to search before starting complex work to discover prior learnings and avoid duplicating efforts.

Test coverage:
- Basic search functionality with keyword matching
- Agent filter to restrict results
- Boolean operators (AND, OR, NOT)
- Global inbox priority in result ranking
- Empty result handling
- Result limit enforcement
