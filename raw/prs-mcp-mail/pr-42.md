## Raw Body
This commit implements automatic inbox fetching after agent registration.

Changes:
- Added optional parameters to register_agent:
  * auto_fetch_inbox: bool (default False) - enables automatic inbox fetching
  * inbox_limit: int (default 20) - max messages to fetch
  * inbox_include_bodies: bool (default False) - include message bodies
  * inbox_urgent_only: bool (default False) - filter for urgent messages only

- When auto_fetch_inbox=True:
  * Automatically calls _list_inbox after registration
  * Returns dict with both 'agent' and 'inbox' keys
  * Maintains backward compatibility (default behavior unchanged)

- Added comprehensive tests:
  * test_register_agent_auto_fetch_inbox: validates basic functionality
  * test_register_agent_auto_fetch_inbox_with_filters: validates filter parameters

All existing tests pass, confirming backward compatibility.
