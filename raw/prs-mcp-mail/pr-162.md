## Summary

Add `scripts/add_global_auto_register.sh` to help users enable MCP Agent Mail auto-registration across all Claude Code projects, not just this repository.

## What It Does

The script safely adds a SessionStart hook to `~/.claude/settings.json` that:
- Auto-registers agents in **any project** that has `scripts/auto_register_agent.sh`
- Only runs in projects with the registration script (safe for all projects)
- Uses the branch-based naming scheme from PR #156

## Implementation Details

- **Safety**: Creates timestamped backup before modification
- **Validation**: Uses `jq` for safe JSON manipulation and validates result
- **Idempotent**: Can be run multiple times safely
- **Smart**: Includes conditional check to only run where applicable

./scripts/add_global_auto_register.sh
