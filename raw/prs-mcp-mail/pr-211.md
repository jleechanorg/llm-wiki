## Summary

Migrates the PyPI publishing workflow to use the organization-wide reusable workflow with smart delta detection.

## Changes

### Workflow Refactoring
- **Replaced inline workflow** with reusable workflow from `jleechanorg/.github`
- **Centralized maintenance**: Updates to publishing logic now happen in one place across all repos

### Smart Delta Detection
- **Automatic skip logic**: Only publishes when source files actually change
- **Watch paths**: Monitors `src/`, `pyproject.toml`, `uv.lock`, `README.md`, `LICENSE`
- **Force publish option**: Manual override via `workflow_dispatch` when needed

### Benefits
- ✅ Reduced duplicate workflow code across repositories
- ✅ Prevents unnecessary PyPI publishes when only docs/CI change
- ✅ Consistent publishing behavior across organization
- ✅ Manual trigger option for edge cases
