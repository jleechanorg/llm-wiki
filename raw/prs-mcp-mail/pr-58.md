## Raw Body
Adds utility script to resolve FastMCP serialization issues when capturing comprehensive test evidence. FastMCP Client returns Root() wrapper objects that aren't dict-subscriptable or JSON-serializable, causing inbox fetch results to be empty arrays. This script uses direct SQLite queries to bypass FastMCP and properly serialize all message fields.
