## Issues Fixed

### 🔴 CRITICAL: Metadata Not Persisted
- **Problem**: Metadata sent via SendMessageParams was lost during Firestore persistence
- **Root Cause**: First message creation didn't store metadata; retrieval didn't read it
- **Fix**: Updated 6 locations across 3 files to persist and retrieve metadata

### 🟡 MINOR: Content Trimming (1 char loss)
- **Problem**: Content trimmed by 1 character (4200 → 4199)
- **Root Cause**: `content.trim()` removing trailing whitespace
- **Fix**: Validate on trimmed content but preserve original
