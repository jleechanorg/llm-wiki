## Client Migration Guide

Clients should update to use the returned `timestamp` field instead of calling `get-history` immediately:

**Before:**
```typescript
const result = await sendMessage({ userId, content, role });
const history = await getHistory({ userId, conversationId: result.conversationId }); // ❌ Race condition
```

**After:**
```typescript
const result = await sendMessage({ userId, content, role });
// Use result.timestamp, result.messageId, result.sequence directly ✅
```
