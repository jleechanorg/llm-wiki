## Summary

Developed 32 detailed quality criteria for GitHub issues. Merged PRs tend to originate from shorter, well-scoped issues with clear guidance. Issues referencing external dependencies correlated with lower merge rates. 72% AUC for predicting merge likelihood.

## Key Claims

- **32 detailed quality criteria** for evaluating GitHub issues
- Merged PRs: shorter, well-scoped, containing clear guidance and hints
- Issues referencing external dependencies/APIs correlated with **lower merge rates**
- **72% AUC** for predicting merge likelihood
