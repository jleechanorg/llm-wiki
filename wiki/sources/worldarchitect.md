# WorldArchitect.AI

AI-Powered Tabletop RPG Platform for D&D 5e. Production-grade AI game master built on MCP architecture with Flask API gateway, Gemini AI, and Firebase Firestore.

## Purpose

Provides an immersive D&D 5e experience where AI acts as game master, managing world state, generating narratives, and handling combat mechanics.

## Key Files

- `/api/main.py` - Flask API gateway
- `/firestore/` - Firebase Firestore integration
- `/mcp/` - MCP server implementations

## How to Use

```bash
cd /Users/jleechan/repos/jleechanorg/worldarchitect.ai
npm install
npm run dev
```
