## Summary
Section J adds the write-side CLI surface to `blog-cli`:
- `delete` — remove a post by ID via `delete_post` MCP tool
- `unregister-repo` — remove a registered repo via `unregister_repo` MCP tool
- `list-repos` — list all registered repos (`--json` for raw JSON)
- `export` — export repo posts/threads to JSON (`--output-file` or stdout)
