## Summary
Python utilities for generating LLM instructions and documentation from JSON Schemas. Part of ADR-0003 Phase 4, Commit 10. Provides functions for resolving $ref pointers, inferring property types, and generating markdown documentation from schema definitions.

## Key Claims
- **Recursive Property Gathering** — `_gather_properties` recursively collects all properties from schema fragments, resolving allOf and $ref references.
- **Reference Resolution** — `resolve_refs` follows $ref pointers to their source in definitions, handling both `#/$defs/` and `#/definitions/` patterns.
- **Type Inference** — `_infer_property_type` infers user-facing type labels when `type` is omitted in schema fragments.
- **Markdown Generation** — `generate_type_markdown` and `generate_schema_documentation` produce formatted documentation blocks from schema definitions.
- **Constraint Extraction** — Automatically extracts minimum, maximum, and enum constraints for display in documentation.

## Key Functions
- `_gather_properties` — Recursively gather properties from schema fragments
- `resolve_refs` — Follow $ref pointers to source definitions
- `_infer_property_type` — Infer type label when explicit type is omitted
- `generate_type_markdown` — Generate markdown documentation for a single type
- `generate_schema_documentation` — Generate full markdown document from schema file
