## Summary

A strategic document outlining the most powerful applications for an LLM Wiki system that transforms raw sources into interlinked Markdown pages. The document describes how the wiki can serve as a self-maintaining second brain for compounding knowledge, with specific applications for WorldAI campaign management, harness engineering documentation, velocity metrics tracking, monetization strategy, and creative world-building. Each application is tailored to existing workflows and provides compounding advantages.

## Key Claims

- The LLM Wiki becomes a living, self-maintaining second brain that compounds like evolve_loop + harness
- Raw sources (transcripts, terminal logs, design docs) auto-convert to interlinked pages
- Future campaigns can start with perfect continuity using wiki as ultimate GM memory
- The wiki can maintain architecture diagrams, decision logs, and lint suggestions for harness systems
- Velocity and output metrics can be tracked and analyzed for workshop and monetization pitches
- Cognitive patterns (like AuDHD advantages) can be documented and shared as teachable knowledge

## Key Applications

### 1. Living WorldAI Campaign Bible
- Drop campaign transcripts, rule systems, dice proofs into raw/
- Auto-creates pages for characters, factions, psychological arcs
- Query: "Generate a new campaign starter that respects every rule system"

### 2. Self-Evolving Harness Laboratory
- Ingest CLAUDE.md, evolve_loop.md, terminal logs, 7-Green rules
- Maintain architecture diagrams and contradiction flags
- Query: "What are the current gaps in evolve_loop for true autonomous agenda generation?"

### 3. Velocity & Output Observatory
- Track GitHub contributions, PR counts, rarity discussions
- Build trend pages and "what worked" analyses
- Query: "Compare side-project velocity this month vs. last"

### 4. Monetization & Product Strategy Command Center
- Synthesize growth plans, subscription tiers, tokenomics
- Cross-reference everything and suggest synergies
- Query: "Prioritize next 3 monetization experiments"

### 5. Cognitive Systems Thinking Archive
- Document AuDHD advantages and harness sustainability
- Build pages on "how my brain works with agents"
- Query: "What cognitive traits make my harness sustainable?"
