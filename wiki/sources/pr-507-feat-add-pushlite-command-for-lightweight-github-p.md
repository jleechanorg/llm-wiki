## Summary
Adds a new `/pushlite` (alias `/pushl`) command for simple GitHub pushes without the full automation of the `/push` command.

### New Command: /pushlite (/pushl)

**Purpose**: Lightweight alternative to `/push` for quick updates and simple operations.

**Usage Options**:
- `/pushlite` or `/pushl` - Push current branch to origin
- `/pushlite pr` or `/pushl pr` - Push and create PR
- `/pushlite force` or `/pushl force` - Force push (with caution)
