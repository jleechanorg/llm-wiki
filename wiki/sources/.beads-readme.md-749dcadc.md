## Summary

Beads is issue tracking that lives directly in your codebase, making it perfect for AI coding agents and developers who want their issues close to their code. No web UI required — everything works through the CLI and integrates seamlessly with git.

## Key Claims

- **CLI-First Design**: Built specifically for AI-assisted development workflows, works seamlessly with AI coding agents
- **Git-Native Storage**: Issues stored in `.beads/issues.jsonl` and synced like code
- **Branch-Aware**: Issues can follow your branch workflow
- **Auto-Sync**: Automatically syncs with your commits
- **Offline Support**: Works offline, syncs when you push
- **Fast & Lightweight**: Stays out of the way

## Key Commands

- `bd create "Issue title"` — Create new issues
- `bd list` — View all issues
- `bd show <id>` — View issue details
- `bd update <id> --status in_progress|done` — Update status
- `bd sync` — Sync with git remote
