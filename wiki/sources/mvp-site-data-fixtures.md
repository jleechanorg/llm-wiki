## Summary

Sample data fixtures for testing campaigns, game states, and AI responses. Provides realistic test data without external dependencies.

- **SAMPLE_GAME_STATE**: Full game state with player, NPCs, world data
- **SAMPLE_STORY_CONTEXT**: Example story entries with actors and text
- **SAMPLE_AI_RESPONSES**: Various response scenarios for validation testing
- **SAMPLE_STATE_UPDATES**: Complex state update examples
- **DISCREPANCY_SCENARIOS**: HP/location/mission discrepancy test cases
