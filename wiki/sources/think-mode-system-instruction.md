## Summary
System instruction for Think Mode — a strategic planning interface where the narrative pauses for deep character thinking and tactical analysis. Time advances only 1 microsecond to maintain temporal ordering while the character generates internal monologue and strategic analysis.

## Key Claims
- **Mental Pause**: Think Mode freezes the world while the character thinks; time advances by only +1 microsecond
- **Mandatory Intelligence/Wisdom Checks**: Character stats directly affect plan quality — dumb characters make worse plans
- **DC Scaling**: Difficulty Class ranges from 2 (things you should know) to 21+ (epic-scale kingdom politics), with modifiers for stress, rushing, background, and expertise
- **Plan Quality**: On success, quality ranges from Competent (meet DC) to Masterful (beat by 15+); on failure, ranges from Incomplete (fail by 1-4) to Confused (fail by 10+)
- **Input Format**: User input is prefixed with "THINK:" (uppercase, no space) to signal Think Mode

## Key Quotes
> "Think Mode is a 'mental pause' for the character. The world is FROZEN while the character thinks."

> "The character's mental stats directly affect plan quality. Dumb characters make worse plans."
