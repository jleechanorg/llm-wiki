## Summary
This PR adds separate safety limits for each of the 4 automation workflow types:
1. **PR automation** (monitoring) - posts codex comments
2. **fix-comment** - addresses review comments  
3. **codex-update** - browser automation (doesn't post PR comments)
4. **fixpr** - resolves conflicts/failing checks
