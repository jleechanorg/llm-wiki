## Summary

Discovers hierarchical agent architectures via multi-armed bandit optimization. Single-agent systems force models to retain irrelevant context. BOAD decomposes into specialized sub-agents (localization, editing, validation) coordinated by an orchestrator. On SWE-bench-Live, 36B system ranked **2nd on leaderboard**, surpassing GPT-4 and Claude.

## Key Claims

- Single-agent systems cause "spurious correlations and poor generalization"
- Human-inspired decomposition into specialized sub-agents improves performance
- Multi-armed bandit (MAB) framework efficiently explores agent hierarchy designs
- **36B ranked 2nd on SWE-bench-Live** leaderboard, surpassing GPT-4 and Claude

## Technique/Method

- Frames agent hierarchy discovery as **multi-armed bandit problem**
- Each arm = candidate sub-agent; reward = collaborative helpfulness
- Orchestrator coordinates specialized sub-agents (localization, editing, validation)
- MAB framework explores designs under limited evaluation budgets

## Results

- Outperforms single-agent and manually-designed multi-agent baselines on SWE-bench-Verified
- **36B ranked 2nd on SWE-bench-Live** leaderboard
