## Summary

Comprehensive reference of names overused by LLMs that cannot be used in any WorldArchitect.AI world content. Includes Primary Banned Names (original list) and Extended Banned Names (top 50 AI overused names), along with replacement history tracking substitutions made.

## Key Claims

- **Primary Banned Names (10)**: Alaric, Blackwood, Corvus, Elara, Kaelen, Lyra, Seraphina, Thorne, Valerius, Isolde — originally banned as overused fantasy names
- **Extended Banned Names (50)**: Additional 50 names including Aiden, Aria, Atlas, Caspian, Dante, Ember, Iris, Luna, Orion, Phoenix, Raven, and more — all commonly generated by AI
- **Replacement Tracking**: History maintained for all substitutions made to banned names in existing world content
- **Total Banned**: 60 unique names across both lists
