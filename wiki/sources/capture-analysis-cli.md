## Summary
Command-line interface for analyzing captured service interactions and validating against mock responses. Provides five subcommands: analyze, compare, baseline, cleanup, and list.

## Key Claims
- **Analyze Command**: Analyzes captured data from JSON files within a specified number of days
- **Compare Command**: Validates captured responses against mock files, reporting accuracy scores
- **Baseline Command**: Generates mock baseline files from captured data for future comparisons
- **Cleanup Command**: Removes capture files older than specified days
- **List Command**: Lists available capture files with size and modification timestamps
