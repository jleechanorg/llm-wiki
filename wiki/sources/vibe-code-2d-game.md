## Summary
"Vibe coding" is a rapid 2D game development methodology using LLMs for code and sprites, with human direction steering iteration. HTML5 Canvas + Vanilla JS is the proven stack. Ship playable games in days, not months.

## Key Claims
- Full game shipped in 2 weeks with Claude Code (capybara delivery game)
- Character + mechanics beats graphics — gameplay > pixel-perfect art
- LLM assists, human steers — use AI for code/sprites, human decides direction
- HTML5 Canvas + Vanilla JS is the simplest, most portable 2D game stack
- Sprite pipeline ranked: ComfyUI SDXL > Pollinations > LPC CC0 > Grok API
- 8-frame walk cycles via sine-wave bob + leg alternation is the animation standard
- State machine pattern for character states: idle, walk, run, attack
- Canvas compositing for layered sprites (body parts assembly)
- 2-week timeline is proven for indie game prototyping
