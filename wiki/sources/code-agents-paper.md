## Summary

This paper proposes a benchmark for evaluating code agents' progress toward AGI. The benchmark focuses on the kinds of coding tasks that require human-level reasoning, planning, and multi-step problem-solving — the core capabilities needed for general AI.

## Key Claims

- Code agents as a proxy for AGI progress measurement
- Focus on tasks requiring human-level reasoning and planning
- Multi-step problem-solving as key AGI benchmark criterion
- Current LLMs show significant gaps on complex code understanding

## Technique/Method

- Task suite covering various coding difficulty levels
- Multi-file understanding requirements
- Long-range dependency tracking
- Planning and architecture reasoning

## Connections

- Relevant to agent evaluation framework design
- Evidence for coding as a key AGI proxy task
