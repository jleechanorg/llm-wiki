## Summary
Core mechanical reference for D&D 5E rules authority. Establishes that SRD rules override narrative preferences, provides proficiency bonus tables by level, and defines all standard roll formulas (attacks, saves, skills, damage, spell DCs). Includes class save proficiency tables and worked examples for a Level 1 Bard.

## Key Claims
- **Mechanical Authority**: D&D 5E SRD rules override narrative preferences
- **Default Attributes**: Six abilities (STR, DEX, CON, INT, WIS, CHA) with modifier calculation `(Score - 10) / 2`
- **Proficiency Scaling**: +2 at levels 1-4, +3 at 5-8, +4 at 9-12, +5 at 13-16, +6 at 17-20
- **Expertise**: Doubles proficiency bonus for selected skills

## Key Formulas
| Roll Type | Formula |
|-----------|---------|
| Attack | d20 + ability mod + prof (if proficient) |
| Save | d20 + ability mod + prof (if proficient) |
| Skill | d20 + ability mod + prof (if proficient) |
| Initiative | d20 + DEX mod |
| Spell DC | 8 + prof + casting mod |

## Class Save Proficiencies
Each class grants proficiency in two specific saving throws, representing their core defensive strengths.
