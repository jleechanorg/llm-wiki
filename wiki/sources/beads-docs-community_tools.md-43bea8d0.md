## Summary

A curated list of community-built UIs, extensions, and integrations for Beads (AI-native issue tracking system). Organized into four categories: Terminal UIs (7 tools), Web UIs (8 tools), Editor Extensions (6 tools), and Native Apps (3 tools). Each tool is ranked by activity and maturity.

## Key Claims

- **Terminal UIs (7 tools)**: beads_viewer (Go), bdui (Node.js), perles (Go with BQL), beads.el (Elisp), lazybeads (Go/Bubble Tea), bsv (Rust), abacus - keyboard-driven and tree navigation interfaces
- **Web UIs (8 tools)**: beads-ui, Monitor WebUI, beads-viz-prototype, beads-dashboard, beads-kanban-ui, beads-pm-ui, Beadspace, beadsmap — local web interfaces with kanban, gantt, and visualization features
- **Editor Extensions (6 tools)**: vscode-beads, ANAL Beads, Beads-Kanban, opencode-beads, nvim-beads, beads-manager — IDE integrations for VS Code, OpenCode, Neovim, and JetBrains
- **Native Apps (3 tools)**: Beads Task-Issue Tracker (Tauri/Vue, cross-platform), Beadster (Swift/macOS), Parade (Electron) — desktop applications for issue management
- **Zero-config deployment**: Many tools run locally with npx, single HTML files, or minimal setup
- **Diverse technology stack**: Go, Node.js, Rust, TypeScript, Python, Elisp, Lua, Swift, Tauri, Electron
