## Summary

Analysis of AI coding assistant command invocations over 30 days (2026-02-21 to 2026-03-23). Scanned 27,305 conversation files with 2,198 messages containing invocations across 190 tracked commands. Top commands are /copilot (552), /claw (363), /e (236), and /er (203). Notably, 131 commands had zero usage during this period, suggesting potential capability gaps or undiscovered features.
