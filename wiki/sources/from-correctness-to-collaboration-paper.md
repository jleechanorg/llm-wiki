## Summary

Proposes Context-Adaptive Behavior (CAB) Framework for evaluating AI agents. Current benchmarks miss nuanced interactive behaviors. Four key behavioral expectations: Adhere to Standards, Ensure Code Quality, Solve Problems Effectively, Collaborate with User.

## Key Claims

- Current benchmarks fail to capture nuanced interactive behaviors
- Behavioral expectations shift along: **Time Horizon** and **Type of Work** axes
- Four key expectations: Adhere to Standards, Ensure Code Quality, Solve Problems Effectively, Collaborate with User
- CAB Framework provides human-centered foundation for evaluating AI agents

## Method

- Analyzed 91 sets of user-defined agent rules
- Interviews with 15 expert engineers
- Prompt analysis of prototyping agent
