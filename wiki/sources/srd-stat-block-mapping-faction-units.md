## Summary
Mapping system that resolves genre-agnostic faction unit labels (e.g., modern or sci-fi) into a small set of D&D 5.1 SRD creature archetypes for simulation purposes. Provides scaling functions for elite units based on D&D character levels.

## Key Claims
- **Genre-Agnostic Mapping**: Unit names from any genre (modern, sci-fi, fantasy) resolve to the same SRD archetypes
- **Small Archetype Set**: Only 8 canonical archetypes (guard, veteran, spy, scout, knight, assassin, gladiator, elite_6/elite_10)
- **Keyword Matching**: Substring-based matching allows flexible naming (e.g., "marine"→soldier, "sniper"→assassin)
- **SRD Compliance**: All creatures from D&D 5.1 System Reference Document under open license

## Key Quotes
> "This mapping is intentionally generic: unit names can come from any genre or era, but they must resolve to a small set of SRD archetypes for simulation."
