## Summary
Character profile framework for D&D 5e and RPG character creation. Provides structured template for developing NPCs with core identity, psychology, behavior patterns, backstory, and game mechanics. Uses behavioral expression over labels—personality shown through actions, not MBTI/alignment tags. Includes deep psychological framework for significant NPCs with persona/break-point analysis.

## Key Claims
- **Behavioral Expression**: Show personality through specific actions, objects, and speech patterns—not labels
- **Internal-Only MBTI/Alignment**: Stored in master_directive.md, never exposed in narrative
- **Break-Point Analysis**: Catalyst, fractures, immediate cost, and potential liberation from overcoming
- **Relational Script**: "If I... then they will..." core belief driving character relationships
