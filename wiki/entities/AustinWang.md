## Profile

Co-creator of [[CMUX]], a native macOS terminal application specifically engineered for AI-agent workflows.

## Background

- Co-creator of CMUX
- Built on top of libghostty (GPU-accelerated terminal rendering)
- Native Swift/AppKit implementation
- Eschews Electron-based IDE overhead

## Key Contributions

CMUX features for AI workflows:
- Vertical workspaces showing directory, git branch, and agent output
- Unix domain sockets for near-zero latency IPC
- Notification rings and badges for agent status
- In-app WebKit browser for DOM inspection
