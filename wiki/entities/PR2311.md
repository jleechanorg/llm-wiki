## Summary
GitHub Pull Request #2311 introduced adaptive truncation behavior for the context truncation system. With extremely low token budgets (~30 tokens), the system now aggressively reduces turns and hard-trims content to guarantee the budget is respected.

## Key Changes
- **Adaptive Budget Allocation**: Recent turns receive 60% of budget, older turns receive 25%
- **Hard-Trimming**: Content is physically truncated to fit within budget when under extreme pressure
- **Guaranteed Budget Fit**: Result always fits within specified character budget
