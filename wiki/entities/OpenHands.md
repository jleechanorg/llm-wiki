## Overview

OpenHands (All-Hands-AI) is an AI coding agent achieving 77.6% on SWE-bench. It provides multiple deployment options (SDK, CLI, Local GUI, Cloud, Enterprise) and enterprise features including Slack/Jira/Linear integrations.

## Key Properties

- **SWE-bench score**: 77.6%
- **Architecture**: Multiple deployment options — SDK, CLI, Local GUI, Cloud, Enterprise
- **Key paper**: arXiv 2511.03690
- **Components**: Software Agent SDK (composable Python library), CLI, REST API + React app
- **Enterprise features**: Slack/Jira/Linear integrations, multi-user support, RBAC
- **License**: MIT (except enterprise/)
