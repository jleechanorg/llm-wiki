## Description
Browser automation framework used for end-to-end testing in the codebase. Enables programmatic control of Chromium (and other) browsers for UI verification tests.

## Connections
- Used by [[TestUIDisplayVerification]] for UI testing
- Part of testing infrastructure for [[E2ETests]]
