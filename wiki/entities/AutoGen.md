## Overview

AutoGen is a Microsoft Research framework for building multi-agent applications. It provides a layered API design (Core API, AgentChat API, Extensions API) supporting two-agent chats, group chats, and event-driven message passing.

## Key Properties

- **Architecture**: Layered API design (Core API, AgentChat API, Extensions API)
- **Multi-agent patterns**: Two-agent chats, group chats, event-driven message passing
- **Tools**: AutoGen Studio (no-code GUI), AutoGen Bench (benchmarking)
- **Status**: Maintenance mode; Microsoft recommends Microsoft Agent Framework for new projects
- **Key paper**: arXiv:2308.00352
