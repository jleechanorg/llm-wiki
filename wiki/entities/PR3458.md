## Summary
Pull request that intentionally removed JSON recovery behavior for malformed and truncated JSON responses. Previously, the system would attempt partial recovery from invalid JSON. After this PR, malformed/truncated JSON returns error responses instead.

## Key Changes
- Removed partial JSON recovery for malformed JSON responses
- Removed partial JSON recovery for truncated JSON responses
- System now returns error response when JSON parsing fails
