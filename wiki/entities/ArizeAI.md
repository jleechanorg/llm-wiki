## Overview

Arize AI is an ML/AI observability platform with drift detection. It provides real-time monitoring for model degradation, concept drift, and data drift in production deployments.

## Key Properties

- **Drift Detection**: Monitors performance degradation and distribution changes
- **Concept Drift**: Monitors for distribution changes in model interpretable features
- **Data Drift**: Input distribution monitoring
- **PSI (Population Stability Index)**: Drift measurement metric
- **Root cause analysis**: Drills down to root cause of ML problems
- **Open source**: Phoenix OSS for local deployment
