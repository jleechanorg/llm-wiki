## Summary
Google Chrome is a web browser used for automated testing via Selenium WebDriver. In test automation contexts, Chrome is often run in headless mode for faster execution without visible UI.

## Key Details
- **Headless Mode**: Chrome can run without visible window for automated testing
- **Common Flags**: --headless, --no-sandbox, --disable-dev-shm-usage, --disable-gpu
- **WebDriver**: ChromeDriver provides WebDriver protocol implementation
