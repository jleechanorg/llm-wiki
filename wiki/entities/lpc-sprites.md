## Overview
LPC (Liberated Pixel Cup) is a CC0-licensed open source sprite library hosted on OpenGameArt. Contains dual-licensed (CC BY-SA 3.0 + GPLv3) 64x64 pixel art sprites in 9x4 spritesheet layout (direction × animation frame).

## Key Properties
- **Type:** Open source sprite library
- **License:** CC0 / CC BY-SA 3.0 + GPLv3
- **Sprite size:** 64x64 pixels
- **Layout:** 9 directions × 4 animation frames
- **Host:** lpc.opengameart.org
- **Best for:** Medieval/fantasy RPG characters, weapons, armor
