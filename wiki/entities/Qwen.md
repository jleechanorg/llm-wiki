## Overview
Qwen is a large language model family developed by Alibaba Cloud, available via Cerebras Cloud. Known for fast inference speeds and strong code generation capabilities.

## Key Attributes
- **Developer**: Alibaba Cloud
- **Availability**: Cerebras Cloud, OpenRouter
- **Use Cases**: Code generation, instruction following
- **Performance**: Fast generation times (500ms benchmarked)
