## Description
The primary combat system in [[Sovereign Protocol]]. Conflict between Sovereigns is resolved through their unique metaphysical laws (Logic).

## Combat Formula
**Attack:** `d20 + GP + SP Expenditure vs Target Defense`

## Modifiers
| Factor | Modifier |
|--------|----------|
| Logic Advantage (opposed) | +5 |
| Logic Disadvantage (similar) | -5 |
| Sub-Deity Support (War) | +GP |
| Defending Home | +3 |
| Hostile Territory | -3 |

## Outcomes
- **Success:** Target loses SP, attacker claims universes
- **Failure:** Attacker loses SP (Logic Rot)
