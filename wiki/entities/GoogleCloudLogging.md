Google Cloud Logging (formerly Stackdriver Logging) is accessed via the `ops_gcloud_logs_read` tool in the WorldAI Tools MCP Proxy. This tool allows reading Cloud Logging entries through an allowlisted gcloud wrapper for diagnostic and operational purposes.
