**PR #6404** — Implementation branch `feat/zfc-level-up-model-computes` on jleechanorg/worldarchitect.ai.

## Purpose

Adds the ZFC-compliant contract: model computes level-up facts, backend validates and formats explicit fields, UI renders only server-provided payloads.

## Key Changes

- `previous_turn_exp`: total character XP before current user turn
- `current_turn_exp`: total character XP after current user turn
- Backend computes deterministic display values (`xp_gained`, `progress_percent`) from explicit totals
- Legacy backend recovery still exists in transitional state
