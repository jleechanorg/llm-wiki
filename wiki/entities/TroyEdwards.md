## Profile

Creator of [[AgentLoop]], an orchestration platform that merges classical deterministic AI with modern probabilistic LLMs.

## Background

- Creator of AgentLoop
- Presented at OpenClaw workshop on behavior trees and orchestration
- AgentLoop visualizes work via Sprint-like Kanban board
- Agents operate continuously like NPCs in a video game

## Key Innovation

AgentLoop's core innovation is using Behavior Trees:
- Forces LLM to output structured JSON at each decision node
- Deterministic programmatic scripts verify the fields
- Reduces hallucinations by 80-90%
- Makes autonomous "auto-YOLO" PR merging viable
