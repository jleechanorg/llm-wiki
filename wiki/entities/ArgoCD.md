## Overview

ArgoCD is a declarative, GitOps continuous delivery tool for Kubernetes. It provides automated deployment with Git-based approval workflows.

## Key Properties

- **GitOps**: Declarative Git-based deployment
- **Kubernetes**: Kubernetes-native deployment tool
- **Declarative**: Application definitions, configuration, and environment in Git
- **Automated sync**: Automatically syncs app state with Git manifest
