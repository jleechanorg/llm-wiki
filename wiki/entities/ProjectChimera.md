## Overview

A dynamic "neural network" of LLM agents where frontier models act as neurons and a GNN learns optimal communication topologies between them. Authored by Jeffrey, Version 1.1 (April 2026). Primary application is a Living Knowledge Wiki.

## Key Details

- **Author:** Jeffrey
- **Version:** 1.1
- **Date:** April 2026
- **Primary Model:** MiniMax M2.7 (with selective GPT-5.4 for synthesis/gate)
- **Budget:** $30–80/month estimated
- **Goal:** 22-agent system beating single-model by ≥20% on quality metrics

## Architecture

5-layer system: Input → Dynamic Topology (GNN) → Execution → Memory/Knowledge → Output/Evaluation
