## Overview

Cole (GitHub: coleam00) is the creator of Archon, a TypeScript workflow engine for AI coding agents. He produced a 2-hour live stream demonstrating the "Dark Factory" — a governance-heavy autonomous issue processing system built on Archon.

## Archon

- ~18K GitHub stars
- v0.3.6, single developer
- TypeScript/Bun monorepo
- Feb 2025–present

## Dark Factory Live Stream

Cole's dark factory stream (https://www.youtube.com/watch?v=Xg0tNz9pICI) demonstrates:
- 3-worktree Archon pipeline: triage → implement → validate
- `mission.md` + `factory-rules.md` governance layer
- 320 lines of governance rules
- Self-described as "level 5 autonomy" but actually level 4

### What Cole Admits

On his own stream he says:
> "I know I already said this, but it's worth repeating. You're not going to get the most reliable results with your coding agent when you give it this much autonomy."

## Honest Assessment (from analysis)

- Governance-first approach is the right instinct
- Execution is rough around every edge
- YAML workflow model, governance layer, holdout pattern — all good ideas
- Premature as a production system (2h live stream proof-of-concept)
- Level 4 system with level 5 branding
