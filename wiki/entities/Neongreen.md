## Overview
@neongreen (Emily) is the author of beads-merge, an open-source 3-way merge library that was vendored into the bd project's `internal/merge/` directory.

## Connections
- [[beads-merge]] — the vendored 3-way merge library
- [[Beads Changelog]] — related project updates

## Notes
The beads-merge library provides field-level 3-way merge logic, issue identity matching, dependency/label merging with deduplication, and conflict marker generation.
