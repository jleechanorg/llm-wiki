## Description
An LLM model served via OpenRouter that provides more creative/risky AI responses. Activated when spicy mode is enabled in the game settings.

## Behavior
- Accessed through OpenRouter provider
- Used for more unconventional narrative responses
- Replaced when spicy mode is disabled
