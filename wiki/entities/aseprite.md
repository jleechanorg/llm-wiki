## Overview
Aseprite is a pixel art editor with animated sprite support, onion skin animation, sprite sheet export, and CLI batch processing. 36.8k GitHub stars. Premium software (one-time purchase) but considered the gold standard for pixel art creation.

## Key Properties
- **Type:** Software tool / pixel art editor
- **Stars:** 36.8k (GitHub)
- **Features:** Animated sprite editing, pixel-perfect stroke, onion skin, sprite sheet export, rotsprite rotation, CLI batch mode
- **Platform:** Windows, macOS, Linux
- **Price:** One-time purchase (no subscription)
