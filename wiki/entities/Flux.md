## Overview

Flux is a CNCF-owned GitOps tool for Kubernetes cluster configuration management. It provides automated deployment with Git-based policy enforcement.

## Key Properties

- **GitOps**: Git-based cluster configuration and deployment
- **CNCF**: Cloud Native Computing Foundation project
- **Automated sync**: Automatically reconciles cluster state with Git
- **Policy enforcement**: Git-based policy at deployment time
