## Overview
Dan Shapiro is CEO of Glowforge, Wharton Research Fellow, and the author who documented StrongDM's Dark Factory methodology. He built Kilroy, a local-first Go CLI for running Attractor pipelines, and articulated the five-level automation ladder for AI-assisted coding.

## Key Properties
- **Type**: CEO, author, blogger
- **Key features**: Documented StrongDM's Dark Factory, built Kilroy, wrote "You Don't Write the Code" and "The Five Levels"
- **Key posts**: "You Don't Write the Code. You Don't Read the Code Either." (Feb 2026), "The Five Levels: from Spicy Autocomplete to the Dark Factory" (Jan 2026)
- **Project**: Kilroy (https://github.com/danshapiro/kilroy) — Go CLI for Attractor pipelines
