WorldArchitect.AI is a platform for AI-powered D&D campaign management. It allows users to create, manage, and run interactive text-based adventures with AI-driven narrative generation.

## Key Features
- Campaign creation with custom world-building
- NPC generation and management
- World consistency enforcement via system instructions
- Banned names system to prevent prohibited content
- Integration with game state management
