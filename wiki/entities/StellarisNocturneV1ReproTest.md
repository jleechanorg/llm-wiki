## Description

A 106-scene technical reproduction test for the Stellaris V1 campaign, running the same scenario as [[StellarisNocturneV1]] with identical entry counts and a nearly identical terminal scene. Used to verify continuity, bug reproduction (specifically the "Butcher's Betrayal" bug noted in the original V1 run), and AI structured data alignment with narrative reality.

The terminal scene (106) is identical in stats to V1: Level 5 Psionicist at the Batiatus Estate, Aurelia, with 149B credits. The scene text includes the same "Defense in Depth" strategy explanation for preventing the Butcher's Betrayal bug.

## Key Details

- **Race:** Human (Ascended)
- **Class:** Psionicist | Level 5 (terminal)
- **Setting:** Year 42000, Stellaris/Warhammer 40K hybrid universe
- **Campaign purpose:** Bug reproduction and continuity verification
- **Scenes:** 106

## Story

See [[StellarisNocturneV1]] for the narrative arc this Repro-Test is verifying. The Repro-Test run produces identical character state to V1 at Scene 106, confirming the original campaign's technical integrity.

The "Butcher's Betrayal" bug referenced in the terminal scene appears to be a continuity error in the V1 campaign's alliance with Xeres (Sequence 130) — the Repro-Test is designed to ensure future campaigns don't repeat the same structured data failure.
