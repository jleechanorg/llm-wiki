## Overview

Microsoft Agent Framework is Microsoft's recommended multi-agent framework for new projects, succeeding AutoGen (which is in maintenance mode).

## Key Properties

- **Vendor**: Microsoft
- **Status**: Recommended for new projects (replacing AutoGen)
- **Relation to AutoGen**: AutoGen is in maintenance mode; Microsoft Agent Framework is the successor
