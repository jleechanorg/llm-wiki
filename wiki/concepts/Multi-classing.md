## Summary
A D&D 5e rule allowing characters to take levels in multiple classes. Each class provides distinct features and progression.

## Key Details
- Characters must meet prerequisite ability scores to multi-class
- Proficiency bonus is based on total character level, not individual class levels
- Spellcasting ability for multi-class depends on which classes have spellcasting
- get_spellcasting_ability parses "/" separators to handle multi-class names
