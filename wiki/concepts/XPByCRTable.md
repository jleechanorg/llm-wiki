## Definition
Experience point awards based on creature Challenge Rating, used for awarding XP when enemies are defeated in combat or through non-combat means (executions, ambushes, trap kills, quests).

## XP Table
| CR | XP |
|---|---|
| 0 | 10 |
| 1/8 | 25 |
| 1/4 | 50 |
| 1/2 | 100 |
| 1 | 200 |
| 2 | 450 |
| 3 | 700 |
| 4 | 1100 |
| 5 | 1800 |

## Key Rules
- Non-combat kills (executions, ambushes, trap kills) MUST award XP + loot using same CR table
- Narrative events (quests, milestones, social victories) MUST display XP + rewards
- DM Reward Check: If roll ≥ DC → award XP (Skill=50-200, Trap=100-300, Knowledge=100-500, Persuasion=100-500). If roll < DC → award 0 XP.
