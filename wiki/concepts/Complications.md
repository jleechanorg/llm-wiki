## Description
Events that introduce challenges or obstacles in the campaign. Generated by the Living World system.

## Fields
- **type** — The kind of complication (e.g., information_leak)
- **severity** — How impactful (e.g., regional)
- **description** — What the complication is

## Example
```python
{
    "type": "information_leak",
    "severity": "regional",
    "description": "Your plans were discovered",
}
```

## Connections
- [[LivingWorld]] — generates complications
- [[DebugEventsExport]] — exports complications in campaign exports
- [[ChallengeSystem]] — related to complication handling
