Type stubs are .pyi files that provide type information for libraries without inline types. Required stubs for WorldArchitect.AI include types-flask and types-requests.

## Installation
Listed in requirements.txt:
- `types-flask` - Flask type stubs
- `types-requests` - Requests type stubs
