Human-centered framework for evaluating AI agents in software engineering. Current correctness-focused benchmarks miss nuanced interactive behaviors essential for human-AI partnership.

## CAB Framework

Context-Adaptive Behavior (CAB) Framework:

**Four Key Expectations:**
1. Adhere to Standards and Processes
2. Ensure Code Quality and Reliability
3. Solve Problems Effectively
4. Collaborate with the User

**Two Behavioral Axes:**
- **Time Horizon**: immediate needs → future ideals
- **Type of Work**: enterprise production → rapid prototyping
