Local Inference refers to running AI models on local hardware rather than calling cloud APIs. This is typically done via an HTTP gateway that exposes the model as an API endpoint.

## In WorldAI
- OpenClaw Gateway provides local inference option
- Default endpoint: http://127.0.0.1:18789
- Eliminates API costs for open-source models
- Requires compatible hardware
