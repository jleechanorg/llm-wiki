# HTML5 Canvas

Browser-based 2D rendering platform. The simplest, most portable option for 2D game development.

## Why Canvas for Games
- Runs anywhere (browsers are universal)
- No framework overhead (Vanilla JS)
- Hardware accelerated via WebGL under the hood
- Simple sprite compositing API

## Common Patterns
- 8-frame walk cycle animation
- State machine for character states
- Canvas layering for body-part compositing
