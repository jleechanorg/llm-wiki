## Overview
The Imperator Class is a custom D&D 5e class for the Rome Pax Julia campaign. It represents Roman military and political leadership, powers tied to command, authority, and divine favor.

## Class Features
- **Command Presence:** Authority that shapes battlefield and social encounters
- **Political Maneuvering:** Bonus to political/social checks
- **Divine Parasite Integration:** Powers from the embedded divine entity

## Context
The Imperator represents Rome's leadership class elevated to heroic status. Combined with the divine parasite, it creates a uniquely Roman heroic framework.
