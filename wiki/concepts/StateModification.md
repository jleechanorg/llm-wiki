## Description
GOD MODE allows direct modification of campaign state including player character data (HP, gold), game state, and story text. The state_updates field in responses contains changes to persist.

## Modification Types
- Character stats (HP, gold, level)
- Location changes
- Story text updates
- Entity modifications
