Technique for capturing and preserving actual LLM outputs during testing. Used to verify LLM behavior, debug response quality, and analyze character consistency.

## Methods
- **Flask Test Client** — In-process API calls without network overhead
- **Campaign Interactions** — Execute predefined interaction sequences
- **JSON Serialization** — Store responses with metadata for analysis
