## Summary
The Destiny Core Rules is a custom tabletop role-playing game system used for the daemon-conquers campaign. It features character development through Aptitudes and Personality Traits, distinguishing it from mainstream systems like D&D 5e.

## Key Details
- **System Type:** Custom fantasy TTRPG
- **Setting:** "A land of dragons" — generic fantasy world
- **Design Focus:** Character-driven narrative with special trait systems

## Key Features
- **Aptitudes:** Character traits that define capabilities
- **Personality Traits:** Character development system shaping behavior and decisions
- **Character Classes:** Specialized paths (like "knight" for Daemon)

## Campaign Relevance
Daemon's brute-strength knight build would be defined through Aptitudes and Personality Traits that emphasize his physical power and ambition toward kingship. The system supports an evil protagonist's journey toward dominion.
