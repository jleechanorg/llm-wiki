## Definition
A design system is a collection of reusable components, guided by clear standards, that can be assembled to build any number of applications. It includes design tokens, component patterns, and documentation.

## Context in this source
The Modern CSS Foundation serves as a design system with:
- **Design tokens**: Primitive values (colors, typography, spacing)
- **Component tokens**: --card, --button, --input styling
- **Theme tokens**: Light vs fantasy theme variants
- **Utility tokens**: Shadows, transitions, z-index scale
