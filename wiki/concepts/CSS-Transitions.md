## Definition
CSS transitions provide a way to animate changes to CSS property values smoothly over a specified duration, instead of having changes take effect immediately.

## Usage in Source
The story pagination styles use `transition: all 0.3s ease` for:
- Button hover states (transform, box-shadow)
- Container opacity transitions
- Scrollbar thumb color changes
