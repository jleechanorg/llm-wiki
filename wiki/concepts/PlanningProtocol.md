## Definition
Planning Protocol is the canonical schema reference for the planning_block structure used in Think Mode. It defines valid risk levels ({{VALID_RISK_LEVELS}}) and choice structure.

## Purpose
Provides the formal schema for:
- planning_block structure in Think Mode output
- Valid risk levels for choices
- Choice format (pros, cons, confidence)

## Related Concepts

- [[PlanQuality]] — affects how detailed the planning_block becomes
- [[DifficultyClass]] — determines baseline complexity of the plan
