## Definition
Google Cloud Firestore is a NoSQL document database that stores data in documents, which are organized into collections. Used in the WorldArchitect MVP for persisting game state, story entries, and user settings.

## Key Characteristics
- Document-based storage with collections
- Real-time capabilities
- Supports nested data structures
- Used for game state, story entries, user profiles
