Multi-LLM agent system for autonomous design, planning, and execution of scientific experiments. Demonstrated on catalyzed cross-coupling reactions. Pioneer work on multi-LLM coordination and safety considerations.

## Core Loop

```
Plan → Execute → Verify → Iterate
```

## Key Components

1. **Multi-LLM coordination**: different models handle planning, execution, verification
2. **Tool use**: accessing external knowledge (literature, databases)
3. **Feedback-driven iteration**: refinement based on experiment results
4. **Safety guardrails**: prevent misuse of autonomous capabilities
