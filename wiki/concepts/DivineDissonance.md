## Description
The core tension mechanic in the Divine Leverage System. Using True Power while wearing a Mortal Mask creates a detectable disturbance in reality's fabric. If Dissonance rises too high, the deception fractures, alerting the Pantheon to an intruder. Each Layer has a Safe Limit that can be exceeded at risk.
