Permissive open-source license family that allows redistribution of source and binary code with attribution, but with minimal restrictions. All four Chromium components use BSD-style licenses.

## Key Characteristics
- Permits proprietary use
- Requires attribution in redistributions
- Disclaims warranties (AS IS)
- Limits liability of contributors
- No copyleft requirement (unlike GPL)
