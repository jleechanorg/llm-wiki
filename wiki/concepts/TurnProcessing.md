Turn Processing is the systematic workflow for handling combatant actions. The fix defines mandatory ally turn processing steps:

1. Announce ally turn with HP
2. Choose tactical action
3. Roll dice via tool_requests
4. Narrate outcome
5. Update state

Player input boundaries: only process player actions on their turn.
