## Definition
The process of creating a player character in D&D 5e, handled as META-GAME only—no narrative should occur until explicit player approval.


## Three Creation Methods
1. **AIGenerated**: AI creates a complete character sheet with justification
2. **StandardDND**: Player selects from standard D&D options (races, classes, backgrounds)
3. **CustomClass**: Player creates a custom class with homebrew features

## Creation State Tracking
Seven explicit steps tracked:
1. Initial Choice (1, 2, or 3 for creation method)
2. Race Selection (if StandardDND)
3. Class Selection (after race)
4. Ability Scores (standard array assignment)
5. Background Selection
6. Name & Details
7. Final Approval (MANDATORY)

## Key Requirements
- All 6 ability scores with modifiers
- HP/AC calculation
- Skills selection
- Equipment from background
- Backstory (after approval)
- Banned name precheck for AIGenerated
