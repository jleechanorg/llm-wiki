## Definition
Responsive design uses CSS media queries to adapt layout and styling based on device characteristics like screen width, enabling mobile-friendly interfaces.

## Usage in Source
Mobile breakpoint at 768px:
- Reduces story content max-height from 80vh to 60vh
- Decreases load-more button min-width from 150px to 100px
- Adjusts button font-size to 0.875rem
