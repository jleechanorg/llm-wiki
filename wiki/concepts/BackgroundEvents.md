## Description
Events generated by the Living World system that occur in the background of the campaign. Can be in dict format (with actor, action, event_type, status) or string format (plain description).

## Event Types
- **immediate** — Events that happen right away
- **long_term** — Events that develop over time

## Status Values
- **pending** — Event queued but not yet occurred
- **discovered** — Event revealed to player
- **resolved** — Event completed

## Connections
- [[LivingWorld]] — generates background events
- [[DebugEventsExport]] — exports events in campaign exports
- [[FactionUpdateMechanics]] — related to faction background events
