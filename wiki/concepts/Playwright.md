## Definition
Playwright is a browser automation framework for reliable end-to-end testing. Supports Chromium, Firefox, and WebKit with a unified API.

## Usage in Project
Used for UI integration tests that navigate to campaign URLs and validate structured field display in the story interface.

## Key Features
- Synchronous and async APIs
- Automatic waiting for elements
- Screenshot capture
- Multi-browser support
