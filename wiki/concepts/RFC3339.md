## Definition
Internet Date/Time Format defined in RFC 3339, a profile of ISO 8601 for use in Internet protocols. Specifies date-time format with optional timezone offset (e.g., "2024-01-01T00:00:00Z" or "2024-06-15T12:30:00-08:00").

## Key Features
- Requires timezone (Z for UTC or ±HH:MM offset)
- Supports fractional seconds (microseconds)
- Used in validation module's check_datetime function
