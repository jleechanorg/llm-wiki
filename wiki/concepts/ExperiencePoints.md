## Definition
A numerical measure of character progression in D&D 5e, awarded for defeating enemies, completing challenges, and achieving milestones.

## XP by Challenge Rating Table
| CR | XP |
|---|---|
| 0 | 10 |
| 1/8 | 25 |
| 1/4 | 50 |
| 1/2 | 100 |
| 1 | 200 |
| 2 | 450 |
| 3 | 700 |
| 4 | 1100 |
| 5 | 1800 |

## Award Rules
- **Combat**: Always award XP when enemies are defeated
- **Non-Combat Kills**: Executions, ambushes, trap kills MUST award XP + loot (same CR table)
- **DM Reward Check**: Roll ≥ DC awards XP (Skill=50-200, Trap=100-300, Knowledge=100-500, Persuasion=100-500). Roll < DC awards 0 XP—NO effort XP for failures
- **Narrative Events**: Quests, milestones, social victories MUST display XP + rewards

## Leveling Methods
- **Milestone Leveling**: +1-3 levels per arc recommended
- **Epic/Mythic**: May exceed Level 20
