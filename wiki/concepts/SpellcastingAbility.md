## Summary
The ability score used for spellcasting attacks and saving throws. Determined by character class: INT for Wizard, Intelligence-based Artificer; WIS for Cleric, Druid, Ranger; CHA for Bard, Paladin, Sorcerer, Warlock.

## Key Details
- Multi-class characters use the highest spellcasting class or follow specific priority rules
- get_spellcasting_ability parses class strings like "Fighter/Wizard" to detect INT
- Subclasses like Eldritch Knight and Blood Hunter also use INT
