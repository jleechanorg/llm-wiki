## Definition
The process of extracting structured data (narrative text and state updates) from JSON responses generated by LLMs in the game system.

## Key Components
- **parse_structured_response**: Function that takes a JSON string and returns tuple of (narrative_text, parsed_response_object)
- **State Updates**: Nested dictionary containing `player_character_data`, `npc_data`, `world_data`, and `custom_campaign_state`
- **Narrative Separation**: Critical requirement that state updates don't appear in the narrative output
