## Description
Data structure containing entity information extracted from game state. Used by EntityPreloader to generate entity preload text.

## Structure
- `player_characters` — list of PC objects with name, display_name, hp_current, hp_max, status
- `npcs` — list of NPC objects with name, display_name, hp_current, hp_max, status, location

## Usage
Generated by create_from_game_state() and consumed by create_entity_preload_text() to build prompt injection content.
