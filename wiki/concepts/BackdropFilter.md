CSS property applying graphical effects to the area behind an element. Used in the loading spinner to create a frosted glass effect that maintains context visibility while signaling background activity.

## Syntax
```css
backdrop-filter: blur(8px);
```

## Browser Support
- Modern browsers (Chrome 76+, Safari 9+, Firefox 103+)
- Fallback: semi-transparent background color

## Use Cases
- Modal overlays
- Floating action buttons
- Loading indicators
- Glassmorphism UI theme
