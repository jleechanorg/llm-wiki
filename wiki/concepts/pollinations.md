# Pollinations.ai

Free AI image generation service. Good alternative for sprite generation when budget is constrained.

## Key Properties
- **Cost:** Free
- **Size:** 256x256 (respects width/height params)
- **Speed:** ~2 seconds per image
- **API:** URL-based `https://image.pollinations.ai/prompt/[prompt]?width=256&height=256`

## Use Cases
- Rapid sprite prototyping
- Environment/props generation
- Free option when ComfyUI unavailable
