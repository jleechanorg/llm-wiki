## Summary
Level-up is a character progression mechanic where characters gain levels and new abilities. In WorldArchitect, level-ups trigger modal dialogs for ability selection.

## Key Claims
- **Modal Lock**: Character creation modal locks during level-up to prevent interference
- **Stale Flags**: Previous level-up flags cleared when new level-up available
- **Exit Choice**: User can choose to exit level-up and return to game
