## Definition
A d20 roll to resist or avoid an effect. Represents a character's ability to avoid, endure, or resist threats.

## Formula
```
Save = d20 + Ability Modifier + Proficiency Bonus (if proficient in that save)
```

## Class Save Proficiencies
| Class     | Proficient Saves |
|-----------|------------------|
| Barbarian | STR, CON         |
| Bard      | DEX, CHA         |
| Cleric    | WIS, CHA         |
| Druid     | INT, WIS         |
| Fighter   | STR, CON         |
| Monk      | STR, DEX         |
| Paladin   | WIS, CHA         |
| Ranger    | STR, DEX         |
| Rogue     | DEX, INT         |
| Sorcerer  | CON, CHA         |
| Warlock   | WIS, CHA         |
| Wizard    | INT, WIS         |

## Resolution
- **Success**: Effect avoided or reduced
- **Failure**: Effect applies fully
