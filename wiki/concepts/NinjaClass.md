## Description

The Ninja is a custom D&D 5e class created for the Itachi Evil Campaign, combining elements from multiple D&D classes to simulate ninja capabilities in the Naruto setting.

## In This Campaign

Itachi's Ninja class combines:
- **Assassin Rogue:** Stealth, precision, and death strike mechanics
- **Extra Attack at Level 5:** Two attacks per turn like a martial character
- **All Illusion Spells:** Full wizard illusion spell list — the Sharingan naturally grants illusion mastery

This creates a character who is simultaneously a master of stealth, a deadly combatant, and a powerful illusionist — perfectly matching Itachi's capabilities in the Naruto canon.

## Class Features

The Ninja class would include:
- Sneak Attack (Assassin)
- Extra Attack (at L5)
- Illusion spellcasting (all wizard illusion spells)
- Skill monkey abilities (rogue skills)
- Possible jutsu-style abilities adapted from spells
