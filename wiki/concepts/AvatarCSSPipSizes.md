Avatar CSS pip sizes refer to the visual indicators (pips/dots) used in the avatar selection UI to represent character attributes or options.

## Purpose

Pips are small circular indicators that show:
- Character class options
- Attribute selections
- Available customization choices

## Validation

The test suite validates that `avatar.css` contains the correct pip size definitions to ensure visual consistency across the avatar selection interface.
