Industry-consensus approach combining deterministic pipelines for predictable lifecycle mechanics with LLM for planning and adaptation. The orchestrator implements the "predictability-adaptability frontier" — choosing when each approach applies.

## Industry Examples
- Microsoft Agent Framework: First to support both in same system
- Composio: 30 parallel agents with deterministic reaction engine
- Spotify Honk: Constrained agents with LLM-as-Judge
