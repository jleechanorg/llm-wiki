## Description
Design approach ensuring websites work well on all device sizes. Uses fluid grids, flexible images, and CSS media queries.

## Implementation in WorldArchitect.AI
- Bootstrap's grid system (`container`, `row`, `col-*`)
- `navbar-expand-lg` for responsive navigation
- `container-fluid` for full-width content areas
- Viewport meta tag for proper scaling

## Key Technique
> "<meta name=\"viewport\" content=\"width=device-width, initial-scale=1.0\">" — essential for responsive behavior
