## Definition
CSS styling for the Milestone 4 interactive features including campaign wizard, search filters, personality cards, and modern mode components.

## Key Styles
- .campaign-wizard — wizard container styling
- .search-filter-container — search interface styling
- .personality-card — card component styling
- .modern-mode — modern interface mode styling
