## Definition
Difficulty Class (DC) is set BEFORE rolling in Think Mode based on what the character is trying to plan. It scales from 2 (things you should know) to 21+ (epic-scale kingdom politics).

## DC Scale

| DC | Category | Examples |
|----|----------|----------|
| 2 | Things You Should Know | Your own class abilities, names of party members, current quest objective |
| 2-5 | Easy to Remember | Recent events, established facts about allies, basic local geography |
| 6-8 | Requires Some Thought | Tactical assessment, recalling details from a week ago, understanding NPC motivations |
| 9-12 | Complicated Planning | Multi-step tactical plans, analyzing complex social situations |
| 13-16 | Elaborate Strategies | Heist planning, political maneuvering, military tactics |
| 17-20 | Mastermind-Level | Anticipating counter-moves, multiple contingencies |
| 21+ | Epic-Scale | Kingdom politics, planar schemes, reality-altering plans |

## DC Modifiers
- **Stressed/Combat**: +2 to DC
- **Rushed**: +3 to DC
- **Relevant Background**: -2 to DC
- **Expert Knowledge**: -3 to DC
