## Overview

Tree of Thoughts (ToT) is a deliberate problem-solving framework where an agent explores multiple reasoning paths and self-evaluates each branch before committing to a solution. It extends chain-of-thought by adding systematic exploration and backtracking.

## Key Properties

- **Deliberate exploration**: Agent considers multiple reasoning paths
- **Self-evaluation**: Each path is evaluated before proceeding
- **Backtracking**: Agent can abandon a path and try another
- **Contrast to chain-of-thought**: Chain-of-thought commits to a single path; ToT explores alternatives

## Relation to Governance

ToT-style self-evaluation is relevant to governance evidence verification — an agent can evaluate multiple pieces of evidence and select the most credible path. Also relevant to Skeptic gate's semantic evaluation.
