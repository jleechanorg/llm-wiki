## Description
UI elements that display structured data from AI responses in the campaign story interface. Include session-header, planning-block, dice-rolls, and resources.

## Components
- **session-header**: Displays session/turn metadata
- **planning-block**: Shows AI planning/thinking process
- **dice-rolls**: Renders dice roll results from game mechanics
- **resources**: Displays resource/state information
