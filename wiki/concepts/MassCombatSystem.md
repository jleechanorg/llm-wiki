## Definition
Strategic layer for large-scale battles using unit blocks instead of individual combatants. Implements round-based combat with distinct phases.

## Combat Phases
1. **Initiative**: Order determination
2. **Ranged**: Range attacks first
3. **Charge**: Cavalry/charging units
4. **Melee**: Close combat
5. **Morale Check**: Unit discipline test

## Unit Structure
- **Block**: 10 soldiers as base unit
- **Upkeep Costs**: Infantry (10gp/day), Cavalry (25-40gp/day), War Mages (100gp/day)
- **Categories**: Infantry, Cavalry, War Mages

## Commands
- army status, recruit, disband, pay troops, fortify, forced march, battle plan
