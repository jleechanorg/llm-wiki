# Vibe Coding

Rapid 2D game development methodology using LLMs for code and sprites with human direction steering iteration.

## Core Principles
- **Ship in days, not months** — 2-week timeline is proven
- **Character + mechanics > graphics** — gameplay > pixel-perfect art
- **LLM assists, human steers** — AI generates, human decides
- **Iterate based on play** — get playable, test, improve
