## Definition
Traditional D&D 5e character creation method where players manually select race, class, ability scores (standard array), background, and equipment. One of three options in the Game Mechanics Protocol.

## Creation Steps
1. Select race from PHB options
2. Select class from PHB options
3. Assign standard array (15, 14, 13, 12, 10, 8) to abilities
4. Choose background
5. Get starting equipment
6. Final approval
