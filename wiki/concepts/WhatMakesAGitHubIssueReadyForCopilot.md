32 detailed quality criteria for GitHub issues predicting AI coding agent success. 72% AUC for predicting merge likelihood.

## Key Findings

**High success factors:**
- Shorter and well-scoped issues
- Clear guidance and hints about relevant artifacts
- Implementation direction included

**Low success factors:**
- References to external dependencies
- API or config setup requirements

## The 32 Criteria

Organized into interpretable ML model categories:
- Issue scope clarity
- Guidance specificity
- Artifact reference quality
- Dependency complexity
