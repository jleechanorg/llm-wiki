## Definition
Markdown formatting refers to the syntax used to structure documentation. Various formatting techniques can dramatically affect file density and readability without affecting the rendered output.

## Compression Techniques

### Symbol-Based Formatting
Using Unicode symbols (🚨, ⚠️, ✅, ❌) as compact modifiers instead of full words.

### Pipe-Separated Lists
Converting:
```
- Item 1
- Item 2
- Item 3
```
To:
```
Item 1 | Item 2 | Item 3
```

### Table Conversion
Using markdown tables for structured data instead of verbose prose descriptions.

### Reference Links
Using → to reference external files instead of inlining content.
