## Definition
A UI pattern that allows users to edit content directly on the page without opening a separate edit form. The content becomes editable inline when interacted with, typically showing an input field with save/cancel controls.

## Characteristics
- **In-place editing**: Content transforms to input field without page navigation
- **Visual cues**: Hover states indicate editable elements
- **Immediate feedback**: Save/cancel buttons for committing or discarding changes
- **Error handling**: Inline validation with user-friendly error messages
