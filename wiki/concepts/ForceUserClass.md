## Description

The Force-User is a custom D&D 5e class created for this campaign, combining Star Wars Force mechanics with the D&D ruleset under the "Destiny" ruleset. This class allows a character to wield the mystical energy field known as the Force, adapting it to a fantasy RPG framework.

## In This Campaign

Luke Skywalker uses this custom class in the crossover campaign combining Star Wars with the Baldur's Gate setting. The class features a Light/Dark subclass system that tracks the character's moral alignment.

## Subclasses

- **Light Side Adept:** Powers focused on healing, protection, and light-based abilities
- **Dark Side Corruptor:** Powers focused on destruction, manipulation, and fear

## Core Features

The Force-User class would include:
- Force Points or similar resource system
- Powers/abilities drawn from the Star Wars tradition
- Force Alignment tracking mechanism
