## Definition

Emergent reasoning capability that arises from structured multi-agent collaboration, where the group's collective output exceeds what any individual agent could produce. In Project Chimera, achieved through triadic co-evolution: agents propose new roles, patterns, and architectures; successful proposals are sandbox-tested and promoted automatically.

## Key Mechanisms

- **Structured Collaboration** — Agents with specialized roles contributing to shared goals
- **Learned Topology** — GNN discovers optimal communication patterns rather than hand-crafting them
- **Self-Evolution** — System can propose and test new agent roles and interaction patterns
- **Feedback Loop** — Quality and efficiency signals update the GNN, which updates agent graphs
