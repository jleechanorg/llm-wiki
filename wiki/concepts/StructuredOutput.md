Structured Output refers to AI models that can reliably generate JSON or other structured data formats rather than free-form text. This is critical for WorldAI where the LLM must output game state in a specific JSON schema.

## Relevance to WorldAI
- Game state updates require specific JSON field structure
- Gemini is noted for strong structured output capability
- Must use models that can reliably follow JSON schemas
