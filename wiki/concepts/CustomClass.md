## Definition
Homebrew character class creation method allowing players to design custom classes with unique mechanics. One of three options in the Game Mechanics Protocol for D&D 5e character creation.

## Requirements
- Define class features by level
- Specify HP, primary ability, saving throws
- List proficiencies (armor, weapons, tools, skills)
- Define spellcasting (if applicable)
- Balance against standard classes
