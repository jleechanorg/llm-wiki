## Definition
The underlying systems, utilities, and frameworks that support automated testing. Includes test runners, assertion libraries, fixtures, helpers, and mock frameworks.

## In This Context
The architecture analysis helpers are part of the test infrastructure, specifically supporting architectural decision tests that need simple, deterministic file analysis without external tooling dependencies.
