## Definition
A d20 roll that determines the order of combatants in a round. Higher rolls act first.

## Formula
```
Initiative = d20 + DEX Modifier (no proficiency)
```

## Key Properties
- **No proficiency** — characters never add proficiency bonus to initiative
- **DEX-based** — uses Dexterity modifier regardless of class
- **Ties**: All tied creatures act in order determined by DM
