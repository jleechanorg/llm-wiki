## Definition
The target number a creature must meet or exceed on a saving throw to resist a spell effect.

## Formula
```
Spell DC = 8 + Proficiency Bonus + Casting Ability Modifier
```

## Examples (Level 1 Bard, CHA 16 = +3)
- Spell DC = 8 + 2 (prof) + 3 (CHA) = **13**

## Usage
- Creatures make saving throws against this value
- Spells specify which ability the save uses (STR, DEX, CON, INT, WIS, or CHA)
