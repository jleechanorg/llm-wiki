An evidence bundle is a structured collection of test output artifacts generated by create_evidence_bundle. It typically includes evidence.md (markdown summary of test results), provenance data, and test result artifacts.

## Bundle Contents
- evidence.md — markdown summary showing total scenarios, passed/failed counts
- Provenance data — git HEAD, branch, timestamp
- Test results — formatted scenarios with success/error details
