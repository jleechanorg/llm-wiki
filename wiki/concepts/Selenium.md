## Definition
Web browser automation framework that allows programmatic control of web browsers for testing and scraping purposes.

## Key Features
- Cross-browser support (Chrome, Firefox, Safari, Edge)
- Multiple language bindings (Python, Java, JavaScript, C#)
- WebDriver protocol for browser interaction
- Headless mode support for background execution

## Usage in This Project
Used for browser automation tests including campaign wizard screenshot capture and end-to-end testing workflows.
