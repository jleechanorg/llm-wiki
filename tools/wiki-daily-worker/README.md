# Wiki Daily Worker

Daily launchd job for `~/llm_wiki`: optional ingest of new `raw/*.md` via Claude MiniMax, plus entity/concept ratio logging.

## Install (recommended)

Installs **only under your home directory** — not under OpenClaw, not via paths into this repo:

```bash
cd /path/to/llm_wiki
chmod +x tools/wiki-daily-worker/install.sh
./tools/wiki-daily-worker/install.sh
```

This will:

1. Copy `wiki-daily-worker.sh` → **`~/Library/Application Support/llm-wiki/daily-worker.sh`**
2. Write **`~/Library/LaunchAgents/com.jleechan.wiki-daily-worker.plist`**
3. `launchctl bootstrap` the agent (daily at **8:00**)

Logs: `~/Library/Logs/wiki-daily-worker.log` and `wiki-daily-worker.error.log`.

### Environment

| Variable | Default |
|----------|---------|
| `LLM_WIKI_DIR` | `$HOME/llm_wiki` |
| `MEMORY_WIKI_DIR` | `$HOME/memory/wiki` |

LaunchAgents do not load your shell profile. To override paths, edit `daily-worker.sh` after install or wrap the script.

### Manual run

```bash
~/Library/Application\ Support/llm-wiki/daily-worker.sh
```

### Uninstall

```bash
launchctl bootout "gui/$(id -u)" ~/Library/LaunchAgents/com.jleechan.wiki-daily-worker.plist
rm -f ~/Library/LaunchAgents/com.jleechan.wiki-daily-worker.plist
rm -rf ~/Library/Application\ Support/llm-wiki
```

(Remove old OpenClaw copy if you still have it: `~/.openclaw/scripts/wiki-daily-worker.sh`.)

## Repo layout

- `wiki-daily-worker.sh` — source script (copied by `install.sh`)
- `install.sh` — installs to `~/Library/Application Support/llm-wiki/`
- `launchd/README.md` — real plist is generated by `install.sh` into `~/Library/LaunchAgents/`

## Requirements

- `claude` CLI (when ingest runs)
- MiniMax routing if you use the `MiniMax-M2.5` model as in the script
- `python3` for ratio logging
